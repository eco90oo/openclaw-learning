# 📚 Day 2：SQL 基礎語法（CRUD 四大天王）

## 1️⃣ 基礎概念

### 什麼是 SQL？

SQL = Structured Query Language（結構化查詢語言）

簡單說就是**和資料庫溝通的語言**。就像：
- 中文是和中國人溝通的語言
- 英文是和外國人溝通的語言
- **SQL 是和資料庫溝通的語言**

### CRUD - 資料庫的四大天王

| 字母 | 意思 | SQL 指令 | 做什麼 |
|------|------|----------|--------|
| C | Create | INSERT | 新增資料 |
| R | Read | SELECT | 讀取資料 |
| U | Update | UPDATE | 修改資料 |
| D | Delete | DELETE | 刪除資料 |

**記住口號：「C建R讀U改D刪」**

### 📝 INSERT - 新增資料（Create）

**語法：**
```sql
INSERT INTO 表格名稱 (欄位1, 欄位2, ...) VALUES (值1, 值2, ...);
```

**實際範例：**

先創建一張學生表（如果還沒有的話）：
```sql
CREATE TABLE students (
    id INTEGER PRIMARY KEY,
    name TEXT NOT NULL,
    age INTEGER,
    grade TEXT
);
```

插入一筆資料：
```sql
INSERT INTO students (name, age, grade) VALUES ('王小明', 20, 'A');
```

插入多筆資料：
```sql
INSERT INTO students (name, age, grade) VALUES 
    ('李美華', 22, 'B'),
    ('張志強', 19, 'A'),
    ('陳小娟', 21, 'C');
```

**小提醒：**
- 字串（文字）要用單引號包起來：`'王小明'`
- 數字不用引號：`20`
- 主鍵（id）通常會自動產生，可以省略不寫

---

### 📖 SELECT - 查詢資料（Read）

**語法：**
```sql
SELECT 欄位1, 欄位2, ... FROM 表格名稱 [WHERE 條件];
```

**各種查詢範例：**

查詢所有欄位：
```sql
SELECT * FROM students;
```
（* 是「全部欄位」的簡寫）

查詢特定欄位：
```sql
SELECT name, grade FROM students;
```

加上條件查詢：
```sql
-- 查詢年齡大於 20 歲的學生
SELECT * FROM students WHERE age > 20;

-- 查詢成績是 A 的學生
SELECT * FROM students WHERE grade = 'A';

-- 查詢年齡 20 到 25 歲之間
SELECT * FROM students WHERE age >= 20 AND age <= 25;
```

排序查詢結果：
```sql
-- 按年齡從小到大排序
SELECT * FROM students ORDER BY age ASC;

-- 按年齡從大到小排序
SELECT * FROM students ORDER BY age DESC;
```

只顯示前幾筆：
```sql
-- 只顯示前 3 筆
SELECT * FROM students LIMIT 3;

-- 跳過前 2 筆，然後顯示 3 筆
SELECT * FROM students LIMIT 3 OFFSET 2;
```

---

### 🔄 UPDATE - 更新資料（Update）

**語法：**
```sql
UPDATE 表格名稱 SET 欄位1 = 新值1, 欄位2 = 新值2 WHERE 條件;
```

**重要：WHERE 一定要加！否則會更新所有資料！**

**實際範例：**

```sql
-- 把王小明的成績改成 A+
UPDATE students SET grade = 'A+' WHERE name = '王小明';

-- 把所有學生的年齡加 1 歲（危險！不要轻易執行）
-- UPDATE students SET age = age + 1;
```

更精確的更新：
```sql
-- 把 id 為 1 的學生年齡改成 21
UPDATE students SET age = 21 WHERE id = 1;
```

---

### 🗑️ DELETE - 刪除資料（Delete）

**語法：**
```sql
DELETE FROM 表格名稱 WHERE 條件;
```

**重要：WHERE 一定要加！否則會刪除所有資料！**

**實際範例：**

```sql
-- 刪除名字叫「張志強」的學生
DELETE FROM students WHERE name = '張志強';

-- 刪除年齡小於 18 歲的學生
DELETE FROM students WHERE age < 18;

-- 刪除所有資料（千萬不要執行！）
-- DELETE FROM students;
```

## 💡 小知識

**SQL 執行順序（內部運作）：**
1. FROM（從哪張表）
2. WHERE（篩選條件）
3. SELECT（選擇欄位）
4. ORDER BY（排序）

雖然我們寫的順序是 SELECT... FROM... WHERE... ORDER BY，但資料庫內部是從 FROM 開始執行的！

**一個小測驗：**
```sql
SELECT name FROM students WHERE age > 20 ORDER BY name;
```
請問執行順序是什麼？
答案：FROM → WHERE → SELECT → ORDER BY
