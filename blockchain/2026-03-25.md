## 1️⃣ 基礎概念（用生活比喻）

### 🎯 什麼是非對稱加密？

想象一下：

- **對稱加密** = 只有一把鑰匙
  - 鎖門和開門都用同一把鑰匙
  - 別人拿到鑰匙就能開門

- **非對稱加密** = 兩把鑰匙！
  - **公鑰** = 鎖（公開給大家）
  - **私鑰** = 鑰匙（只有自己知道）
  - 別人用公鑰鎖門，只有你能用私鑰開門！

### 🗝️ 公鑰與私鑰

```
比喻：郵箱

📬 公鑰 = 郵箱的投遞口
   - 任何人都可以把信投進來
   - 但只能投，不能取出

🔑 私鑰 = 郵箱的鑰匙
   - 只有你有
   - 只有你能打開取出信件
```

### ✍️ 數位簽章

```
比喻：手寫簽名

手寫簽名 = 證明「這是我本人簽的」
數位簽章 = 證明「這是我本人發的訊息」

過程：
1. 我用私鑰「簽名」訊息
2. 對方用我的公鑰「驗證」簽名
3. 驗證成功 = 確認是我本人
```

## 2️⃣ 核心知識（詳細教學）

### 🔐 比特幣地址生成過程

```
1. 生成私鑰（256位隨機數）
   └─> 私鑰是安全的根源
   
2. 私鑰 → 橢圓曲線運算 → 公鑰
   └─> 只能從私鑰推出公鑰，無法反推
   
3. 公鑰 → Hash → 地址
   └─> 地址是公鑰的 hash
```

### 📊 常見加密演算法

| 演算法 | 用途 | 說明 |
|--------|------|------|
| **RSA** | 加密/簽章 | 最早的非對稱加密 |
| **ECDSA** | 比特幣簽章 | 橢圓曲線，更短更快 |
| **EdDSA** | 新一代簽章 | 更安全，現代區塊鏈用 |

### 🧮 橢圓曲線（ECC）

比特幣使用的是 **secp256k1** 曲線：

```
橢圓曲線方程：y² = x³ + 7

特點：
- 運算速度快（同等安全性下，key 更短）
- 適合資源受限的設備（手機、區塊鏈）
- 256 位 ECC ≈ 3072 位 RSA 安全性
```

## 3️⃣ 實際操作（Python 範例）

```python
# 需要安裝：pip install coincurve（橢圓曲線庫）

from coincurve import PrivateKey, PublicKey
import hashlib
import base64

# ====== Step 1: 生成私鑰 ======
# 私鑰 = 256 位隨機數
# 就像產生一個密碼一樣

private_key = PrivateKey()  # 自動生成隨機私鑰

# 取得私鑰（Hex 格式）
print("私鑰（Hex）:", private_key.hex())
# 輸出範例：8f9c...（64個十六進制字符）

# ====== Step 2: 從私鑰推導公鑰 ======
# 橢圓曲線運算：私鑰 × G = 公鑰
public_key = private_key.public_key

# 取得公鑰（未壓縮格式，65 bytes）
print("公鑰（Hex）:", public_key.format().hex())
# 輸出範例：04a3...（130個字符，04開頭）

# ====== Step 3: 公鑰 → 地址 ======
# 步驟：公鑰 → SHA256 → RIPEMD160 → 檢查碼 → 地址

def public_key_to_address(public_key):
    # 1. 公鑰先做 SHA256 hash
    sha256_hash = hashlib.sha256(public_key).digest()
    
    # 2. 再做 RIPEMD160
    import hashlib
    ripemd160 = hashlib.new('ripemd160')
    ripemd160.update(sha256_hash)
    hashed = ripemd160.digest()
    
    # 3. 加入版本字首（比特幣用 0x00）
    versioned = b'\x00' + hashed
    
    # 4. 計算校驗碼（前4 bytes）
    checksum = hashlib.sha256(
        hashlib.sha256(versioned).digest()
    ).digest()[:4]
    
    # 5. 組合並用 Base58 編碼
    import base58
    address = base58.b58encode(versioned + checksum)
    
    return address.decode()

address = public_key_to_address(public_key.format())
print("比特幣地址:", address)
# 輸出範例：1A1zP1eP5QGefi2DMPTfTL5SLmv7DivfNa
```

### 🔏 數位簽章實作

```python
from coincurve import PrivateKey, PublicKey
import hashlib
import base64
import json

# ====== 假設這是你要發送的訊息 ======
message = json.dumps({
    "from": "小明",
    "to": "小華",
    "amount": 1.5
})

# ====== 發送方：簽名 ======
# 1. 計算訊息的 hash
message_hash = hashlib.sha256(message.encode()).digest()

# 2. 用私鑰簽名
private_key = PrivateKey()  # 發送方的私鑰
signature = private_key.sign(message_hash)

print("簽名:", base64.b64encode(signature).decode())
# 輸出範例：MEUCIQCK...（Base64編碼的簽名）

# ====== 接收方：驗證 ======
# 1. 取得發送方的公鑰
public_key = private_key.public_key

# 2. 驗證簽名
is_valid = public_key.verify(signature, message_hash)

print("驗證結果:", "✓ 簽名有效" if is_valid else "✗ 簽名無效")
# 輸出：✓ 簽名有效
```

## 4️⃣ 常見錯誤提醒

| 錯誤 | 說明 |
|------|------|
| ❌ 把私鑰存在雲端 | 私鑰等於銀行密碼，必須離線存放 |
| ❌ 用別人的公鑰地址匯款 | 匯錯就拿不回來 |
| ❌ 沒備份私鑰 | 硬碟壞掉 = 錢包歸零 |
| ❌ 把地址當私鑰 | 地址是公開的，私鑰是秘密 |
