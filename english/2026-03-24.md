## 3️⃣ 聽力進步的秘密

### 你為什麼聽不懂？

1. **太快」不是問題 - 是因為「不熟悉」
2. **連音** - 英语會把两个单词连在一起读
3. **不熟悉主題」 - 專業術語不懂

### 解決方法：i+1 原則

```
i = 你現在的程度
+1 = 只比現在難一點點的內容

不要一開始就聽很難的！
從適合的難度開始，慢慢增加難度
```

### 推薦的聽力資源

| 程度 | 資源 |
|------|------|
| 初級 | 空中英語教室、ESL Pod、VoiceTube 初中級 |
| 中級 | CNN Student News、TED-Ed |
| 中高級 | TED 演講、LinkedIn Learning |
| 高級 | 任何 native speaker 演講 |

## 4️⃣ 閱讀進步的秘密

### 什麼是「意思串聯不起來」？

這叫做「翻譯式閱讀」：
- 看到每個單字 → 翻成中文
- 翻完後 → 忘記前面說什麼

### 解決方法：用英文理解英文

```
不要翻譯！
看到 "fantastic" → 直接想到「太棒了」的感覺
不要經過中文！
```

### 練習技巧：Chunking（分塊閱讀）

```
不要這樣讀：The / cat / is / on / the / table
要這樣讀：The cat / is on the table / 
（一個意思單位一個單位讀）
```

## 5️⃣ 口說進步的秘密

### 你需要的是「口腔肌肉記憶」

說英語需要的肌肉：
- 舌頭位置
- 嘴唇形狀
- 氣流控制

這些肌肉你從小沒練過！所以要訓練！

### 跟讀法（Shadowing）

```
步驟：
1. 找一個音檔（1-2分鐘）
2. 聽第一遍 - 理解意思
3. 聽第二遍 - 注意發音、語調
4. 聽第三遍 - 開始跟著說
5. 一直重複 - 直到可以跟上速度
6. 錄音 - 比較自己和原聲的差別
```
