# Day 1: 建立正確的心態與方法 - 了解學習盲點

## 1️⃣ 基礎概念（用生活比喻）

### 為什麼學英文這麼難？

想像一下：你學騎腳踏車時，爸媽告訴你「維持平衡、踩踏板、看前方」，但實際上，你是在一次次摔倒中，找到那個「感覺」——英文也是這樣！

**中文母語者的學習盲點：**

| 盲點 | 問題 | 比喻 |
|------|------|------|
| 翻譯式學習 | 中翻英、英翻中 | 戴著兩層手套握手 |
| 只背單字 | 單獨記意思 | 背了磚塊不會蓋房子 |
| 害怕犯錯 | 不敢開口 | 學游泳卻不下水 |
| 追求完美 | 等准备好再說 | 等學會所有廚藝再下廚 |

## 2️⃣ 核心知識（英文解說 + 中文解釋）

### 🧠 正確的心態

> **"Language is a skill, not a knowledge."**
> 語言是一種技能，不是知識。就像騎腳踏車或游泳，你需要**身體力行**，不能只看書就學會。

**三大錯誤心態：**

1. **「我等英文變好再說」** ❌
   - 正確：邊做邊學，犯錯就是學習
   
2. **「我需要標準答案」** ❌  
   - 正確：英文沒有唯一正確，很多表達都可行

3. **「我怕別人笑我」** ❌
   - 正確：沒人會笑你努力！他們可能還會佩服

### 🎯 正確的学习方法

```
輸入（Input）→ 處理（Process）→ 輸出（Output）

正確循環：
聽讀 → 理解 → 說寫 → 修正 → 進步

錯誤循環：
學語法 → 背單字 → 不敢說 → 永遠學不會
```
