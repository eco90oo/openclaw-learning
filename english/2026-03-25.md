## 1️⃣ 為什麼你聽不懂英文？

### 真相：不是因為太快！

```
你以為的問題：
❌ 英文太快了
❌ 我的字彙量不夠
❌ 我發音不好

真正的問題：
✅ 連音 (Linking) - 單字連在一起
✅ 省略音 (Reduction) - 某些音被吃掉
✅ 不熟悉發音規則
```

### 聽不懂的意思？

```
看這句：
"I went to the store to buy some food."

慢慢讀：
I / went / to / the / store / to / buy / some / food

Native speaker 讀：
IwenthetostoreTbuysomefood
```

**所有單字連在一起了！**

## 2️⃣ 最常見的連音規則（全部都是真的在用的！）

### 規則一：輔音 + 元音 = 黏在一起

這是最常見的連音！當一個字結尾是輔音，下一個字開頭是元音，會「黏」成一個字：

| 口語說法 | 真正意思是 | 範例句子 |
|---------|-----------|----------|
| **gonna** | going to | I'm **gonna** go to the meeting. |
| **wanna** | want to | I **wanna** grab some coffee. |
| **gotta** | got to | I **gotta** finish this today. |
| **kinda** | kind of | It's **kinda** hard. |
| **sorta** | sort of | It's **sorta** complicated. |
| **outta** | out of | Get **outta** here! |
| **lemme** | let me | **Lemme** think about it. |
| ** Gimme** | Give me | **Gimme** a break! |
| **dunno** | I don't know | I **dunno** what to do. |

### 練習方法：

```
1. 大聲念出上面的表格
2. 念到很自然為止
3. 記住：這些是「正式發音的簡化版」
```

### 規則二：相同輔音 = 只發一次

兩個相同的輔音在一起，發一次但拉長：

| 錯誤發音 | 正確發音 | 範例 |
|---------|---------|------|
| big girl | bi' girl | That's a big girl. |
| bad boy | ba' boy | He's a bad boy. |
| hot tea | ho' tea | Do you want some hot tea? |
| sit down | si' down | Sit down please. |
| what time | wha' time | What time is it? |

### 規則三：t/d 省略 = 輕到聽不見

在兩個母音之間，t 和 d 的聲音會變得很輕或消失：

| 你以為是 | 實際是 | 範例 |
|---------|--------|------|
| better | bet'er | You **better** think about it. |
| wanted | wan'ed | I **wanted** to go. |
| what do | wha' do | **What** do you think? |
| could you | coul' you | **Could** you help me? |
| not really | no' really | It's **not** really important. |
| yesterday | yes'erday | I saw him **yesterday**. |

## 4️⃣ Shadowing（跟讀法）練習步驟

### Shadowing 是什麼？

```
Shadow = 影子

就像影子跟著人一樣，
你的聲音跟著 Native Speaker 的聲音
```

### 具體步驟：

**第一步：選擇一段 30 秒的影片**

建議從這個開始：
```
 Rachel's English - Connected Speech
 https://www.youtube.com/watch?v=n4NVPg2gHv4
 從 2:30 開始的片段
```

**第二步：聽 3 次**
```
第1次：了解意思
第2次：注意他的發音和語調
第3次：注意連音在哪裡
```

**第三步：開始跟著說**
```
暫停 → 說 → 暫停 → 說
盡量模仿他的：
- 語調
- 速度
- 連音位置
```

**第四步：錄音**
```
手機錄音功能打開
錄自己說的
```

**第五步：對比**
```
播放原聲
播放自己的錄音
找出不一樣的地方
```

## 5️⃣ 今日必學會話常用句（連音版）

### 辦公室常用：

| 正式說法 | 口語連音版 | 中文 |
|---------|-----------|------|
| I want to | I wanna | 我想要 |
| I have to | I gotta | 我必須 |
| I don't know | I dunno | 我不知道 |
| What do you | Whatd'you | 你覺得 |
| Did you | Didju | 你有沒有 |
| Would you | Wouldja | 你可以 |
| Let me | Lemme | 讓我 |
| Give me | Gimme | 給我 |
| Going to | Gonna | 將要 |
| Have to | Hafta | 必須 |

### 對話練習：

```
A: Hey, whatd'you gonna do this weekend?
B: I dunno yet. Gotta work on my project.
A: Wanna grab some coffee later?
B: Sure! Lemme finish this first.
A: No problem. Whad'you say 3pm?
B: Sounds good! See ya then!
```
