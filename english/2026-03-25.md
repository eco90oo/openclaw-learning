# 📚 【英文學習｜Day 2：聽力突擊 - 從含糊不清的連音開始】

> 目標：從 TOEIC 600 到職場流利英語

## 1️⃣ 為什麼聽力那麼難？

### 聽力不好的根本原因

你以為聽力不好是因為：
- ❌ 太快了
- ❌ 單字不够
- ❌ 发音不对

其實真正的原因是：
- ✅ **連音 (Linking)** - 英文單字和單字連在一起
- ✅ **省略 (Reduction)** - 某些音被吃掉了
- ✅ **不熟悉發音規則** - 以為每個音都會清楚發出

### 例子示範

```
慢慢讀：I / went / to / the / store / to / buy / some / food
快速讀：Iwent to thestore to buysome food
```

有沒有發現？單字和單字之間幾乎沒有空隙了！

## 2️⃣ 最常見的連音規則

### 規則 1：子音 + 母音 = 連在一起

當一個字結尾是子音，下一個字開頭是母音，會「黏」在一起：

| 範例 | 發音 |
|------|------|
| not at all | notatall |
| think about it | thinkaboutit |
| get out of here | getoutahere |
| kind of | kinda |
| going to | gonna |
| want to | wanna |
| got to | gotta |

**練習方法**：把上面每個片語大聲念出来，注意要連貫！

### 規則 2：相同子音連音

兩個相同的子音在一起，只發一次音，但拉長一點：

| 範例 | 發音 |
|------|------|
| big girl | bi'r girl（g只發一次）|
| bad boy | ba' boy |
| hot tea | ho' tea |
| sit down | si' down |

### 規則 3：結尾的 t/d 被「吃掉了」

在兩個母音之間，t 和 d 的聲音會變得很輕或消失：

| 範例 | 實際發音 |
|------|---------|
| better think about it | bet'er think about it |
| I wanted to go | I wan'ed to go |
| what do you think | wha' do you think |
| could you help me | coul' you help me |

## 3️⃣ 聽力練習：如何訓練？

### 第一步：找對教材

初學者（多益 400-600）：
- VoiceTube「日常英語」
- ESL Pod
- CNN Student News

中級（多益 600-750）：
- TED 演講（可開字幕）
- Bloomberg Quicktake
- 技術 Podcast

高級（多益 750+）：
- 無字幕看電影
- Native speaker 任何演講

### 第二步：聽三遍法

```
第一遍（了解大意）：
- 開中文字幕
- 了解說了什麼

第二遍（注意細節）：
- 開英文字幕
- 把不會的單字記下來

第三遍（聽力訓練）：
- 關字幕
- 盡量聽懂每個字
- 特別注意連音的地方
```

### 第三步：Shadowing（跟讀法）

聽完三遍後，找一個你喜歡的片段（30秒）：
1. 聽一句
2. 暫停
3. 跟著說一遍（模仿語調和連音）
4. 錄音
5. 聽回放，注意哪裡不一樣

## 4️⃣ 今日重點單字片語

| 英文 | 中文 | 例句 |
|------|------|------|
| gonna | 要（going to的口語）| I'm gonna go to the meeting. |
| wanna | 想要（want to的口語）| I wanna grab some coffee. |
| gotta | 必須（got to的口語）| I gotta finish this by today. |
| kinda | 有點（kind of的口語）| It's kinda hard. |
| sort of | 有點 | It's sort of complicated. |
| yeah | 是的（yes的口語）| Yeah, I agree. |
| gonna | 将要 | We're gonna launch next week. |
