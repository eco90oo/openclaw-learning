## 1️⃣ 基礎概念（用生活比喻，越詳細越好）

### 🎯 什麼是 React？

**比喻**：想像你在玩樂高積木

- **傳統網頁**：像是一張固定的圖畫，畫好就不能動
- **React**：像是一盒樂高，每次看到的是「組裝說明書」，樂高會自己組合成畫面

**更生活化的比喻**：
> React 就像餐廳的「自助點餐機」。你點什麼，它就顯示什麼。而且當廚房做好菜時，螢幕會自動更新，不需要人再去按重新整理。

### 🧩 什麼是「元件」（Component）？

**比喻**：元件就像「組裝玩具的零件」

- 一輛玩具車有：輪子、車身、方向盤、引擎
- 每個零件都可以獨立生產、獨立測試
- 組裝在一起就變成一輛車

**在 React 裡**：
- 一個按鈕是一個元件
- 一張卡片是一個元件
- 一整個頁面也是一個元件
- 我們把小的元件組合成大的元件，最後組成整個網頁

## 2️⃣ 核心知識（每行程式碼都要有中文註解！）

### 👶 第一個 React 元件

```jsx
// 這一行叫做「函式宣告」
// 我們定義了一個叫做 App 的函式（也就是元件）
// 這個函式會回傳一段 HTML（嚴格說是 JSX）
function App() {
  // return 是「回傳」的意思
  // 我們回傳一段 HTML（這叫做 JSX）
  return (
    // 這裡可以放 HTML，但嚴格說是 JavaScript 寫的 HTML
    // 這個 <div> 包住了所有內容
    <div>
      {/* 這是一個標題標籤 */}
      <h1>你好，我是 React！</h1>
      
      {/* 這是一個段落 */}
      <p>歡迎來到我的第一個網頁</p>
    </div>
  );
}

// 把這個元件導出，讓其他地方可以使用
export default App;
```

### 🧱 元件可以重複使用

```jsx
// 定義一個叫 Button 的元件（按鈕）
function Button() {
  // 回傳一個按鈕的 HTML
  return <button>按我一下</button>;
}

// 定義主要的 App 元件
function App() {
  return (
    <div>
      {/* 我們可以一直重複使用 Button 元件 */}
      <Button />
      <Button />
      <Button />
    </div>
  );
}

export default App;
```

**⚡ 執行結果**：畫面上會出現三個按鈕！

### 📝 元件可以接收「道具」（Props）

**比喻**：Props 就像「結婚聘禮」或是「禮物盒」

- 父母給你禮物，你打開來看
- 元件收到 Props，就可以使用裡面的資料

```jsx
// 定義一個 Card 元件（卡片）
// 「props」是「properties」的縮寫
// 也就是「屬性」的意思 - 像是一個禮物盒
function Card(props) {
  // props.title 就是禮物盒裡的「標題」
  // 我們把它顯示在 h2 標籤裡
  return (
    <div className="card">
      <h2>{props.title}</h2>
      <p>{props.content}</p>
    </div>
  );
}

// 主元件
function App() {
  return (
    <div>
      {/* 我們在使用 Card 時，可以傳資料進去 */}
      {/* 這叫做「傳遞 props」 */}
      <Card title="第一章" content="這是第一章的內容" />
      <Card title="第二章" content="這是第二章的內容" />
      <Card title="第三章" content="這是第三章的內容" />
    </div>
  );
}

export default App;
```

**⚡ 執行結果**：
```
第一章
這是第一章的內容

第二章
這是第二章的內容

第三章
這是第三章的內容
```

### 🎨 加入 CSS 樣式

```jsx
// 首先在 JS 中引入 CSS 檔案
import './App.css';

// 定義一個按鈕元件
function Button(props) {
  // 我們可以根據 props 決定按鈕的類型
  // 如果 props.primary 存在，就用 primary class
  return (
    // className 是 CSS 的「class」
    // 就像HTML的 class 屬性，但因為 JavaScript 關鍵字衝突，要寫 className
    <button className={props.primary ? "btn-primary" : "btn-secondary"}>
      {props.text}
    </button>
  );
}

function App() {
  return (
    <div>
      {/* 傳遞不同 props，產生不同樣式的按鈕 */}
      <Button text="確認" primary={true} />
      <Button text="取消" primary={false} />
    </div>
  );
}

export default App;
```

```css
/* App.css 檔案的內容 */

/* 基本的按鈕樣式 */
.btn-primary {
  /* 背景顏色設為藍色 */
  background-color: blue;
  /* 文字顏色設為白色 */
  color: white;
  /* 邊框設為無 */
  border: none;
  /* 內邊距：上下 10px，左右 20px */
  padding: 10px 20px;
  /* 圓角：讓按鈕角落圓圓的 */
  border-radius: 5px;
}

.btn-secondary {
  /* 灰色背景 */
  background-color: gray;
  /* 白色文字 */
  color: white;
  /* 無邊框 */
  border: none;
  /* 內邊距 */
  padding: 10px 20px;
  /* 圓角 */
  border-radius: 5px;
}
```

## 3️⃣ 實際操作（一步一步帶著做）

### Step 1：建立 React 專案

```bash
# 打開終端機（Terminal）
# 進入你要放專案的資料夾
cd Desktop

# 使用 create-react-app 建立新專案
# 這會幫你安裝好所有需要的東西
npx create-react-app my-first-app

# 進入專案資料夾
cd my-first-app

# 啟動開發伺服器
npm start
```

**🎉 執行結果**：瀏覽器會自動打開 http://localhost:3000，看到 React 預設畫面

### Step 2：修改 App.js

找到 `src/App.js` 檔案，內容改成：

```jsx
// 引入 React 函式庫
import React from 'react';

// 引入 CSS 樣式
import './App.css';

// 定義一個 greeting 元件（打招呼）
function Greeting(props) {
  // props.name 是传入的名字
  return <h1>你好，{props.name}！</h1>;
}

// 定義一個按鈕元件
function MyButton(props) {
  // 回傳一個按鈕，按鈕文字從 props.text 取得
  return <button className="my-btn">{props.text}</button>;
}

// 主要的 App 元件
function App() {
  return (
    // HTML 結構
    <div className="app">
      {/* 使用 Greeting 元件，傳入 name="小明" */}
      <Greeting name="小明" />
      <Greeting name="小華" />
      
      {/* 使用 MyButton 元件 */}
      <MyButton text="按我" />
      <MyButton text="不要按我" />
    </div>
  );
}

// 導出 App 元件
export default App;
```

### Step 3：修改 App.css

找到 `src/App.css` 檔案，內容改成：

```css
/* 設定整個 app 的樣式 */
.app {
  /* 文字置中 */
  text-align: center;
  /* 上方間距 */
  margin-top: 50px;
  /* 字體大小 */
  font-size: 20px;
}

/* 按鈕的樣式 */
.my-btn {
  /* 背景藍色 */
  background-color: #4CAF50;
  /* 文字白色 */
  color: white;
  /* 無邊框 */
  border: none;
  /* 內邊距 */
  padding: 15px 32px;
  /* 文字置中 */
  text-align: center;
  /* 行內區塊（讓按鈕可以並排） */
  display: inline-block;
  /* 字體大小 */
  font-size: 16px;
  /* 上方外邊距 */
  margin: 4px 2px;
  /* 游標變成指標 */
  cursor: pointer;
  /* 圓角 */
  border-radius: 8px;
}

/* 滑到按鈕上時的效果 */
.my-btn:hover {
  /* 背景變深 */
  background-color: #45a049;
}
```

3. 你會看到：
   - 「你好，小明！」和「你好，小華！」兩個標題
