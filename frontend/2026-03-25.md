# Day 1：HTML 基礎標籤 — 超詳細註解版

今天的課程是「HTML 基礎標籤」。我們要把網頁想成一間房子，每個 HTML 標籤就像是房子裡的不同傢俱：標題是看板、段落是文章、圖片是裝飾品、連結是門。讓我們一個一個認識它們！

---

## 1️⃣ 基礎概念（用生活比喻）

### 什麼是 HTML？

**比喻**：HTML 就像是蓋房子的藍圖。它告訴瀏覽器（像是施工團隊）：「這裡要放客廳、那裡要放臥室、臥室裡要擺床。」

- **HTML** = HyperText Markup Language（超文字標記語言）
- 它不是程式語言，而是一套「標記」系統
- 瀏覽器（Chrome、Edge、Safari）會讀取 HTML 並「翻譯」成我們看到的網頁

### 標籤（Tag）是什麼？

**比喻**：標籤就像是「標籤貼紙」。你在一個東西上貼標籤，別人就知道這是什麼。例如：
- 在箱子上貼「書籍」→ 別人就知道這是書
- 在門上貼「入口」→ 別人就知道這是門

HTML 的標籤語法長這樣：`<標籤名稱>內容</標籤名稱>`
- 開頭標籤：`<h1>`
- 結束標籤：`</h1>`

## 2️⃣ 核心知識（每行程式碼都要有中文註解！）

讓我們從零開始建立一個網頁！請跟著一步一步做。

### 第一步：創建 HTML 檔案

```html
<!DOCTYPE html>
<!-- 
  這一行不是給人看的，是給瀏覽器看的「聲明」
  意思是：「我要用最新版本的 HTML5 語法，請用 HTML5 的規則來解讀這個檔案」
  就像是把護照递给海關，告訴他你要用哪個國家的入境規則
-->
<html lang="zh-TW">
<!-- 
  <html> 是「網頁的根標籤」，所有網頁內容都要包在裡面
  lang="zh-TW" 是告訴瀏覽器：「這個網頁是用繁體中文寫的」
  這有助於搜尋引擎和語音朗讀工具正確理解網頁語言
-->
  
  <head>
    <!-- 
      <head> 是「網頁的頭部」，放的是「看不見」但很重要的設定
      就像公司的總經理辦公室 — 從外面看不出來，但管理著整家公司
    -->
    <meta charset="UTF-8">
    <!-- 
      <meta> 是「設定」的標籤，charset="UTF-8" 是設定「字元編碼」
      UTF-8 是一種萬國碼，確保中文、英文、表情符號都能正確顯示
      沒有這一行，中文可能會變成亂碼（例如「???」）
    -->
    <title>我的第一個網頁</title>
    <!-- 
      <title> 設定「網頁標題」，會顯示在瀏覽器的分頁上
      就像書的書名，別人看到分頁就知道這個網頁是什麼
    -->
  </head>
  <!-- head 結束 -->

  <body>
    <!-- 
      <body> 是「網頁的身體」，放的是「看得見」的內容
      這是我們實際要在網頁上顯示的所有東西
    -->
    
    <!-- 這是標題標籤 -->
    <h1>歡迎來到我的網站</h1>
    <!-- 
      <h1> 是「第一階標題」，通常用於網頁的主標題
      h1 的「h」是「heading（標題）」的縮寫
      數字 1 表示這是最重要的標題
      瀏覽器會自動把 h1 顯示成大字、粗體
      一個網頁通常只有一個 h1（像一本書只有一個書名）
    -->

    <h2>關於我</h2>
    <!-- 
      <h2> 是「第二階標題」，用於章節標題
      比 h1 小一號，表示重要性第二
      就像書的「第一章」、「第二章」
    -->

    <!-- 這是段落標籤 -->
    <p>大家好！我是一位正在學習前端的学生。</p>
    <!-- 
      <p> 是「段落（paragraph）」標籤
      用於包住一段文字，段落前後會自動留空白
      就像word文件裡的段落，會自動換行
    -->

    <!-- 這是強調標籤 -->
    <p>我正在學習 <strong>HTML</strong> 和 <strong>CSS</strong>。</p>
    <!-- 
      <strong> 是「強調」標籤，會讓文字變粗體
      通常用於強調重要的關鍵字
      就像在書本上用螢光筆畫重點
    -->
    
    <!-- 這是斜體強調 -->
    <p><em>持續學習</em> 是成為工程師的關鍵！</p>
    <!-- 
      <em> 是「斜體強調」標籤
      em 是 "emphasis"（強調）的縮寫
      瀏覽器會把它顯示為斜體
    -->

    <!-- 這是無序列表（bullet points） -->
    <h2>我喜歡的科目</h2>
    <ul>
      <!-- <ul> 是「無序列表（unordered list）」，會顯示圓點或星星 -->
      <li>數學</li>
      <!-- <li> 是「列表項目（list item）」，每一個項目都用 li 包住 -->
      <li>物理</li>
      <li>化學</li>
    </ul>
    <!-- 就像購物清單前面的圓點符號 -->

    <!-- 這是有序列表（編號） -->
    <h2>學習步驟</h2>
    <ol>
      <!-- <ol> 是「有序列表（ordered list）」，會顯示 1. 2. 3. 數字編號 -->
      <li>學好 HTML</li>
      <li>學好 CSS</li>
      <li>學好 JavaScript</li>
    </ol>
    <!-- 就像食譜裡的步驟 1、2、3 -->

    <!-- 這是連結（ hyperlink ） -->
    <p>想了解更多？<a href="https://www.google.com">點擊這裡</a>搜尋更多資訊！</p>
    <!-- 
      <a> 是「錨點（anchor）」標籤，用來建立連結
      href="網址" 是「hypertext reference」的縮寫，設定要連結到哪裡
      就像在書本上貼一個「請參閱第 30 頁」的標籤
      點擊「點擊這裡」就會跳到 Google
    -->

    <!-- 這是圖片標籤 -->
    <img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools 標誌">
    <!-- 
      <img> 是「圖片（image）」標籤
      src="..." 是「source（來源）」，設定圖片檔案的路徑或網址
      alt="..." 是「alternative text（替代文字）」，當圖片載入失敗時會顯示這段文字
      alt 也對搜尋引擎和視障人士有幫助（螢幕閱讀器會朗讀 alt 文字）
      注意：<img> 是「自閉合標籤」，不需要 </img>，在標籤結尾加 / 即可
    -->

    <!-- 這是水平線（分隔線） -->
    <hr>
    <!-- 
      <hr> 是「水平線（horizontal rule）」，會畫一條横線
      用於分隔不同主題的內容
      這也是自閉合標籤
    -->

    <!-- 這是換行標籤 -->
    <p>第一行文字<br>第二行文字</p>
    <!-- 
      <br> 是「break（換行）」，強制讓文字換到下一行
      就像在 word 按 Enter 換行
      這也是自閉合標籤
    -->

    <!-- 這是區塊容器 -->
    <div>
      <!-- 
        <div> 是「division（區塊）」，一個「容器」標籤
        它本身沒有特殊意義，主要用來「分組」和「排版」
        就像把相關的傢俱放在同一個箱子里
        div 是 HTML 裡最常用的標籤之一！
      -->
      <h3>我是 div 裡面的標題</h3>
      <p>我是 div 裡面的段落。</p>
    </div>

    <!-- 這是行內容器 -->
    <p>這是普通文字，<span style="color: blue;">這是藍色的文字</span>，這又是普通文字。</p>
    <!-- 
      <span> 是「行內（inline）」容器，與 div 类似但不會強制換行
      通常配合 CSS 使用，改變部分文字的樣式
      就像在句子中用螢光筆畫重點，但不改變整段結構
    -->

    <!-- 這是語義標籤範例 -->
    <header>這是網頁的標題區（header）</header>
    <!-- <header> 表示「網頁的標頭區」，讓瀏覽器和搜尋引擎知道這是標題區 -->
    
    <main>
      <!-- <main> 表示「主要內容區」 -->
      <article>
        <!-- <article> 表示「文章區」，代表獨立的內容區塊 -->
        <h2>文章標題</h2>
        <p>這是一篇文章的內容...</p>
      </article>
    </main>

    <footer>這是網頁的底部（footer）</footer>
    <!-- <footer> 表示「網頁的底部」，通常放著作權資訊、聯絡方式等 -->

  </body>
  <!-- body 結束 -->
</html>
<!-- html 結束 — 整個網頁檔案結束 -->
```

## 3️⃣ 實際操作（一步一步帶著做）

### 步驟 1：在電腦上創建檔案

1. 打開「記事本」（Windows）或「文字編輯」（Mac）
2. 點擊「另存新檔」
3. 檔名輸入：`index.html`
4. 存檔類型選擇：「所有檔案」（不是文字檔 .txt）

### 步驟 2：把上面的程式碼複製進去

把「核心知識」部分的程式碼完整複製，貼上到你的文字編輯器。

### 步驟 3：用瀏覽器打開

1. 找到你存檔的 `index.html`
2. 對著它按兩下
3. 它會用瀏覽器（Chrome/Edge）打開
4. 你就會看到一個「真正的網頁」了！

### 步驟 4：嘗試修改

試著修改以下內容，感受一下 HTML 的運作：

1. 把 `<h1>歡迎來到我的網站</h1>` 改成 `<h1>我的個人履歷</h1>`
2. 把 `<li>數學</li>` 改成 `<li>程式設計</li>`
3. 把 `<a href="https://www.google.com">` 改成 `href="https://www.youtube.com"`（連結到 YouTube）
4. 修改 `<img>` 的 `src` 屬性，換成任何圖片網址

每次修改後存檔，按 F5 刷新瀏覽器，看看變化了什麼！

## 4️⃣ 今日小任務

### ✅ 初級挑戰
1. 建立一個 `index.html` 檔案
2. 加入你的「自我介紹」，包含：
   - 一個 `<h1>` 主標題（你的名字）
   - 一個 `<h2>` 副標題（你的興趣）
   - 兩三個 `<p>` 段落（簡單介紹自己）
   - 一張你的照片（用 `<img>` 標籤）

### ✅ 中級挑戰
1. 新增一個「我最喜歡的東西」列表
2. 用 `<ul>` 和 `<li>` 列出 3-5 個項目
3. 為每個項目加上 `<a>` 連結，連到相關網站

### ✅ 高級挑戰（選做）
1. 嘗試使用 `<table>` 標籤做一個簡單的表格
2. 嘗試使用 `<form>` 標籤建立一個「留言表單」的框架
（不需要會處理表單提交，只要写出 HTML 結構即可）

## 📝 今日重點回顧

| 標籤 | 用途 | 比喻 |
|------|------|------|
| `<h1>` ~ `<h6>` | 標題 | 書的章節名稱 |
| `<p>` | 段落 | 文章的一段 |
| `<strong>` / `<em>` | 強調 | 螢光筆畫重點 |
| `<ul>` / `<ol>` / `<li>` | 列表 | 清單 |
| `<a>` | 連結 | 書籤/門 |
| `<img>` | 圖片 | 牆上的畫 |
| `<div>` / `<span>` | 容器 | 箱子/盒子 |
| `<header>` / `<main>` / `<footer>` | 語義區塊 | 公司的部門 |
