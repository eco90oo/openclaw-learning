## 1️⃣ 基礎概念（用生活比喻，越詳細越好）

### 🎯 什麼是「狀態」（State）？

**狀態 = 會變化的資料**

打個比方：
- 你銀行存款的餘額是「狀態」—— 會因為存錢、花錢而改變
- 遊戲中角色的血量是「狀態」—— 會因為受傷、治療而改變
- 購物車裡的商品數量是「狀態」—— 會因為加入商品、移除商品而改變

### 🏠 比喻：狀態就像「霓虹燈看板」

想像你有一個餐廳的霓虹燈看板：

```
┌─────────────────┐
│   🍕 披薩餐廳    │
│   營業中 / 打烊  │  ← 這會根據時間改變
│   排隊人數: 5    │  ← 這也會一直變化
└─────────────────┘
```

在 React 中：
- **組件** = 這個看板本身
- **狀態** = 「營業中/打烊」、「排隊人數」這些會變的資料
- **setState** = 有人打電話來說「來了新客人」，你去更新看板上的數字

### 📝 什麼是 useState？

`useState` 是 React 的一個「鉤子」（Hook），讓函式組件也能擁有狀態。

就像你的手機有「快捷鍵」一樣，`useState` 就是 React 的快捷鍵，讓你快速設定狀態。

### 🤔 為什麼需要狀態？

沒有狀態的網頁就像：
- 一張靜態的海報 - 永遠不會改變
- 一本印刷好的書 - 內容固定

有狀態的網頁就像：
- 一個會即時更新的股價看板
- 一個可以互動的問卷調查表
- 一個可以購物的購物車

## 2️⃣ 核心知識（每行程式碼都要有中文註解！）

### 📋 useState 的基本語法

```jsx
// 引入 useState 這個鉤子（Hook）
// 就像從工具箱裡拿出你要用的工具
import { useState } from 'react';

// 建立一個「狀態變數」
// 語法：const [變數名, 設定函式] = useState(初始值)
// 
// - count 是「目前的状态值」（就像現在的時間）
// - setCount 是「用來更新狀態的函式」（就像設定鬧鐘）
// - useState(0) 代表初始值是 0（從零開始計數）
const [count, setCount] = useState(0);

// 上面的寫法，等同於傳統的這樣寫：
// const count = state.count;
// const setCount = (newValue) => { state.count = newValue };
// 但 React 幫你封裝好了，更簡單！
```

### 📋 第一個範例：計數器

```jsx
import { useState } from 'react';  // 1. 從 React 引入 useState

// 建立一個計數器組件
function Counter() {
  // 2. 使用 useState 建立狀態
  //    - count 是目前的數字（初始值 0）
  //    - setCount 是用來修改數字的函式
  const [count, setCount] = useState(0);
  
  // 3. 建立一個「加一」的函式
  //    當使用者點擊按鈕時，會呼叫這個函式
  const handleAdd = () => {
    // setCount 的用法：放進新的值
    // 這裡放 count + 1，也就是「現在的值 + 1」
    setCount(count + 1);
  };
  
  // 4. 建立一個「減一」的函式
  const handleSubtract = () => {
    // 同樣的道理，只是變成減 1
    setCount(count - 1);
  };
  
  // 5. 建立一個「重設」的函式
  const handleReset = () => {
    // 設回初始值 0
    setCount(0);
  };
  
  // 6. 回傳 JSX（HTML 結構）
  return (
    <div>
      {/* 顯示目前的 count 值 */}
      {/* 就像看板上的數字，會即時更新 */}
      <h1>計數器：{count}</h1>
      
      {/* 按鈕區域 */}
      <button onClick={handleAdd}>➕ 加一</button>
      {/* 
          onClick 是 React 的事件處理
          等於 HTML 的 onclick
          點擊時會執行 handleAdd 函式
      */}
      
      <button onClick={handleSubtract}>➖ 減一</button>
      
      <button onClick={handleReset}>🔄 重設</button>
    </div>
  );
}

// 匯出這個組件，讓其他檔案可以使用
export default Counter;
```

### 📋 範例二：切換開關（布林值狀態）

```jsx
import { useState } from 'react';

function LightSwitch() {
  // 建立一個布林值（true/false）狀態
  // true = 燈亮，false = 燈暗
  // 初始值設為 false（預設關燈）
  const [isOn, setIsOn] = useState(false);
  
  // 切換函式：不是開就是關
  const toggleLight = () => {
    // 如果本來是 true，就改成 false
    // 如果本來是 false，就改成 true
    // ! 是「否則」运算符，表示「相反」
    setIsOn(!isOn);
  };
  
  return (
    <div>
      {/* 根據 isOn 的值，顯示不同的文字和樣式 */}
      {/* 三元运算符：條件 ? 成立時 : 不成立時 */}
      <h1 style={{ 
        // style 寫在大括號裡面
        // 燈亮時文字是黃色，否則是灰色
        color: isOn ? 'yellow' : 'gray' 
      }}>
        電燈狀態：{isOn ? '💡 開' : '🌑 關'}
      </h1>
      
      {/* 點擊按鈕來切換狀態 */}
      <button onClick={toggleLight}>
        {isOn ? '關閉' : '開啟'}電燈
      </button>
      
      {/* 視覺化展示：用背景色表示燈的開關 */}
      <div style={{
        width: '100px',
        height: '100px',
        borderRadius: '50%',
        // 燈亮時是黃色光，否則是暗灰色
        backgroundColor: isOn ? '#ffeb3b' : '#333',
        // 加一點陰影讓它發光
        boxShadow: isOn ? '0 0 20px #ffeb3b' : 'none',
        // 動畫效果
        transition: 'all 0.3s'
      }}></div>
    </div>
  );
}

export default LightSwitch;
```

### 📋 範例三：表單輸入（字串狀態）

```jsx
import { useState } from 'react';

function NameForm() {
  // 建立一個字串狀態，儲存使用者輸入的名字
  // 初始值是空字串 ''
  const [name, setName] = useState('');
  
  // 處理輸入框變化的函式
  const handleChange = (event) => {
    // event 是 React 傳給我們的「事件物件」
    // event.target 是觸發事件的元素（這個輸入框）
    // event.target.value 是輸入框目前的值
    // 把這個值存進狀態裡
    setName(event.target.value);
  };
  
  // 處理表單提交的函式
  const handleSubmit = (event) {
    // event.preventDefault() 是「阻止預設行為」
    // 表單預設會重新整理頁面，這行讓它不要重整
    event.preventDefault();
    
    // 顯示輸入的名字
    alert(`你好，${name}！`);
  };
  
  return (
    <form onSubmit={handleSubmit}>
      {/* 標題會根據輸入即時更新 */}
      {/* 每當 name 改變，這裡就會重新渲染 */}
      <h1>你好，{name || '訪客'}！</h1>
      {/* || '訪客' 是「如果 name 是空的就顯示 '訪客'」 */}
      
      {/* 輸入框 */}
      {/* onChange 是「當輸入框內容改變」的事件 */}
      {/* 每次打字都會觸發 handleChange，更新狀態 */}
      <input 
        type="text" 
        value={name}           // 輸入框的值來自 name 狀態
        onChange={handleChange} // 改變時更新狀態
        placeholder="請輸入你的名字"
      />
      
      {/* 提交按鈕 */}
      <button type="submit">提交</button>
    </form>
  );
}

export default NameForm;
```

### 📋 範例四：陣列狀態（購物車範例）

```jsx
import { useState } from 'react';

function ShoppingCart() {
  // 購物車是個陣列，裡面放商品
  // 初始值是空的
  const [cart, setCart] = useState([]);
  
  // 新增商品到購物車
  const addItem = (productName) => {
    // 方法一：建立新的陣列，加上新商品
    // ...cart 是「展開運算子」，複製現有的 cart 內容
    const newCart = [...cart, productName];
    setCart(newCart);
    
    // 方法二（更簡潔的寫法）：
    // setCart([...cart, productName]);
  };
  
  // 移除指定索引的商品
  const removeItem = (index) => {
    // filter 是陣列的方法，回傳一個「過濾後的新陣列」
    // (item, i) => i !== index 的意思是：
    // 「如果這個項目的索引不等於要移除的索引，就留下來」
    const newCart = cart.filter((item, i) => i !== index);
    setCart(newCart);
  };
  
  // 清空購物車
  const clearCart = () => {
    setCart([]);  // 設回空陣列
  };
  
  return (
    <div>
      <h1>🛒 購物車</h1>
      
      {/* 顯示購物車裡的商品數量 */}
      <p>商品數量：{cart.length}</p>
      {/* .length 是陣列的屬性，表示有幾個元素 */}
      
      {/* 商品按鈕區 */}
      <div>
        {/* 點擊就新增商品到購物車 */}
        <button onClick={() => addItem('🍎 蘋果')}>➕ 蘋果</button>
        <button onClick={() => addItem('🍌 香蕉')}>➕ 香蕉</button>
        <button onClick={() => addItem('🍇 葡萄')}>➕ 葡萄</button>
        <button onClick={() => addItem('🍊 橘子')}>➕ 橘子</button>
      </div>
      
      {/* 顯示購物車內容 */}
      <ul>
        {/* map 是陣列的方法，用來「遍歷」每個元素
            (item, index) 是「目前的元素」和「索引」
            key 是 React 用来識別每個項目的 unique ID
        */}
        {cart.map((item, index) => (
          // 每個清單項目要有獨特的 key
          // 這裡用 index 作為 key（簡易做法）
          <li key={index}>
            {item}
            {/* 刪除按鈕，傳入要移除的索引 */}
            <button onClick={() => removeItem(index)}>🗑️</button>
          </li>
        ))}
      </ul>
      
      {/* 顯示購物車是空的的情況 */}
      {cart.length === 0 && <p>🛒 購物車是空的</p>}
      {/* && 是「並且」，如果前面 true 才顯示後面 */}
      
      {/* 清空購物車按鈕 */}
      {cart.length > 0 && (
        <button onClick={clearCart}>清空購物車</button>
      )}
    </div>
  );
}

export default ShoppingCart;
```

### 📋 範例五：多個狀態

```jsx
import { useState } from 'react';

function UserProfile() {
  // 你可以建立多個狀態！
  // 每個 useState 都是獨立的狀態
  
  const [name, setName] = useState('');
  const [age, setAge] = useState(0);
  const [city, setCity] = useState('');
  const [isStudent, setIsStudent] = useState(false);
  
  return (
    <div>
      <h1>👤 使用者資料</h1>
      
      {/* 名稱輸入 */}
      <label>名字：</label>
      <input 
        type="text" 
        value={name}
        onChange={(e) => setName(e.target.value)}
      />
      
      {/* 年齡輸入 */}
      <label>年齡：</label>
      <input 
        type="number" 
        value={age}
        onChange={(e) => setAge(Number(e.target.value))}
      />
      {/* Number() 是把字串轉成數字 */}
      
      {/* 城市選擇 */}
      <label>城市：</label>
      <select 
        value={city}
        onChange={(e) => setCity(e.target.value)}
      >
        <option value="">請選擇</option>
        <option value="taipei">台北</option>
        <option value="taichung">台中</option>
        <option value="kaohsiung">高雄</option>
      </select>
      
      {/* 核取方塊（布林值） */}
      <label>
        <input 
          type="checkbox" 
          checked={isStudent}
          onChange={(e) => setIsStudent(e.target.checked)}
        />
        {/* checked 是核取方塊的「是否勾選」屬性 */}
        {/* e.target.checked 是勾選狀態（true/false） */}
        我是學生
      </label>
      
      {/* 顯示所有輸入的資料 */}
      <hr/>
      <h2>📋 資料摘要</h2>
      <p>名字：{name}</p>
      <p>年齡：{age} 歲</p>
      <p>城市：{city}</p>
      <p>身份：{isStudent ? '學生' : '非學生'}</p>
    </div>
  );
}

export default UserProfile;
```

### 📝 步驟 5：理解狀態的變化流程

```
┌─────────────────────────────────────────────────────┐
│                   狀態更新流程                        │
├─────────────────────────────────────────────────────┤
│                                                     │
│  1. 使用者點擊按鈕                                  │
│         ↓                                          │
│  2. 觸發 onClick 事件                               │
│         ↓                                          │
│  3. 執行 handleAdd 函式                             │
│         ↓                                          │
│  4. 呼叫 setCount(新值)                            │
│         ↓                                          │
│  5. React 偵測到狀態改變                            │
│         ↓                                          │
│  6. 重新執行組件（重新渲染）                         │
│         ↓                                          │
│  7. 更新畫面上的數字                                 │
│         ↓                                          │
│  8. 完成！                                          │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 📝 步驟 6：常見錯誤練習

嘗試故意犯錯，然後看錯誤訊息：

```jsx
// 錯誤 1：直接修改狀態（錯誤做法！）
const handleWrong = () => {
  count = count + 1;  // ❌ 這不會觸發重新渲染！
};

// 正確做法是使用 setCount
const handleCorrect = () => {
  setCount(count + 1);  // ✅ 這樣才會更新
};

// 錯誤 2：在 render 之外修改狀態
function MyComponent() {
  const [count, setCount] = useState(0);
  
  // ❌ 錯誤：這不是事件處理函式，會直接執行
  setCount(count + 1);
  
  return <div>{count}</div>;
}

// 正確做法：
function MyComponent() {
  const [count, setCount] = useState(0);
  
  const handleClick = () => {
    // ✅ 在事件處理函式中更新狀態
    setCount(count + 1);
  };
  
  return <button onClick={handleClick}>{count}</button>;
}
```

## 📚 延伸學習資源

### 🔑 重點複習

1. **useState 是函式**：每次渲染都會重新執行
2. **setState 是非同步的**：不會立即更新，要等下一次渲染才會看到變化
3. **狀態更新會觸發重新渲染**：React 會重新執行組件函式
4. **狀態是獨立的**：每個 useState 呼叫都是獨立的狀態
