## 1️⃣ 基礎概念：用「蓋房子」比喻 HTML

### 🎯 什麼是 HTML？

**HTML = HyperText Markup Language**  
翻成白話就是：「**超文字標記語言**」

### 🏠 比喻：HTML 就像房屋的鋼筋骨架

```
┌─────────────────────────────────────┐
│           房屋示意圖                │
├─────────────────────────────────────┤
│  🏠 屋頂（標題）                    │
│     ↓                               │
│  ┌─────────────────────────────┐   │
│  │  客廳（主要內容區塊）         │   │
│  │  ↓                          │   │
│  │  📺 電視機（圖片/影片）       │   │
│  │  🪑 沙發（文字段落）          │   │
│  └─────────────────────────────┘   │
│     ↓                              
│  🚪 大門（連結/按鈕）              
└─────────────────────────────────────┘
```

- **HTML 標籤（Tag）** = 建築工人用的「標記指令」
- 每一個標籤都有特定功能，像是：
  - `<h1>` = 這是最大標題（屋頂）
  - `<p>` = 這是段落文字（房間裡的擺設）
  - `<a>` = 這是連結（門/樓梯）

### 📝 重點提醒

> HTML 不是程式語言，它是「標記語言」——就像在文件上寫「這是標題、這是重點」，告訴瀏覽器該如何呈現內容。

## 2️⃣ 核心知識：常見 HTML 標籤大全

### 完整程式碼（複製貼上就能用）

```html
<!DOCTYPE html>
<!-- ↑ 這一行告訴瀏覽器：「這是一個 HTML5 網頁」 -->
<html lang="zh-TW">
<!-- ↑ 網頁的根標籤，lang="zh-TW" 表示語言是繁體中文 -->

<head>
    <!-- ↑ head 是「頭部」，放網頁的設定資訊 -->
    
    <meta charset="UTF-8">
    <!-- ↑ meta 是「元資料」，charset="UTF-8" 設定文字編碼，讓中文正常顯示 -->
    
    <title>我的第一個網頁</title>
    <!-- ↑ title 是「標題」，會顯示在瀏覽器分頁上 -->
    
</head>
<!-- ↑ 結束頭部設定區塊 -->

<body>
    <!-- ↑ body 是「身體」，放我們實際看到的網頁內容 -->

    <!-- ========== 標題區塊 ========== -->
    <h1>我是最大標題</h1>
    <!-- <h1> 是 Heading 1，最大的一級標題，一個頁面通常只有一個 -->
    <h2>我是二級標題</h2>
    <!-- <h2> 是二級標題，通常用於章節名稱 -->
    <h3>我是三級標題</h3>
    <!-- <h3> 是三級標題，用於小節標題 -->
    <!-- HTML 支援 h1 到 h6，數字越大字越小 -->

    <!-- ========== 段落文字 ========== -->
    <p>這是一個段落，裡面可以放很多文字。</p>
    <!-- <p> 是 Paragraph，代表一個段落 -->
    <p>瀏覽器會自動幫段落加上行距，讓文字更容易閱讀。</p>

    <!-- ========== 換行（不會產生新段落） ========== -->
    第一行文字<br>
    <!-- <br> 是 Break，代表「換行」，注意它是單獨的標籤，沒有結束標籤 -->
    第二行文字（接在下面）

    <!-- ========== 水平分隔線 ========== -->
    <hr>
    <!-- <hr> 是 Horizontal Rule，畫一條水平線分隔內容 -->

    <!-- ========== 連結（超連結） ========== -->
    <a href="https://www.google.com">點我連結到 Google</a>
    <!-- <a> 是 Anchor（錨點），href="網址" 設定連結目的地 -->
    <!-- href = Hypertext REFerence（超文字引用） -->

    <br><br>

    <!-- ========== 圖片 ========== -->
    <img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools 圖片">
    <!-- <img> 是 Image（圖片），沒有結束標籤 -->
    <!-- src = Source（來源），放圖片網址 -->
    <!-- alt = Alternative（替代文字），圖片無法顯示時會顯示這段文字 -->

    <!-- ========== 無序列表（ bullet points） ========== -->
    <ul>
        <!-- <ul> 是 Unordered List（無序列表），用圓點標記項目 -->
        <li>第一個項目</li>
        <!-- <li> 是 List Item（列表項目） -->
        <li>第二個項目</li>
        <li>第三個項目</li>
    </ul>

    <!-- ========== 有序列表（編號） ========== -->
    <ol>
        <!-- <ol> 是 Ordered List（有序列表），用數字標記項目 -->
        <li>第一步：開啟電腦</li>
        <li>第二步：打開瀏覽器</li>
        <li>第三步：開始學習</li>
    </ol>

    <!-- =========> 按鈕 ========== -->
    <button>我是按鈕</button>
    <!-- <button> 是按鈕，點擊後可以觸發 JavaScript 動作 -->

    <!-- =========> 強調文字 ========== -->
    <p>這是<strong>粗體文字</strong>，用於強調重要內容。</p>
    <!-- <strong> 會讓文字變粗，表示語意上的強調 -->
    <p>這是<em>斜體文字</em>，用於輕微強調。</p>
    <!-- <em> 會讓文字變斜，表示語意上的強調 -->

</body>
<!-- ↑ 結束網頁內容區塊 -->

</html>
<!-- ↑ 結束整個 HTML 文件 -->
```

### 📋 常見標籤速查表

| 標籤 | 功能說明 | 比喻 |
|------|----------|------|
| `<h1>`~`<h6>` | 標題 | 書籍的章節標題 |
| `<p>` | 段落 | 文章的一段文字 |
| `<a href="...">` | 連結 | 網頁的「門」 |
| `<img src="...">` | 圖片 | 牆上的畫 |
| `<ul>`/`<ol>`/`<li>` | 列表 | 清單、菜單 |
| `<button>` | 按鈕 | 門鈴、開關 |
| `<strong>`/`<em>` | 強調 | 螢光筆畫的重點 |
| `<br>` | 換行 | 按Enter換行 |
| `<hr>` | 分隔線 | 書本的分隔頁 |

## 3️⃣ 實際操作：一步步動手做

### Step 1：建立你的第一個 HTML 檔案

1. 在電腦上新增一個資料夾，名稱叫 `my-first-website`
2. 在資料夾內新增一個檔案，命名為 `index.html`
3. 將上面的程式碼完整複製貼上進去
4. 存檔

### Step 2：用瀏覽器打開看看

1. 對著 `index.html` 檔案按右鍵
2. 選擇「用 Chrome/Edge/Safari 開啟」
3. 你會看到一個完整的網頁！

### Step 3：動手修改體驗一下

修改以下內容，感受 HTML 的運作：

```html
<!-- 修改這行，把標題改成你的名字 -->
<h1>我的第一個網頁</h1>

<!-- 修改這行，改成你想說的話 -->
<p>今天是我學習 HTML 的第一天！</p>

<!-- 修改這行，把連結改成你喜歡的網站 -->
<a href="https://www.youtube.com">點我去看 YouTube</a>
```

## 💡 小技巧

### 為什麼一個 HTML 檔案就能顯示網頁？

因為瀏覽器（Chrome、Edge、Safari）內建了一個「HTML 翻譯機」：
```
人類寫的 HTML → 瀏覽器翻譯 → 螢幕上的網頁
```

這就是為什麼我們不需要安裝任何東西，只要寫 HTML，瀏覽器就能幫我們呈現！

### 常見錯誤提醒

❌ 漏寫結束標籤（除了 `<br>`、`<img>`、`<hr>` 這種單標籤）  
❌ 中英文符號混淆（HTML 只接受英文字符）  
❌ 忘記存檔就按重新整理
