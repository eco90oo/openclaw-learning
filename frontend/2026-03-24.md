## 1️⃣ 基礎概念（用生活比喻，越詳細越好）

### 🎯 什麼是 HTML？

**HTML = 超文本標記語言**（HyperText Markup Language）

### 🏠 比喻：HTML 就像是房子的骨架

想像你要蓋一間房子：
- **HTML** = 房子的鋼筋水泥骨架（決定結構）
- **CSS** = 油漆、磁磚、窗簾（決定外觀）
- **JavaScript** = 電器、開關、水龍頭（決定功能）

沒有骨架，房子無法站立；沒有 HTML，網頁就沒有結構。

### 📝 什麼是「標籤」（Tag）？

標籤就像是你寫信時的「信封格式」：

```
┌─────────────────────────────────┐
│  親愛的 XXX：                   │  ← 開頭標籤 <tag>
│                                  │
│  內容在這裡...                   │  ← 內文
│                                  │
│  祝好                             │  ← 結尾標籤 </tag>
└─────────────────────────────────┘
```

標籤的規則：
- 大部分標籤需要「開頭」和「結尾」
- 開頭：`<標籤名>`
- 結尾：`</標籤名>`
- 結尾比開頭多一個 `/`

## 2️⃣ 核心知識（每行程式碼都要有中文註解！）

### 📋 HTML 文件的基本結構

```html
<!DOCTYPE html>
<!-- 
  ↑ 這一行告訴瀏覽器：「這是一個 HTML5 文件」
  ↑ 就像在書本封面寫「這是一本小說」
  ↑ 沒有這行，瀏覽器可能會用舊版方式解釋網頁
-->

<html lang="zh-TW">
<!-- 
  ↑ <html> 是網頁的根標籤，所有內容都要包在裡面
  ↑ lang="zh-TW" 告訴搜尋引擎這是繁體中文網頁
  ↑ 就像是護照上註明你的國籍
-->

<head>
  <!-- 
    <head> 是網頁的「檔頭」
    ↑ 這裡放的是「關於網頁」的資訊，不是畫面上直接顯示的內容
    ↑ 就像一本書的「版權頁」和「目錄」
  -->
  
  <meta charset="UTF-8">
  <!-- 
    ↑ meta 標籤是「關於資料的資料」（metadata）
    ↑ charset="UTF-8" 設定網頁使用的文字編碼
    ↑ UTF-8 支援幾乎所有國家的文字，包含中文！
    ↑ 沒有這行，中文可能會變成亂碼（如「」）
  -->
  
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <!-- 
    ↑ viewport 是「視窗」的意思
    ↑ 這行是為了讓網頁在手機上也能正常顯示
    ↑ width=device-width 表示「寬度跟隨設備」
    ↑ initial-scale=1.0 表示「初始縮放比例為 100%」
    ↑ 沒有這行，手機看網頁會被縮小，很難閱讀
  -->
  
  <title>我的第一個網頁</title>
  <!-- 
    ↑ <title> 設定瀏覽器標籤上的文字
    ↑ 這會顯示在 Chrome/Edge 標籤列上
    ↑ 也是 Google 搜尋結果顯示的標題
  -->
  
</head>
<!-- head 標籤結束 -->

<body>
  <!-- 
    <body> 是網頁的「主體」
    ↑ 所有你在網頁上看到的內容都放在這裡
    ↑ 就像書本的「正文」部分
  -->
  
  <!-- 👇 這裡開始放網頁的實際內容 👇 -->
  
  <h1>歡迎來到我的網頁！</h1>
  <!-- 
    ↑ <h1> 是「標題 1」（Heading 1）
    ↑ h1 是最重要的標題，通常用於主標題
    ↑ 數字越大，重要性越低：h1 > h2 > h3 > h4 > h5 > h6
    ↑ 就像是書籍的章節標題：第一章 > 第一節 > 第一小節
  -->
  
  <p>這是我的第一個網頁，我正在學習 HTML！</p>
  <!-- 
    ↑ <p> 是「段落」（Paragraph）
    ↑ 用於包住一段文字
    ↑ 瀏覽器會自動在段落之間留空行
  -->

  <h2>我的興趣</h2>
  <!-- ↑ <h2> 是「二級標題」，比 h1 小一點 -->
  
  <ul>
    <!-- ↑ <ul> 是「無序列表」（Unordered List）
        ↑ 前面會顯示圓點或黑點，不會編號
    -->
    <li>看書</li>
    <!-- ↑ <li> 是「列表項目」（List Item）
        ↑ 每個項目都用 <li> 包住
    -->
    <li>運動</li>
    <li>寫程式</li>
  </ul>
  <!-- ↑ </ul> 結束無序列表 -->

  <h2>我喜歡的水果</h2>
  <!-- ↑ 另一個二級標題 -->
  
  <ol>
    <!-- ↑ <ol> 是「有序列表」（Ordered List）
        ↑ 會自動顯示 1. 2. 3. 的編號
    -->
    <li>蘋果</li>
    <li>香蕉</li>
    <li>橘子</li>
  </ol>
  <!-- ↑ </ol> 結束有序列表 -->
  
  <hr>
  <!-- ↑ <hr> 是「水平線」（Horizontal Rule）
      ↑ 會畫一條分隔線
      ↑ 注意：這是「自結束標籤」，沒有 </hr>
      ↑ 有些標籤不需要結尾標籤，我們稱為「空標籤」
  -->
  
  <h2>關於我</h2>
  <!-- ↑ 又一個二級標題 -->
  
  <!-- 粗體和斜體的用法 -->
  <p>
    我叫 <strong>小明</strong>，今年 <em>20 歲</em>。
    <!-- 
        ↑ <strong> 表示「強調」，通常會變成粗體
        ↑ <em> 表示「斜體強調」
        ↑ 這兩個標籤除了改變外觀，還有語義上的意義
        ↑ 螢幕閱讀器會特別強調這些詞
    -->
  </p>
  
  <!-- 連結標籤 -->
  <p>
    歡迎訪問我的 
    <a href="https://www.google.com">Google</a>！
    <!-- 
        ↑ <a> 是「錨點」（Anchor）標籤，用於建立連結
        ↑ href="網址" 指定要連結到哪里
        ↑ 點擊文字就會跳轉到 href 指定的網頁
        ↑ 「超連結」就是這樣來的
    -->
  </p>
  
  <!-- 圖片標籤 -->
  <p>
    這是一張範例圖片：<br>
    <!-- ↑ <br> 是「換行」（Break）
        ↑ 會強制換行，不像 <p> 會自動留空行
        ↑ 也是自結束標籤
    -->
    <img src="https://via.placeholder.com/300" alt="範例圖片">
    <!-- 
        ↑ <img> 是「圖片」（Image）標籤
        ↑ src="網址" 指定圖片的位置（可以是本地檔案或網路上的圖片）
        ↑ alt="文字" 是「替代文字」
           - 圖片載入失敗時會顯示這段文字
           - 視障人士使用的螢幕閱讀器會讀出這段文字
           - SEO 也會用到這段文字
        ↑ 這也是自結束標籤
    -->
  </p>
  
  <!-- 表格標籤 -->
  <h2>我的學習進度</h2>
  <!-- ↑ 另一個標題 -->
  
  <table border="1">
    <!-- ↑ <table> 是「表格」標籤
        ↑ border="1" 表示边框宽度为 1（為了讓大家看到格子）
    -->
    <tr>
      <!-- ↑ <tr> 是「表格列」（Table Row）-->
      <th>科目</th>
      <!-- ↑ <th> 是「表格標題格」（Table Header）
          ↑ 通常會自動變粗體並置中
      -->
      <th>進度</th>
    </tr>
    <tr>
      <td>HTML</td>
      <!-- ↑ <td> 是「表格資料格」（Table Data） -->
      <td>✅ 完成</td>
    </tr>
    <tr>
      <td>CSS</td>
      <td>📚 學習中</td>
    </tr>
    <tr>
      <td>JavaScript</td>
      <td>⏳ 待開始</td>
    </tr>
  </table>
  <!-- ↑ </table> 結束表格 -->
  
  <!-- 語義化標籤（HTML5 新增） -->
  <header>
    <!-- ↑ <header> 表示「頁首」或「區塊標題」 -->
    <h1>這是網頁的標題區塊</h1>
  </header>
  
  <nav>
    <!-- ↑ <nav> 表示「導航選單」 -->
    <a href="#">首頁</a> | 
    <a href="#">關於</a> | 
    <a href="#">聯絡</a>
  </nav>
  
  <main>
    <!-- ↑ <main> 表示「主要內容區域」 -->
    <article>
      <!-- ↑ <article> 表示「文章」內容 -->
      <h2>文章標題</h2>
      <p>這是文章的主要內容...</p>
    </article>
    
    <aside>
      <!-- ↑ <aside> 表示「側邊欄」或「附屬內容」 -->
      <h3>側邊資訊</h3>
      <p>這裡可以放相關連結或廣告</p>
    </aside>
  </main>
  
  <footer>
    <!-- ↑ <footer> 表示「頁尾」 -->
    <p>&copy; 2026 我的網站。保留所有權利。</p>
    <!-- 
        ↑ &copy; 是「HTML 實體」（Entity）
        ↑ 用於顯示特殊符號，因為 < > & 在 HTML 有特殊意義
        ↑ 其他常用實體：
          - &lt;   = < 小於符號
          - &gt;   = > 大於符號
          - &amp;  = & 和號符號
          - &nbsp; = 不斷行空格
    -->
  </footer>
  
</body>
<!-- body 標籤結束 -->

</html>
<!-- html 標籤結束，整個網頁檔案結束 -->
```

## 3️⃣ 實際操作（一步一步帶著做）

### 📝 步驟 1：建立第一個 HTML 檔案

1. 打開 VS Code（或任何文字編輯器）
2. 新增一個檔案，命名為 `index.html`
3. 複製上面的程式碼貼進去

### 📝 步驟 2：在瀏覽器中查看

1. 在 VS Code 中，按右鍵
2. 選擇「Open with Live Server」（或直接用瀏覽器打開檔案）
3. 你會看到一個完整的網頁！

### 📝 步驟 3：嘗試修改內容

試著修改以下內容，感受 HTML 的運作：

### 📝 步驟 4：理解標籤的巢狀結構

```
<html>
  └─ <head>
  └─ <body>
       ├─ <h1>
       ├─ <p>
       ├─ <ul>
       │    └─ <li> × 3
       ├─ <table>
       │    ├─ <tr> (標題列)
       │    └─ <tr> (資料列) × 3
       └─ <footer>
```

**重要規則**：標籤要正確「包覆」，不能交叉！

✅ 正確：
```html
<p>這是<strong>重要</strong>文字</p>
```

❌ 錯誤：
```html
<p>這是<strong>重要</p></strong>
```
