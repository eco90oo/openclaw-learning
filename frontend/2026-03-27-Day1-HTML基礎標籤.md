## 1️⃣ 基礎概念（用生活比喻，越詳細越好）

### 🎯 什麼是 HTML？

**比喻時間**：HTML 就像是一棟房子的「骨架」和「結構」。

想象一下你要蓋一間房子：
- **HTML** = 房子的鋼筋、水泥牆、門框、窗戶框架
- **CSS** = 油漆、瓷磚、窗簾、裝飾（讓房子變漂亮）
- **JavaScript** = 房子的電器設備、燈光、門鎖（讓房子有功能）

沒有 HTML，房子就只是一堆鋼筋；有了 HTML，房子才有了「形狀」。

### 🎯 標籤（Tag）是什麼？

**比喻時間**：標籤就像是「指示牌」或「標籤貼紙」。

想象你有很多箱東西要寄送：
- 在箱子上貼「易碎物品」的標籤 = `<p>`（段落標籤）
- 在箱子上貼「書籍」的標籤 = `<h1>`（標題標籤）
- 在箱子上貼「聯絡方式」的標籤 = `<a>`（連結標籤）

HTML 標籤就是這樣的「指示牌」，告訴瀏覽器這段文字是什麼類型的內容。

### 🎯 標籤的組成：開頭標籤 + 內容 + 結尾標籤

```
<標籤名稱>內容</標籤名稱>
```

**比喻時間**：這就像是一個「三明治」🍔

```
<麵包>（開頭標籤）
   生菜、漢堡肉、番茄（內容）
<麵包>（結尾標籤）
```

中間的「內容」被夾在兩個標籤中間，瀏覽器就知道這是什麼類型的內容了。

## 2️⃣ 核心知識（每行程式碼都要有中文註解！）

### 讓我們先看一個最基本的 HTML 檔案：

```html
<!DOCTYPE html>
<!-- 這一行是「文件類型宣告」，告訴瀏覽器這個檔案是 HTML5 網頁 -->
<!-- 就像是你告訴快遞員：「這是一個包裹，請用寄包裹的方式處理」 -->

<html>
<!-- 這是「根標籤」，所有的 HTML 內容都要包在這個標籤裡面 -->
<!-- 就像是一個最大的盒子，所有的東西都要放進這個盒子裡 -->

    <head>
    <!-- 這是「 head 區塊」，放的是「看不到」但很重要的資訊 -->
    <!-- 就像是房子的「地契」，裡面寫的是關於這間房子的基本資料 -->
        
        <meta charset="UTF-8">
        <!-- 這一行設定「字元編碼」，讓網頁能正確顯示中文 -->
        <!-- 就像是你告訴翻譯員：「這份文件用的是中文，請用中文解讀」 -->
        
        <title>我的第一個網頁</title>
        <!-- 這一行設定「網頁標題」，會顯示在瀏覽器的分頁上 -->
        <!-- 就像是給這間房子取個名字：「小明的心愛小屋」 -->
        
    </head>
    <!-- head 區塊結束 -->

    <body>
    <!-- 這是「 body 區塊」，放的是「看得到」的內容 -->
    <!-- 就像是房子的「客廳、臥室、廚房」，是實際可以住的空間 -->
        
        <!-- 這裡放的是我們網頁要顯示的內容 -->
        
    </body>
    <!-- body 區塊結束 -->

</html>
<!-- html 區塊結束，這個網頁檔案完成了！ -->
```

### 常見的 HTML 標籤（每天學幾個，慢慢來）：

#### 🔤 標題標籤（Headings）- 就像書籍的章節名稱

```html
<!-- h1 是「最大」的標題，通常用於網頁的主標題 -->
<h1>我是第一標題</h1>

<!-- h2 是「第二大的」標題，用於次要標題 -->
<h2>我是第二標題</h2>

<!-- h3 更小，用於更細的章節 -->
<h3>我是第三標題</h3>

<!-- h4、h5、h6 可以繼續細分，但通常不太常用到這麼深 -->
<h4>我是第四標題</h4>
```

**比喻**：就像一本書
- h1 = 第一章（最大的章節名稱）
- h2 = 1-1 節（章節內的小節）
- h3 = 1-1-1 小小節（更細的內容）

#### 📝 段落標籤（Paragraph）- 普通的文字區塊

```html
<!-- p 是「段落」標籤，用於顯示一般的文字內容 -->
<p>這是一段普通的文字。</p>
<p>這是另一個段落，會自動跟上面那段分開。</p>
```

**比喻**：就像word文件裡的「ENTER」換行一樣，p標籤會自動在前後留空白。

#### 🔗 連結標籤（Anchor）- 可以點擊的文字

```html
<!-- a 是「連結」標籤，href 是「要連結到哪裡」 -->
<!-- 點擊這個文字，會跳轉到 Google 網站 -->
<a href="https://www.google.com">點擊我去 Google</a>
```

**比喻**：就像是商店的「指標牌」，寫著「往右轉 → 超商」，指引你去另一個地方。

#### 🖼️ 圖片標籤（Image）- 顯示圖片

```html
<!-- img 是「圖片」標籤，src 是「圖片在哪裡」 -->
<!-- 這裡使用網路上的圖片當範例 -->
<img src="https://www.google.com/logo.png" alt="Google 標誌">

<!-- 註解： -->
<!-- src = "source"（來源），圖片檔案的路徑或網址 -->
<!-- alt = "alternative"（替代文字），當圖片載入失敗時，會顯示這段文字 -->
```

**比喻**：就像是掛在牆上的一幅畫，src 告訴你畫放在哪裡，alt 是如果畫掉了別人還知道這是什麼。

#### 📋 列表標籤（Lists）- 條列式內容

```html
<!-- ul 是「無序列表」（unordered list），前面會顯示圓點 -->
<ul>
    <li>第一個項目</li>
    <li>第二個項目</li>
    <li>第三個項目</li>
</ul>

<!-- ol 是「有序列表」（ordered list），前面會顯示數字 -->
<ol>
    <li>第一步：打開電腦</li>
    <li>第二步：開啟瀏覽器</li>
    <li>第三步：輸入網址</li>
</ol>

<!-- li 是「列表項目」（list item），每一個項目都要用 li 包起來 -->
```

**比喻**：
- ul = 購物清單（不考慮順序：「買牛奶、買麵包、買雞蛋」）
- ol = 食譜步驟（「第一步：打蛋，第二步：攪拌，第三步：下鍋」）

#### 📦 容器標籤（Div）- 分區塊

```html
<!-- div 是「區塊」標籤，用來把相關的內容包在一起 -->
<!-- div 本身沒有特殊意義，但搭配 CSS 可以做很多效果 -->
<div>
    <h2>我是標題</h2>
    <p>我在同一個區塊裡面</p>
</div>
```

**比喻**：就像是把很多東西放進一個箱子裡，然後可以在箱子外面貼標籤、畫花紋。

#### 📝 輸入框標籤（Input）- 使用者可以輸入文字

```html
<!-- input 是「輸入框」標籤，type 可以是文字、密碼、Email 等 -->
<input type="text" placeholder="請輸入你的名字">
<!-- type="text" = 這是一個「文字」輸入框 -->
<!-- placeholder = 灰色提示文字，還沒輸入時顯示的內容 -->

<input type="password" placeholder="請輸入密碼">
<!-- type="password" = 密碼輸入框，輸入會顯示圓點●●●● -->

<input type="email" placeholder="請輸入 Email">
<!-- type="email" = Email 輸入框，會驗證格式是否正確 -->
```

**比喻**：就像是一張表格上的空白欄位，讓你填寫資料。

#### 📝 按鈕標籤（Button）- 可以點擊的按鈕

```html
<!-- button 是「按鈕」標籤，點擊後會觸發某些動作 -->
<button>按我一下</button>

<!-- 按鈕也可以放在表單裡面 -->
<button type="submit">送出表單</button>
```

**比喻**：就像是門鈴，碰一下就會觸發某些事情（門鈴響起）。

## 3️⃣ 實際操作（一步一步帶著做）

### 步驟 1️⃣：建立你的第一個 HTML 檔案

1. 在電腦上新增一個資料夾，取名為 `my-first-website`（我的第一個網站）
2. 在這個資料夾裡面，新增一個檔案，命名為 `index.html`

> 💡 **小技巧**：副檔名一定要是 `.html` 喔！這樣瀏覽器才會知道這是 HTML 檔案。

### 步驟 2️⃣：寫入第一個網頁內容

用記事本（或任何文字編輯器）打開 `index.html`，輸入以下內容：

```html
<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>我的第一個網頁</title>
    </head>
    <body>
        <!-- 這裡開始是我們的內容 -->
        
        <!-- 第一個標題 -->
        <h1>歡迎來到我的網站！</h1>
        
        <!-- 一個段落 -->
        <p>你好！很高興你來到這裡。這是我的第一個網頁。</p>
        
        <!-- 一個連結 -->
        <a href="https://www.youtube.com">點擊去 YouTube</a>
        
        <!-- 換行 -->
        <br><br>
        
        <!-- 我的作品列表 -->
        <h2>我的作品：</h2>
        <ul>
            <li>作品一：天氣App</li>
            <li>作品二：計算機</li>
            <li>作品三：待辦清單</li>
        </ul>
        
    </body>
</html>
```

### 步驟 3️⃣：在瀏覽器中打開看看

1. 找到你建立的 `index.html` 檔案
2. 對著檔案按兩下（或按右鍵 → 用瀏覽器打開）
3. 你應該會看到一個網頁出現在瀏覽器中！

> 🎉 **恭喜！你已經建立了你的第一個網頁！**

### 步驟 4️⃣：嘗試加入圖片

在 `<body>` 標籤裡面，加入一張圖片：

```html
<!-- 在標題下面加入圖片 -->
<h1>這是一張可愛的貓咪照片：</h1>
<img src="https://placekitten.com/300/200" alt="可愛的貓咪">
```

刷新瀏覽器，你應該會看到一張貓咪圖片！

### 步驟 5️⃣：嘗試加入表單

```html
<!-- 加入一個簡單的表單 -->
<h2>聯絡我</h2>
<input type="text" placeholder="你的名字">
<br>
<input type="email" placeholder="你的 Email">
<br>
<button>送出</button>
```

## 4️⃣ 今日小任務

### 🎯 挑戰題：建立「自我介紹網頁」

請利用今天學到的標籤，建立一個自我介紹網頁，包含：

1. ✅ 一個大標題（h1）：你的名字
2. ✅ 一個次要標籤（h2）：「關於我」
3. ✅ 一個段落（p）：簡短的自我介紹（50字以內）
4. ✅ 一個連結（a）：連結到你喜歡的網站
5. ✅ 一張圖片（img）：放一張你喜歡的圖片（可以用網路上的圖片）
6. ✅ 一個列表（ul 或 ol）：列出你的興趣（至少3項）
7. ✅ 一個輸入框（input）：讓別人可以輸入名字
8. ✅ 一個按鈕（button）：寫上「提交」

### 📁 檔案命名
請將檔案儲存為 `about-me.html`

### 🔍 預期結果
完成後，用瀏覽器打開，應該會看到一個完整的個人介紹網頁！

## 📝 今日重點回顧

| 標籤 | 用途 | 比喻 |
|------|------|------|
| `<h1>` ~ `<h6>` | 標題 | 書的章節名稱 |
| `<p>` | 段落 | 文章的一段文字 |
| `<a>` | 連結 | 指標牌，指引去處 |
| `<img>` | 圖片 | 掛在牆上的畫 |
| `<ul>` / `<ol>` | 列表 | 購物清單 / 食譜步驟 |
| `<div>` | 區塊 | 裝東西的箱子 |
| `<input>` | 輸入框 | 表格的空白欄位 |
| `<button>` | 按鈕 | 門鈴 |
