## 1️⃣ 基礎概念（用生活比喻，越詳細越好）

### 🎯 什麼是 CSS？

**比喻時間**：CSS 就像是幫 HTML「穿衣服」和「化妝」。

想象你有一個「裸」的芭比娃娃（這就是 HTML）：
- **HTML** = 芭比娃娃的身體（骨架、身形）
- **CSS** = 漂亮的衣服、髮型、口紅、鞋子
- **JavaScript** = 芭比娃娃會說話、會動（賦予功能）

如果只有 HTML，網頁就像個裸的芭比娃娃；加上 CSS 後，網頁就變得漂漂亮亮的了！

### 🎯 CSS 的全名是什麼？

**C**ascading **S**tyle **S**heets
- **Cascading** = 「瀑布式」的意思，意思是樣式會「流」下去
- **Style** = 樣式（顏色、大小、位置等）
- **Sheets** = 表格/單子（可以想像成一本「樣式目錄」）

### 🎯 為什麼要學 CSS？

有了 CSS，我們可以：
- 🎨 讓文字變成任何顏色
- 📏 調整任何元素的大小
- 📍 決定元素放在哪個位置
- 🖼️ 加上邊框、陰影、圓角
- 📱 讓網頁在手機和電腦上都能正常顯示

### 🎯 CSS 怎麼運作的？（三種方式）

**比喻時間**：就像「穿衣服」有三種方法：

#### 方法一：行內樣式（Inline Style）- 直接穿在身上
```html
<!-- 就像直接在芭比娃娃身上纏一條絲巾 -->
<p style="color: red;">紅色的文字</p>
```

#### 方法二：內部樣式（Internal Style）- 寫在衣服裡
```html
<!-- 就像在衣服內側縫一個標籤 -->
<head>
    <style>
        p { color: blue; }
    </style>
</head>
```

#### 方法三：外部樣式（External Style）- 放在衣櫃裡
```html
<!-- 就像把衣服收在衣櫃裡，需要時拿出來穿 -->
<head>
    <link rel="stylesheet" href="style.css">
</head>
```

## 2️⃣ 核心知識（每行程式碼都要有中文註解！）

### 第一個 CSS 範例：幫文字穿衣服

```css
/* 這是一個 CSS 註解（不會被執行）*/
/* 這個語法的意思是：「找到所有的 p 標籤，幫它們穿上藍色的衣服」 */

p {
    /* color = 文字的顏色 */
    color: blue;           /* 文字變成藍色 */
    
    /* font-size = 文字的大小 */
    font-size: 20px;      /* 文字大小是 20 像素 */
    
    /* font-family = 字體（文字的長相） */
    font-family: "微軟正黑體", Arial, sans-serif;
}
```

### CSS 的基本語法結構：

```css
/* 
 selector（選擇器）= 要挑選誰來穿衣服
*/ 

h1 {                          /* h1 = 選擇器，選擇所有的 h1 標籤 */
    color: red;               /* 屬性：值； */
    font-size: 30px;         /* 屬性：值； */
}

/* 
 就像是對 h1 說：
 「嘿！所有的 h1 標題，你們聽好了：
  - 把文字變成紅色
  - 把字體大小改成 30 像素」
*/
```

### 常見的 CSS 屬性（每天學幾個，越來越厲害）：

#### 🎨 文字相關屬性

```css
/* 文字顏色 */
p {
    color: #333333;           /* 文字顏色：深灰色（用 16 進位表示） */
    /* 也可以用顏色名稱 */
    color: red;
    /* 或 rgb 函式 */
    color: rgb(255, 0, 0);    /* 紅色：R=255, G=0, B=0 */
}

/* 文字大小 */
h1 {
    font-size: 32px;          /* 32 像素（1px = 1 個螢幕點） */
}

/* 文字粗細 */
p {
    font-weight: bold;        /* 粗體 */
    font-weight: normal;     /* 正常 */
    font-weight: 600;        /* 數字：100~900，數字越大越粗 */
}

/* 文字對齊 */
h1 {
    text-align: center;       /* 居中對齊 */
    text-align: left;         /* 靠左對齊（預設） */
    text-align: right;       /* 靠右對齊 */
    text-align: justify;     /* 兩端對齊 */
}
```

#### 📏 背景相關屬性

```css
/* 背景顏色 */
div {
    background-color: #f0f0f0;    /* 背景：淺灰色 */
    background-color: pink;       /* 背景：粉紅色 */
    background-color: rgba(0, 0, 0, 0.5);  /* 半透明黑色 */
}

/* 背景圖片 */
div {
    background-image: url("https://example.com/image.jpg");
    /* 
     background-image = 背景圖片
     url() = 圖片的路徑（括號內放圖片網址或檔名）
    */
}
```

#### 📦 盒子模型（Box Model）- 最重要的概念！

**比喻時間**：想象有一個快遞包裹：

```
┌─────────────────────────────┐
│       margin（外邊距）        │  ← 包裹和包裹之間的距離
│  ┌─────────────────────────┐ │
│  │   border（邊框）         │ │  ← 包裹的紙箱厚度
│  │  ┌───────────────────┐  │ │
│  │  │  padding（內邊距） │  │ │  ← 填充物的厚度
│  │  │  ┌─────────────┐  │  │ │
│  │  │  │  content   │  │  │ │  ← 實際的內容物
│  │  │  │（內容本身）  │  │  │ │
│  │  │  └─────────────┘  │  │ │
│  │  └───────────────────┘  │ │
│  └─────────────────────────┘ │
└─────────────────────────────┘
```

```css
.box {
    /* margin = 外邊距（這個盒子外面的空間） */
    margin: 20px;              /* 上下左右都是 20px */
    margin-top: 10px;          /* 只設定上面的外邊距 */
    margin-right: 15px;        /* 只設定右邊的外邊距 */
    margin-bottom: 10px;       /* 只設定下面的外邊距 */
    margin-left: 15px;         /* 只設定左邊的外邊距 */
    
    /* 也可以用簡寫（順序：上、右、下、左） */
    margin: 10px 15px 10px 15px;
    /* 等同於 */
    margin: 10px 15px;         /* 上下 10px，左右 15px */
    
    /* border = 邊框 */
    border: 2px solid black;  /* 寬度 2px，實線，黑色 */
    border-width: 3px;        /* 邊框寬度 */
    border-style: dashed;     /* 虛線邊框 */
    border-style: dotted;     /* 點線邊框 */
    border-color: blue;        /* 邊框顏色 */
    
    /* padding = 內邊距（內容和邊框之間的距離） */
    padding: 20px;            /* 上下左右都是 20px */
    padding-top: 10px;        /* 只設定上面的內邊距 */
    
    /* width = 寬度 */
    width: 300px;             /* 寬度 300 像素 */
    width: 50%;               /* 寬度是父元素的 50% */
    
    /* height = 高度 */
    height: 200px;            /* 高度 200 像素 */
    height: auto;             /* 自動（讓內容決定高度） */
}
```

#### 🔗 連結（a 標籤）的特殊樣式

```css
/* 連結有四種狀態，就像人有四種心情 */

/* 1. 連結還沒被點擊過的樣子 */
a:link {
    color: blue;              /* 藍色 */
    text-decoration: none;    /* 沒有底線 */
}

/* 2. 連結被點擊過的樣子 */
a:visited {
    color: purple;            /* 紫色（表示去過了） */
}

/* 3. 連結滑鼠移上去的樣子 */
a:hover {
    color: red;                /* 變成紅色 */
    text-decoration: underline;/* 出現底線 */
}

/* 4. 連結正在被點擊的樣子 */
a:active {
    color: orange;             /* 橘色 */
}
```

#### 🎯 類別選擇器（Class Selector）- 給特定元素穿衣服

**比喻時間**：就像全班同學穿制服，只有「衛生股長」戴不一樣的帽子。

```html
<!-- 我們先在 HTML 裡給元素取個「名字」（class） -->
<p class="important">這是很重要的文字</p>
<p>這是普通文字</p>
<p class="important">這也是很重要的文字</p>
```

```css
/* 現在我們對「important」這個類別進行造型 */

/* 記得在 class 名稱前加一個「.」 */
.important {
    color: red;                /* 紅色文字 */
    font-weight: bold;         /* 粗體 */
    font-size: 18px;           /* 18px 大小 */
}
```

#### 🆔 ID 選擇器（ID Selector）- 給單一元素穿衣服

**比喻時間**：就像每個人有不同的身份證字號。

```html
<!-- id = 身份證字號，每個頁面只能有一個 -->
<div id="header">這是網頁的標題區塊</div>
```

```css
/* 在 id 名稱前加一個「#」 */
#header {
    background-color: black;   /* 黑色背景 */
    color: white;              /* 白色文字 */
    padding: 20px;             /* 內邊距 20px */
}
```

## 3️⃣ 實際操作（一步一步帶著做）

### 步驟 1️⃣：建立 HTML 檔案

1. 在資料夾中新增 `day2-css.html`
2. 輸入以下內容：

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>CSS 基礎教學</title>
    
    <!-- 第二種方式：內部樣式（Internal Style） -->
    <!-- 這是給「整個網頁」用的樣式 -->
    <style>
        /* 在這裡寫 CSS 樣式 */
        
        /* 首先，幫 body 穿衣服（網頁的背景） */
        body {
            /* 背景顏色：淡黃色 */
            background-color: #fff9e6;
            /* 字體：微軟正黑體 */
            font-family: "微軟正黑體", Arial, sans-serif;
            /* 讓文字和邊緣保持距離 */
            margin: 0;
            padding: 20px;
        }
        
        /* 幫 h1 標題穿衣服 */
        h1 {
            /* 文字顏色：咖啡色 */
            color: #8B4513;
            /* 文字置中 */
            text-align: center;
        }
        
        /* 幫 p 段落穿衣服 */
        p {
            /* 字體大小：18px */
            font-size: 18px;
            /* 行高（行與行之間的距離）：1.6倍 */
            line-height: 1.6;
        }
        
        /* 建立一個類別叫 "box" */
        .box {
            /* 背景：白色 */
            background-color: white;
            /* 邊框：虛線，灰色 */
            border: 2px dashed #cccccc;
            /* 內邊距：20px */
            padding: 20px;
            /* 外邊距：20px */
            margin: 20px;
            /* 圓角：10px */
            border-radius: 10px;
            /* 陰影：水平偏移 2px，垂直偏移 2px，模糊 5px，灰色 */
            box-shadow: 2px 2px 5px #888888;
        }
        
        /* 建立一個類別叫 "btn"（按鈕） */
        .btn {
            /* 背景：藍色 */
            background-color: #007bff;
            /* 文字：白色 */
            color: white;
            /* 去掉底線 */
            text-decoration: none;
            /* 內邊距：10px 上下，20px 左右 */
            padding: 10px 20px;
            /* 邊界：圓形 */
            border-radius: 5px;
            /* 讓它顯示為行內區塊（可以和其他元素並排） */
            display: inline-block;
            /* 游標變成手指（像連結一樣） */
            cursor: pointer;
        }
        
        /* 當滑鼠移到 btn 上面的時候 */
        .btn:hover {
            /* 背景變成深藍色 */
            background-color: #0056b3;
        }
        
    </style>
</head>
<body>
    <!-- 這裡開始是內容 -->
    
    <h1>🎨 CSS 基礎教學</h1>
    
    <p>你好！歡迎來到 CSS 的世界。現在讓我們一起來幫網頁穿衣服吧！</p>
    
    <!-- 使用我們設計的「box」類別 -->
    <div class="box">
        <h2>我是區塊 1</h2>
        <p>這個區塊有白色背景、虛線邊框、圓角和陰影。看起來很有設計感對吧？</p>
    </div>
    
    <!-- 另一個區塊 -->
    <div class="box">
        <h2>我是區塊 2</h2>
        <p>你可以看到每個區塊之間有距離（margin），區塊內的文字也有留白（padding）。</p>
    </div>
    
    <!-- 按鈕範例 -->
    <p>下面是一個按鈕，滑鼠移上去會變色：</p>
    <a href="#" class="btn">按我一下</a>
    
</body>
</html>
```

### 步驟 2️⃣：用瀏覽器打開看看

1. 對著 `day2-css.html` 檔案按兩下
2. 你應該會看到：
   - 🎨 淡黃色的網頁背景
   - 📝 咖啡色的標題
   - 📦 兩個有陰影的白色區塊
   - 🔘 一個藍色的按鈕（滑鼠移上去會變深藍色）

### 步驟 3️⃣：嘗試修改顏色

找到 `<style>` 標籤裡的內容，試著修改：

```css
/* 把背景改成淡綠色 */
body {
    background-color: #e6fff2;  /* 試試這個顏色！ */
}

/* 把按鈕改成紅色 */
.btn {
    background-color: #dc3545;  /* 試試這個顏色！ */
}
```

刷新頁面，看看有什麼變化！

### 步驟 4️⃣：挑戰 - 建立「外部 CSS 檔案」

1. 在同一個資料夾中，新增一個檔案叫做 `style.css`
2. 把 `<style>` 標籤裡面的內容（不包括 `<style>` 本身）複製到 `style.css`
3. 修改 HTML 檔案，把 `<style>` 標籤整個刪除，加入：

```html
<head>
    <meta charset="UTF-8">
    <title>CSS 基礎教學</title>
    <!-- 加入這一行！ -->
    <link rel="stylesheet" href="style.css">
</head>
```

4. 刷新頁面，應該會看到一模一樣的結果！

> 🎉 **恭喜！你已經學會了 CSS 的三種使用方式！**
