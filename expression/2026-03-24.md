# 📚 【表達力與領導力｜Day 1：為什麼你說不出重點？】

> 目標：從表達不好到成為有影響力的領導者

---

## 1️⃣ 為什麼大多數人表達不好？

### 常見問題

| 問題 | 原因 | 解決方法 |
|------|------|---------|
| 說很多但別人聽不懂 | 沒有結構 | 用結論先行 |
| 說話沒有重點 | 不知道重點是什麼 | 練習抓重點 |
| 開會說不出話 | 不知道要說什麼 | 準備框架 |
| 寫作很爛 | 沒有練習 | 每天寫作 |
| 接話接不住 | 不會觀察氛圍 | 練習回應 |

### 表達好的三個層次

```
第一層：說清楚
- 讓別人聽懂你說什麼

第二層：說服人
- 讓別人相信你的想法

第三層：影響人
- 讓別人願意跟隨你
```

## 2️⃣ 表達的核心：結論先行

### 什麼是結論先行？

```
❌ 錯誤示範：
「因為最近市場競爭很激烈，然後我們的產品進度可能有點落後，然後对手一直在降價，所以我們可能需要調整策略...」

✅ 正確示範：
「我們需要調整定價策略。原因是：1) 市場競爭激烈 2) 對手降價 3) 我們產品進度落後」
```

### 結論先行的公式

```
結論 + 原因 + 證據 + 行動

範例：
「我建議我們採用方案B（結論）。
因為方案A成本太高且風險大（原因）。
根據上季度的數據，方案A的失敗率是30%（證據）。
我建議立即開始評估方案B（行動）。」
```

## 3️⃣ 練習方法：電梯簡報

### 什麼是電梯簡報？

```
你在電梯裡遇到大老闆
他問你：「最近專案怎麼樣？」

你只有 30 秒到 2 分鐘
要說明：
- 現在的狀況
- 遇到的問題
- 你需要什麼幫助
```

### 練習模板

```
公式：狀況 + 問題 + 需要的幫助

「專案目前進度 70%（狀況）。
遇到的主要問題是人力不足（問題）。
如果能增加 2 位工程師，我們可以在兩週內完成（需要的幫助）。」
```

## 4️⃣ 寫作不好的根本原因

### 為什麼你寫不好？

```
1. 不知道讀者是誰
2. 沒有目標
3. 沒有結構
4. 練習不夠
```

### 寫作的四個問題

每次寫東西前，先問自己：

```
1. 誰會讀這篇文章？
2. 我要讓讀者做什麼？
3. 最重要的是什麼？
4. 讀完後會記得什麼？
```

---

## 5️⃣ 寫作練習：每天 5 分鐘

### Day 1 練習：寫一句話

```
今天學到的最重要的東西是什麼？

用這個格式：
「今天我學到___。這是因為___。
這代表___。」

例如：
「今天我學到結論先行。這是因為
沒有結構的說話讓人聽不懂。這代表
說話前要先想清楚重點。」
```

## 6️⃣ 今日小任務

### 任務 1：電梯簡報練習

```
找一個你工作中的問題
用「狀況 + 問題 + 需要的幫助」的格式
寫成 30 秒可以說完的版本

例如：
「這個月的進度落後兩週（狀況）。
主要因為測試環境不穩定（問題）。
如果 IT 可以協助修復環境，我們可以趕上進度（需要的幫助）。」
```

### 任務 2：每天寫作練習

```
今天用 5 分鐘
寫下你今天學到的最重要的東西
用「結論先行」的方式
```

### 任務 3：觀察身邊的人

```
今天觀察一個說話很有說服力的人
注意他說話的結構
記錄下來

明天我們會分析！
```
