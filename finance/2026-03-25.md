# Day 93：金融危機與系統性風險 - 歷史案例與現代防範

## 一、基礎概念：什麼是系統性風險？

### 1.1 用生活比喻：金融系統就像都市排水系統

想象一下城市的排水系統：
- **正常情況**：雨水的時候，水順暢地流進排水溝，不會造成問題
- **系統性風險**：當排水系統某個关键节点（如下水道入口）被堵住，水無處可去，就會 flooding，影響整個城市

金融系統也是一樣：
- **個別銀行倒閉**：就像你家門口的水溝堵住，影響你家
- **系統性風險**：就像主要排水系統堵塞，水回流到所有家庭，影響整個城市

### 1.2 系統性風險的定義

> **系統性風險（Systemic Risk）**：指金融體系中某個環節出問題，導致整個金融系統功能喪失的風險。它會像病毒一樣傳染，影響銀行、保險公司、股市、債券市場等所有金融機構。

**三大特徵**：
1. **傳染性**：一個機構倒閉會影響其他機構
2. **全局性**：影響整個金融市場的功能
3. **嚴重性**：可能導致經濟衰退或蕭條

## 二、核心知識：歷史上的重大金融危機

### 2.1 1929年大蕭條（Great Depression）

**背景**：
- 1920年代美國股市狂歡，道瓊指數從1921年的60點漲到1929年9月的381點
- 1929年10月28日「黑色星期一」，股市暴跌13%
- 10月29日「黑色星期二」，再跌12%

**危機過程**：
```
1929年10月 → 股市崩盤 → 銀行倒閉 → 消費萎縮 → 失業飆升
         ↓
1933年 → 美國GDP下降46%，失業率高達25%
```

**教訓**：
- 金融監管不足導致銀行過度槓桿
- 聯準會未能及時提供流動性
- 保護主義抬頭加劇經濟衰退

### 2.2 1997年亞洲金融風暴

**背景**：
- 1990年代亞洲經濟高速成長，泰國、印尼、馬來西亞等地吸引大量熱錢
- 泰國維持固定匯率，但外匯儲備不足
- 國際炒家索羅斯做空泰銖

**危機過程**：
```
1997年7月 → 泰國放棄固定匯率 → 泰銖暴跌50%
         ↓
1997年底 → 危機蔓延至印尼、馬來西亞、韓國
         ↓
1998年 → 俄羅斯違約 → LTCM倒閉
```

**教訓**：
- 固定匯率制度不可持續
- 外債過多導致脆弱性
- 資本管制可以緩解但不能根除問題

### 2.3 2008年全球金融海嘯

**背景**：
- 2000年代美國房價持續上漲
- 次級房貸（Subprime Mortgage）泛濫，金融機構將房貸包裝成複雜的金融商品（CDO）
- 銀行過度槓桿，槓桿率高達30:1

**危機過程**：
```
2007年 → 次級房貸違約率上升 → CDO價值暴跌
    ↓
2008年3月 → 貝爾斯登倒閉
    ↓
2008年9月 → 雷曼兄弟破產 → 全球金融系統癱瘓
    ↓
2009年 → 全球GDP萎縮2%，股市暴跌50%以上
```

**教訓**：
- 金融創新不能脫離監管
- 信用評等機構有利益衝突
- 系統重要性機構（Systemically Important Financial Institutions, SIFIs）需要特別監管

## 三、實際應用：現代金融危機防範機制

### 3.1 各國央行的危機應對工具

**傳統工具**：
| 工具 | 功能 | 說明 |
|------|------|------|
| 利率政策 | 調整資金成本 | 降息刺激經濟，升息抑制通膨 |
| 公開市場操作 | 調節流動性 | 購買/出售政府債券 |
| 貼現窗口 | 緊急流動性 | 銀行緊急借錢的管道 |

**2008年後的新工具**：
| 工具 | 功能 | 說明 |
|------|------|------|
| 量化寬鬆（QE） | 大規模資產購買 | 央行直接購買國債、MBS |
| 前瞻指引 | 未來利率預期 | 承諾維持低利率一段時間 |
| 負利率 | 存放央行要收費 | 鼓勵銀行放款而非存放 |
| 收益率曲線控制 | 控制特定期限利率 | 日本央行實施 |

### 3.2 金融監管改革（Basel III）

**Basel III 主要內容**：

1. **資本要求提高**：
   - 普通股比率（CET1）從2%提高到4.5%
   - 加上資本保護緩衝，總計7%

2. **流動性要求**：
   - 流動性覆蓋率（LCR）：優質流動資產/30天淨現金流出 ≥100%
   - 淨穩定資金比率（NSFR）：可用穩定資金/所需穩定資金 ≥100%

3. **槓桿率限制**：
   - 槓桿率（總資產/一級資本）不得超過3%

### 3.3 壓力測試（Stress Testing）

**什麼是壓力測試**？

就像建築工程要測試抗震能力，金融機構也要測試在極端情境下的承受力。

**壓力測試情境示例**：

| 情境 | GDP增速 | 失業率 | 房價跌幅 | 股市跌幅 |
|------|---------|--------|----------|----------|
| 基準 | 2.0% | 4.0% | 0% | 0% |
| 嚴重衰退 | -3.0% | 10% | -30% | -40% |
| 金融危機 | -5.0% | 15% | -50% | -60% |

**計算範例：銀行資本壓力測試**

假設某銀行：
- 一級資本：100億美元
- 風險加權資產：1000億美元
- 目前的CET1比率：10%

**壓力情境下**：
- 房產抵押贷款違約率從2%上升到8%
- 信用卡違約率從3%上升到12%
- 導致預期信用損失增加：30億美元
- 股票投資損失：20億美元

**計算結果**：
```
壓力後資本 = 100億 - 30億 - 20億 = 50億美元
壓力後CET1比率 = 50億 / 1000億 = 5%
```

這低於Basel III要求的7%（包含緩衝），銀行需要補充資本。

## 四、常見錯誤提醒

### 4.1 誤以為金融危機可以完全避免

❌ **錯誤觀點**：「加強監管就可以避免危機」

✅ **正確觀點**：
- 監管可以降低危機發生頻率和強度，但不能完全消除
- 金融創新會不斷挑戰監管邊界
- 投資者應該学会在波動中保護自己

### 4.2 誤以為大機構不會倒閉

❌ **錯誤觀點**：「大型銀行太大不能倒（Too Big to Fail）」

✅ **正確觀點**：
- 2008年雷曼兄弟倒閉證明大型機構也會倒
- 現在有「生前遺囑」（Living Will）要求大型機構準備倒閉計畫
- 系統重要性機構需要額外資本要求

### 4.3 忽視流動性風險

❌ **錯誤觀點**：「只要資產價值足夠，就不怕」

✅ **正確觀點**：
- 2008年很多機構有大量優質資產，但無法變現
- 流動性風險可以在短時間內摧垮健康的機構
- 平時要關注流動性覆蓋率和資金來源穩定性
