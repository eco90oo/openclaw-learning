# 📚 【Day 2】利率與折現 - 現值、終值、複利計算

---

## 1️⃣ 基礎概念（用生活比喻，越詳細越好）

### 🎯 為什麼要學利率？

想象一下：你今天口袋裡有100元，與其花掉，你可以選擇：
1. **現在吃掉**（享受）
2. **借給別人**（要利息）
3. **存到銀行**（生利息）

問題來了：**今天的100元，等於未來的多少錢？**

這就是利率與折現要回答的問題！

### 💰 利率的本質：時間的價格

**比喻**：利率就像是「時間的租金」。

就像租房子要付租金，**借錢也要付「租金」**——這就是利息。

```
今天的100元 → 明年變成105元（利率5%）
意思是：明年那105元 = 今年這100元
```

### ⏳ 為什麼現在的錢比未來的錢值錢？

**三個原因**：

1. **購買力下降**（通膨）
   - 30年前的100元可以買很多東西，現在只能買一杯珍奶
   - 錢會「變薄」

2. **機會成本**
   - 現在的錢可以先投資賺錢，為什麼要等？
   - 借給你我要放棄其他投資機會

3. **不確定性**
   - 未來能不能拿到錢？說不定對方倒閉了
   - 借錢有風險，要收「風險溢酬」

### 📐 折現（Discounting）：未來的錢如何變成現在的錢？

**比喻**：折現就像「把未來的禮物換成現在的現金」。

```
未來的100元 → 今天的多少元？
如果利率是5%：
未來的100元 → 今天的 100/(1+0.05) ≈ 95.24元
```

## 2️⃣ 核心知識（詳細教學，包含公式與計算）

### 📊 終值（Future Value, FV）：現在的錢，未來值多少？

**公式**：
```
FV = PV × (1 + r)^n

其中：
PV = 現在的錢（Present Value）
r = 年利率（以小數表示，如5% = 0.05）
n = 期數（年）
```

#### 計算範例1：單利 vs 複利

**情境**：你投資10萬元，年利率5%，投資10年。

**單利計算**：
```
單利利息 = PV × r × n = 100,000 × 0.05 × 10 = 50,000元
終值 = 100,000 + 50,000 = 150,000元
```

**複利計算**：
```
FV = 100,000 × (1 + 0.05)^10
FV = 100,000 × 1.629
FV = 162,889元
```

📌 **結論**：複利比單利多賺 12,889元！這就是「複利的奇蹟」。

#### 計算範例2：72法則

**快速估算**：本金翻倍需要多久？

```
72法則：72 ÷ 利率 = 翻倍年數

例子：
利率5% → 72 ÷ 5 = 14.4年翻倍
利率10% → 72 ÷ 7.2 = 10年翻倍
```

📌 **生活應用**：如果你的投資每年報酬率是7.2%，10年翻倍；如果是10%，約7.2年翻倍！

---

### 📊 現值（Present Value, PV）：未來的錢，現在值多少？

**公式**（折現）：
```
PV = FV / (1 + r)^n
```

#### 計算範例1：折現計算

**情境**：銀行告訴你，5年後會給你100萬元，利率3%，這筆錢現在值多少？

```
PV = 1,000,000 / (1 + 0.05)^5
PV = 1,000,000 / 1.276
PV = 783,526元
```

📌 **意思**：未來的100萬，只等於現在的78.4萬！因為你有機會用這78.4萬先投資，5年後會變成100萬。

#### 計算範例2：年金現值

**情境**：每年底會收到10萬元，持續5年，利率4%，現在值多少？

**公式**：
```
PV = C × [1 - (1+r)^-n] / r

其中：
C = 每年收到的金額
r = 利率
n = 期數
```

**計算**：
```
PV = 100,000 × [1 - (1+0.04)^-5] / 0.04
PV = 100,000 × [1 - 0.8219] / 0.04
PV = 100,000 × 0.1781 / 0.04
PV = 100,000 × 4.452
PV = 445,200元
```

📌 **應用**：這就是保險年金、房租收入的定價方式！

---

### 📊 內部報酬率（IRR）：投資的真實報酬率

**定義**：讓投資的現值等於0的利率。

**公式**：找出 r 使得：
```
Σ [CFt / (1+r)^t] = 0
```

#### 計算範例：專案投資

**情境**：
- 第0年投入：-100萬（花錢）
- 第1年收益：+30萬
- 第2年收益：+40萬
- 第3年收益：+50萬

**計算**：找到 r 使得：
```
-100 + 30/(1+r) + 40/(1+r)² + 50/(1+r)³ = 0
```

**試誤法**（Trial and Error）：

假設 r = 10%：
```
-100 + 30/1.1 + 40/1.1² + 50/1.1³
= -100 + 27.27 + 33.06 + 37.57
= -2.10（接近0）
```

假設 r = 9%：
```
-100 + 30/1.09 + 40/1.09² + 50/1.09³
= -100 + 27.52 + 33.68 + 38.58
= -0.22
```

假設 r = 8.8%：
```
-100 + 30/1.088 + 40/1.088² + 50/1.088³
= -100 + 27.57 + 33.78 + 38.80
= 0.15
```

📌 **結論**：IRR ≈ 8.8%

## 3️⃣ 實際應用（案例分析、計算範例）

### 🖥️ 案例1：房貸月付款計算

**情境**：
- 房價：1000萬元
- 自備款：200萬元
- 贷款：800萬元
- 利率：2.5%（年利率）
- 期限：20年

**公式**（等額本息）：
```
每月還款 = P × [r(1+r)^n] / [(1+r)^n - 1]

其中：
P = 贷款本金
r = 月利率（年利率÷12）
n = 總月數
```

**計算**：
```
P = 8,000,000元
r = 0.025/12 = 0.002083
n = 20年 × 12 = 240個月

每月還款 = 8,000,000 × [0.002083×(1.002083)^240] / [(1.002083)^240 - 1]
        = 8,000,000 × [0.002083 × 1.644] / [1.644 - 1]
        = 8,000,000 × 0.00342 / 0.644
        = 8,000,000 × 0.00531
        = 42,480元/月
```

📌 **總利息** = 42,480 × 240 - 8,000,000 = 2,195,200元

---

### 🖥️ 案例2：退休規劃 - 現在要存多少？

**情境**：
- 你想要在60歲時累積2000萬元
- 現在年齡：30歲
- 預期投資報酬率：6%
- 需要工作到60歲（30年）

**計算**：
```
PV = FV / (1+r)^n
PV = 20,000,000 / (1.06)^30
PV = 20,000,000 / 5.743
PV = 3,482,000元
```

📌 **結論**：現在需要存348.2萬，到60歲時會變成2000萬！

---

### 🖥️ 案例3：股票估價 - 股息折現模型

**情境**：
- 一檔股票每年發放股息5元
- 預期報酬率要求：8%
- 假設股息不成長

**公式**（永續年金）：
```
股票價值 = 股息 / 報酬率

PV = D / r
PV = 5 / 0.08 = 62.5元
```

📌 **結論**：如果這檔股票合理價值是62.5元，現在股價如果在50元以下，就是便宜！

---

### 🖥️ 案例4：年金險保單定價

**情境**：
- 你買了一張年金險，65歲開始每月領2萬元
- 預期領到85歲（20年 = 240個月）
- 假設月利率0.5%

**計算**：
```
每月領取 = 20,000元
期數 = 240個月
月利率 = 0.5%

年金現值 = 20,000 × [1 - (1.005)^-240] / 0.005
         = 20,000 × [1 - 0.302] / 0.005
         = 20,000 × 0.698 / 0.005
         = 20,000 × 139.6
         = 2,792,000元
```

📌 **結論**：這個年金險至少要繳279.2萬保費，才不會「吃虧」！

## 4️⃣ 常見錯誤提醒

### ❌ 錯誤1：忽略複利的威力
- 很多人低估了長期複利的效果
- 10萬元，7%報酬，40年後 = 1497萬元！（149倍！）

### ❌ 錯誤2：把「名目利率」當「實質利率」
- 名目利率 = 銀行公告的利率
- 實質利率 = 名目利率 - 通膨率
- 如果通膨3%，存款利率2%，你的錢其實在變少！

### ❌ 錯誤3：錯誤計算月利率
- 年利率12%，不是月利率12%！
- 月利率 = 12% ÷ 12 = 1%（不是12%！）

### ❌ 錯誤4：忽略貨幣的時間價值
- 欠錢時：盡快還（利息一直在累積）
- 借錢投資：要確認報酬率 > 利率

### ❌ 錯誤5：只用單利思考
- 很多人以為「每年賺5%，10年就是50%」
- 其實是「1.05^10 - 1 = 63%」！

## 📚 今日重點總結

| 概念 | 公式 | 意義 |
|------|------|------|
| **終值** | FV = PV × (1+r)^n | 現在的錢，未來值多少 |
| **現值** | PV = FV / (1+r)^n | 未來的錢，現在值多少 |
| **複利公式** | FV = PV × (1+r)^n | 利息滾利息 |
| **72法則** | 翻倍年數 = 72 ÷ 利率 | 快速估算翻倍時間 |
| **年金現值** | PV = C × [1-(1+r)^-n]/r | 未來一系列現金流的現在價值 |
| **IRR** | 使NPV=0的利率 | 投資的真正報酬率 |

## 📊 關鍵數字記憶

- **72法則**：72 ÷ 利率 = 翻倍年數
- **rule of 100**：100 - 年齡 = 股票比重（年輕人可承受更多風險）
- **7%**：長期股市平均報酬率（歷史經驗）
