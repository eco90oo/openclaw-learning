# 📚 【Day 1】金融市場是什麼？股票、債券、外匯、商品的基礎概念

---

## 1️⃣ 基礎概念（用生活比喻，越詳細越好）

### 🎯 什麼是金融市場？

想象一下：你有一顆蘋果樹，每年結100顆蘋果。但你今年想吃橘子，別人想要蘋果。這時就需要一個「交換場所」——金融市場就是這樣的地方，只不過交換的不是水果，而是「錢」和「未來的权利」。

金融市場就像是一個**大型、超級市場**，裡面買賣的是：
- **時間**（現在的錢 vs 未來的錢）
- **風險**（把風險賣給願意承擔的人）
- **所有權**（公司的股份）
- **債權**（借條）

### 🏪 金融市場的四大天王

| 市場 | 買什麼？ | 比喻 |
|------|----------|------|
| **股票市場** | 公司的所有權 | 買下一家店的股份 |
| **債券市場** | 借條（債權） | 把錢借給政府或企業 |
| **外匯市場** | 各國貨幣 | 兌換外幣 |
| **商品市場** | 實體物資（黃金、石油） | 買賣原材料 |

## 2️⃣ 核心知識（詳細教學，包含公式與計算）

### 📈 股票市場（Stock Market）

**定義**：股票市場是買賣公司所有權（股份）的場所。當你買入一家公司的股票，你就成為了這家公司的「老闆之一」（稱為「股東」）。

#### 股票的基本概念：

```
📌 股票 = 公司的部分所有權
📌 股價 = 市場對公司價值的「投票」
📌 市值 = 股價 × 總股數
```

**例子**：
- 台積電（2330）股價 = 500元
- 總股數 = 259億股
- 市值 = 500元 × 259億 = 12.95兆元

#### 股票的收益來源：

1. **資本利得**：買低賣高賺差價
   - 買入股價 100元 → 賣出股價 150元 = 賺 50元（50%報酬）

2. **股息**：公司賺錢後分給股东的錢
   - 每年發放，相當於「分紅」

3. **股票股利**：發放更多股票而非現金

---

### 💵 債券市場（Bond Market）

**定義**：債券是「借條」。當你買債券，你就是把錢借給發行債券的政府或企業，他們答應：
- 固定期限後還你本金
- 定期支付利息給你

#### 債券的關鍵術語：

| 術語 | 意思 |
|------|------|
| **面值（Face Value）** | 債券到期時拿回的金額（通常為100元） |
| **票面利率（Coupon Rate）** | 年利率，例如5% |
| **殖利率（Yield）** | 實際投資報酬率 |
| **到期日（Maturity）** | 還本日期 |

#### 債券的定價公式：

```
債券價格 = Σ (利息 / (1+y)^t) + (面值 / (1+y)^n)

其中：
y = 殖利率
t = 第幾期
n = 總期數
```

**計算例子**：
- 面值：100元
- 票面利率：5%
- 到期日：2年
- 殖利率：4%

```
第1年利息 = 100 × 5% = 5元 → 折現值 = 5 / (1+0.04)^1 = 4.81元
第2年利息 = 5元 → 折現值 = 5 / (1+0.04)^2 = 4.62元
本金 = 100元 → 折現值 = 100 / (1+0.04)^2 = 92.46元

債券價格 = 4.81 + 4.62 + 92.46 = 101.89元
```

📌 **重點**：殖利率上升 → 債券價格下降（反向關係！）

---

### 💱 外匯市場（Foreign Exchange Market, Forex）

**定義**：外匯市場是買賣不同國家貨幣的市場。這是全球最大的市場，每天交易量超過**7兆美元**！

#### 匯率的表示方式：

```
USD/TWD = 32.50
意思是：1美元 = 32.50新台幣
```

#### 匯率的報價方式：

| 術語 | 說明 |
|------|------|
| **基本貨幣** | 報價中的第一個貨幣（基準） |
| **報價貨幣** | 第二個貨幣 |
| **買入價（Bid）** | 銀行向你買入的價格 |
| **賣出價（Ask）** | 銀行賣給你的價格 |
| **點差（Spread）** | 買入價與賣出價的差異 |

**例子**：
- USD/TWD = 32.50/32.55
- 意思是：銀行用32.50向你買美元，用32.55賣給你
- 點差 = 0.05元（這是銀行的「手續費」）

---

### 🪙 商品市場（Commodity Market）

**定義**：商品市場交易的是實體物資，分為兩大類：

#### 商品分類：

| 種類 | 例子 | 特性 |
|------|------|------|
| **硬商品** | 黃金、白銀、原油 | 有形的實物 |
| **軟商品** | 小麥、玉米、咖啡 | 農產品 |

#### 商品的投資功能：

1. **避險工具**：對抗通膨（黃金是傳統「避險天堂」）
2. **投機標的**：短線交易獲利
3. **工業需求**：反映經濟景氣

**重要商品指標**：
- 黃金價格：通常以美元/盎司計算
- 原油價格：布蘭特原油（Brent）、西德州原油（WTI）

## 3️⃣ 實際應用（案例分析、計算範例）

### 🖥️ 案例1：從生活理解金融市場

**情境**：你想開一家珍珠奶茶店，需要100萬元。

**選項A**：自己存錢
- 需要時間，且賺到的錢只有你自己賺

**選項B**：發行股票（IPO）
- 找10個人，每人投資10萬元
- 每人獲得10%股份
- 賺的錢大家分，風險也大家承擔

**選項C**：發行債券
- 向銀行或親友借100萬元
- 每年支付5%利息（5萬元）
- 無論賺多少，利息都要給！
- 還本後奶茶店還是你的

📌 **結論**：股票適合「共擔風險、共享收益」；債券適合「確定成本、固定支出」。

---

### 🖥️ 案例2：計算投資報酬率

**情境**：你在2020年買進台積電股票100張，成本100元/股。

**計算**：
- 初始成本：100張 × 100元 = 10,000元
- 2025年股價：500元
- 資本利得：500元 - 100元 = 400元/股
- 總報酬：400元 × 100 = 40,000元
- **報酬率**：40,000 / 10,000 = **400%**（5年！）
- **年化報酬率**：((1+4)^(1/5) - 1) = 38%/年

---

### 🖥️ 案例3：債券價格變化

**情境**：
- 你買了一張面額100元、票面利率5%、10年到期的債券
- 當時殖利率也是5%，所以價格是100元

**問題**：如果明年殖利率上升到7%，債券價格會變多少？

**計算**：
```
剩下9年到期，每年利息5元，殖利率7%

價格 = 5/(1.07) + 5/(1.07)² + ... + 5/(1.07)^9 + 100/(1.07)^9
     = 5 × [1-(1/1.07)^9]/0.07 + 100/(1.07)^9
     = 5 × 6.515 + 100 × 0.554
     = 32.58 + 55.40
     = 87.98元
```

📌 **結果**：債券價格從100元跌到87.98元，**下跌12%**！

📌 **教訓**：利率上升時，債券投資人會受傷！

## 4️⃣ 常見錯誤提醒

### ❌ 錯誤1：把股票當彩票
- 股票是「ownership」，不是「 lottery ticket」
- 要研究公司基本面，不是盲目跟風

### ❌ 錯誤2：忽略風險
- 高報酬一定伴隨高風險
- 「保證高收益」通常是騙局！

### ❌ 錯誤3：把所有錢放同一個市場
- 分散投資是「免費的午餐」
- 不要把所有雞蛋放在同一個籃子

### ❌ 錯誤4：只看價格不看價值
- 股價只是「投票機」，不是「體重機」
- 要理解公司的內在價值

### ❌ 錯誤5：輕忽匯率風險
- 海外投資時，匯差可能吃掉所有獲利
- 要把匯率納入投資決策
