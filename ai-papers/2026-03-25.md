#### 論文 1：Hyperagents
- **原文標題**：Hyperagents
- **翻譯標題**：超級代理人：自我反思的元認知代理框架
- **作者**：Jenny Zhuoting Zhang 等（8 位作者，Meta FAIR）
- **發布日期**：2026年3月19日
- **摘要（英文）**：Self-improving AI systems aim to reduce reliance on human engineering by learning to improve their own learning and problem-solving processes. Existing approaches to self-improvement rely on fixed, handcrafted meta-level mechanisms, fundamentally limiting how fast such systems can improve. We introduce **hyperagents**, self-referential agents that integrate a task agent (which solves the target task) and a meta agent (which modifies itself and the task agent) into a single editable program. Crucially, the meta-level modification procedure is itself editable, enabling metacognitive self-modification, improving not only the task-solving behavior, but also the mechanism that generates future improvements.
- **摘要（中文）**：自我改進的 AI 系統旨在減少對人類工程的依賴，透過學習改進自身的學習和問題解決過程。現有的自我改進方法依賴固定的、手工設計的元層機制，根本上限制了這類系統的改進速度。我們提出超級代理人（hyperagents），這是一種自我反思的代理框架，將任務代理（解決目標任務）與元代理（修改自身和任務代理）整合為單一可編輯程式。關鍵在於，元層修改程序本身也是可編輯的，實現了元認知自我修改，不僅改進任務解決行為，還改進了產生未來改進的機制。
- **論文連結**：https://arxiv.org/abs/2603.19461
- **代碼連結**：https://github.com/facebookresearch/Hyperagents

#### 論文 2：OpenClaw-RL
- **原文標題**：OpenClaw-RL: Train Any Agent Simply by Talking
- **翻譯標題**：OpenClaw-RL：透過對話訓練任何代理
- **作者**：Ling Yang 等（Gen-Verse）
- **發布日期**：2026年3月10日
- **摘要（英文）**：Every agent interaction generates a next-state signal, namely the user reply, tool output, terminal or GUI state change that follows each action, yet no existing agentic RL system recovers it as a live, online learning source. We present OpenClaw-RL, a framework built on a simple observation: next-state signals are universal, and policy can learn from all of them simultaneously. Personal conversations, terminal executions, GUI interactions, SWE tasks, and tool-call traces are not separate training problems, but complementary signals for the same policy. OpenClaw-RL is simple, scalable, and foundation model-agnostic, requiring no special action space or environment API.
- **摘要（中文）**：每個代理互動都會產生下一狀態信號，即使用者回覆、工具輸出、終端或 GUI 狀態變更，但現有的代理強化學習系統都沒有将其作為即時、線上學習來源。我們提出 OpenClaw-RL，這個框架基於一個簡單的觀察：下一狀態信號是普遍存在的，原則可以同時從所有信號中學習。個人對話、終端執行、GUI 互動、SWE 任務和工具呼叫軌跡不是獨立的訓練問題，而是同一原則的互補信號。OpenClaw-RL 簡單、可擴展，且與基礎模型無關，不需要特殊的動作空間或環境 API。
- **論文連結**：https://arxiv.org/abs/2603.10165
- **代碼連結**：https://github.com/Gen-Verse/OpenClaw-RL

#### 論文 3：Memento-Skills
- **原文標題**：Memento-Skills: Let Agents Design Agents
- **翻譯標題**：Memento-Skills：讓代理設計代理
- **作者**：Huichi Zhou 等（Memento Teams）
- **發布日期**：2026年3月20日
- **摘要（英文）**：We introduce Memento-Skills, a general and可持续发展的 LLM agent system that can autonomously construct and improve task-specific agents through experience without updating LLM parameters. It maintains a memory of successful behaviors as "skills", and these skills can be dynamically composed to address new tasks. When encountering new tasks, Memento-Skills retrieves relevant skills from its memory and uses them as a scaffolding to guide the LLM in generating a more precise reasoning process.
- **摘要（中文）**：我們提出 Memento-Skills，這是一個通用且可持續發展的 LLM 代理系統，能夠透過經驗自主建構和改進任務特定的代理，無需更新 LLM 參數。它將成功的行為維持為「技能」記憶，這些技能可以動態組合以處理新任務。當遇到新任務時，Memento-Skills 從記憶中檢索相關技能，並用它們作為支架來引導 LLM 生成更精確的推理過程。
- **論文連結**：https://arxiv.org/abs/2603.18743
- **代碼連結**：https://github.com/Memento-Teams/Memento-Skills

#### 論文 4：OpenSeeker
- **原文標題**：OpenSeeker: Democratizing Frontier Search Agents by Fully Open-Sourcing Training Data
- **翻譯標題**：OpenSeeker：透過完全開源訓練數據實現前沿搜尋代理民主化
- **作者**：Rui Ye 等（港大、阿里）
- **發布日期**：2026年3月16日
- **摘要（英文）**：Deep search capabilities have become an indispensable competency for frontier LLM agents, yet the development of high-performance search agents remains dominated by industrial giants due to a lack of transparent, high-quality training data. To bridge this gap, we introduce OpenSeeker, the first fully open-source search agent that achieves frontier-level performance through (1) Fact-grounded scalable controllable QA synthesis, and (2) Denoised trajectory synthesis. Trained on only 11.7k samples, OpenSeeker outperforms DeepDive (29.5% vs 15.3% on BrowseComp).
- **摘要（中文）**：深度搜尋能力已成為前沿 LLM 代理的必備技能，然而高性能搜尋代理的開發仍被產業巨頭壟斷，因為缺乏透明、高品質的訓練數據。為了解決這個差距，我們推出 OpenSeeker，首個完全開源的搜尋代理，透過 (1) 事實基礎的可擴展可控 QA 合成，以及 (2) 去噪軌跡合成實現前沿級性能。僅用 11.7k 樣本訓練，OpenSeeker 超越 DeepDive（BrowseComp 上 29.5% vs 15.3%）。
- **論文連結**：https://arxiv.org/abs/2603.15594
