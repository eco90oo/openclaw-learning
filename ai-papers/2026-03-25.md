## 🔹 Foundation Models / LLM

### 1. Attention Residuals
- **原文標題**：Attention Residuals
- **翻譯標題**：注意力殘差
- **作者**：Kimi Team (36位作者，含清華、北大等)
- **發布機構/日期**：2026年3月16日
- **摘要**（英文）：Residual connections with PreNorm are standard in modern LLMs, yet they accumulate all layer outputs with fixed unit weights. This uniform aggregation causes uncontrolled hidden-state growth with depth, progressively diluting each layer's contribution. We propose Attention Residuals (AttnRes), which replaces this fixed accumulation with softmax attention over preceding layer outputs, allowing each layer to selectively aggregate earlier representations with learned, input-dependent weights.
- **摘要**（中文）：PreNorm 殘差連接是現代 LLM 的標準配置，但它們以固定權重累積所有層輸出。這種統一聚合導致隱藏狀態隨深度失控增長，逐步稀釋每層的貢獻。我們提出「注意力殘差」(AttnRes)，用 softmax 注意力取代固定累積，讓每層能夠選擇性地聚合先前的表示，透過學習到的輸入依賴權重。
- **論文連結**：[arXiv:2603.15031](https://arxiv.org/abs/2603.15031)
- **代碼連結**：技術報告，無開源

### 2. OpenSeeker: Democratizing Frontier Search Agents
- **原文標題**：OpenSeeker: Democratizing Frontier Search Agents by Fully Open-Sourcing Training Data
- **翻譯標題**：OpenSeeker：透過完全開源訓練數據實現前沿搜尋 Agent 的民主化
- **作者**：Rui Ye 等 (港大、阿里)
- **發布機構/日期**：2026年3月16日
- **摘要**（英文）：Deep search capabilities have become an indispensable competency for frontier LLM agents, yet the development of high-performance search agents remains dominated by industrial giants due to a lack of transparent, high-quality training data. To bridge this gap, we introduce OpenSeeker, the first fully open-source search agent that achieves frontier-level performance through (1) Fact-grounded scalable controllable QA synthesis, and (2) Denoised trajectory synthesis. Trained on only 11.7k samples, OpenSeeker outperforms DeepDive (29.5% vs 15.3% on BrowseComp).
- **摘要**（中文）：深度搜尋能力已成為前沿 LLM Agent 的必備技能，然而高性能搜尋 Agent 的開發仍被產業巨頭壟斷，因為缺乏透明、高品質的訓練數據。為解決這個差距，我們推出 OpenSeeker，首個完全開源的搜尋 Agent，透過 (1) 事實基礎的可擴展可控 QA 合成，以及 (2) 去噪軌跡合成實現前沿級性能。僅用 11.7k 樣本訓練，OpenSeeker 超越 DeepDive（BrowseComp 上 29.5% vs 15.3%）。
- **論文連結**：[arXiv:2603.15042](https://arxiv.org/abs/2603.15042)
- **代碼連結**：即將開源
