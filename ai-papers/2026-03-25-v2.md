#### 📄 論文 1

- **標題**: Describe-Then-Act: Proactive Agent Steering via Distilled Language-Action World Models
- **翻譯標題**: 先描述後行動：透過蒸餾語言-動作世界模型實現主動式 Agent 引導
- **作者**: Luca Romani 等
- **發布日期**: 2026-03-24
- **摘要 (English)**: Deploying safety-critical agents requires anticipating the consequences of actions before they are executed. While world models offer a paradigm for this proactive foresight, current approaches relying on visual simulation incur prohibitive latencies, often exceeding several seconds per step. In this work, we challenge the assumption that visual processing is necessary for failure prevention. We show that a trained policy's latent state, combined with its planned actions, already encodes sufficient information to anticipate action outcomes, making visual simulation redundant for failure prevention. To this end, we introduce DILLO (DIstiLLed Language-ActiOn World Model), a fast steering layer that shifts the paradigm from "simulate-then-act" to "describe-then-act." DILLO is trained via cross-modal distillation, where a privileged Vision Language Model teacher annotates offline trajectories and a latent-conditioned Large Language Model student learns to predict semantic outcomes. This creates a text-only inference path, bypassing heavy visual generation entirely, achieving a 14x speedup over baselines. Experiments on MetaWorld and LIBERO demonstrate that DILLO produces high-fidelity descriptions of the next state and is able to steer the policy, improving episode success rate by up to 15 pp and 9.3 pp on average across tasks.
- **摘要 (中文)**: 部署安全關鍵的 Agent 需要在執行前預測動作後果。雖然世界模型提供了這種主動預見的範式，但目前依賴視覺模擬的方法會產生巨大的延遲，通常每步驟超過幾秒鐘。本研究挑戰了視覺處理對於故障預防是必要的這一假設。我們展示了經過訓練的 policy 的潛在狀態結合其規劃的動作，已經編碼了足夠的信息來預測動作結果，使視覺模擬對於故障預防變得多餘。為此，我們引入了 DILLO（一個蒸餾語言-動作世界模型），這是一個快速的引導層，將範式從「模擬後行動」轉變為「描述後行動」。DILLO透過跨模態蒸餾進行訓練，其中特權視覺語言模型教師註釋離線軌跡，而潛在條件化的大型語言模型學生學習預測語義結果。這創建了一個僅文字的推論路徑，完全繞過繁重的視覺生成，比基線實現了 14 倍的加速。在 MetaWorld 和 LIBERO 上的實驗表明，DILLO 能產生下一狀態的高保真描述，並能夠引導 policy，在各任務上平均將 episode 成功率提高多達 15 個百分點和 9.3 個百分點。
- **論文連結**: https://arxiv.org/abs/2603.23149

#### 📄 論文 2

- **標題**: Cross-Agent Memory Collaboration via Contrastive Trajectory Distillation
- **翻譯標題**: 跨 Agent 記憶協作：透過對比軌跡蒸餾
- **作者**: Yurui Chang 等
- **發布日期**: 2026-03-24
- **摘要 (English)**: Large language model (LLM)-based agents rely on memory mechanisms to reuse knowledge from past problem-solving experiences. Existing approaches typically construct memory in a per-agent manner, tightly coupling stored knowledge to a single model's reasoning style. In modern deployments with heterogeneous agents, a natural question arises: can a single memory system be shared across different models? We found that naively transferring memory between agents often degrades performance, as such memory entangles task-relevant knowledge with agent-specific biases. To address this challenge, we propose MemCollab, a collaborative memory framework that constructs agent-agnostic memory by contrasting reasoning trajectories generated by different agents on the same task. This contrastive process distills abstract reasoning constraints that capture shared task-level invariants while suppressing agent-specific artifacts. We further introduce a task-aware retrieval mechanism that conditions memory access on task category, ensuring that only relevant constraints are used at inference time. Experiments on mathematical reasoning and code generation benchmarks demonstrate that MemCollab consistently improves both accuracy and inference-time efficiency across diverse agents, including cross-modal-family settings. Our results show that the collaboratively constructed memory can function as a shared reasoning resource for diverse LLM-based agents.
- **摘要 (中文)**: 基於大型語言模型 (LLM) 的 Agent 依賴記憶機制來重用過去問題解決經驗中的知識。現有方法通常以每個 Agent 的方式構建記憶，將儲存的知識與單一模型的推理風格緊密耦合。在具有異構 Agent 的現代部署中，一個自然的問題出現了：單一記憶系統能否跨不同模型共享？我們發現，僅僅在 Agent 之間轉移記憶通常會降低性能，因為這種記憶將任務相關知識與 Agent 特定的偏差糾纏在一起。為了解決這個挑戰，我們提出 MemCollab，這是一個協作記憶框架，通過對比不同 Agent 在同一任務上生成的推理軌跡來構建與 Agent 無關的記憶。這種對比過程蒸餾出捕獲共享任務級不變量的抽象推理約束，同時抑制 Agent 特定的偽跡。我們進一步引入了一種任務感知檢索機制，該機制根據任務類別條件化記憶訪問，確保在推理時只使用相關約束。在數學推理和代碼生成基準測試上的實驗表明，MemCollab 在包括跨模態家族設置在內的多樣化 Agent 中持續提高準確性和推理時間效率。我們的結果表明，協作構建的記憶可以作為多樣化 LLM 基礎 Agent 的共享推理資源發揮作用。
- **論文連結**: https://arxiv.org/abs/2603.23151

#### 📄 論文 3

- **標題**: Hyperagents
- **翻譯標題**: 超級代理人：自我反思的元認知代理框架
- **作者**: Jenny Zhuoting Zhang 等（Meta FAIR）
- **發布日期**: 2026-03-19
- **摘要 (English)**: Self-improving AI systems aim to reduce reliance on human engineering by learning to improve their own learning and problem-solving processes. Existing approaches to self-improvement rely on fixed, handcrafted meta-level mechanisms, fundamentally limiting how fast such systems can improve. We introduce hyperagents, self-referential agents that integrate a task agent (which solves the target task) and a meta agent (which modifies itself and the task agent) into a single editable program. Crucially, the meta-level modification procedure is itself editable, enabling metacognitive self-modification, improving not only the task-solving behavior, but also the mechanism that generates future improvements.
- **摘要 (中文)**: 自我改進的 AI 系統旨在減少對人類工程的依賴，透過學習改進自身的學習和問題解決過程。現有的自我改進方法依賴固定的、手工設計的元層機制，根本上限制了這類系統的改進速度。我們提出超級代理人（hyperagents），這是一種自我反思的代理框架，將任務代理（解決目標任務）與元代理（修改自身和任務代理）整合為單一可編輯程式。關鍵在於，元層修改程序本身也是可編輯的，實現了元認知自我修改，不僅改進任務解決行為，還改進了產生未來改進的機制。
- **論文連結**: https://arxiv.org/abs/2603.19461
- **代碼連結**: https://github.com/facebookresearch/Hyperagents

#### 📄 論文 4

- **標題**: OpenSeeker: Democratizing Frontier Search Agents by Fully Open-Sourcing Training Data
- **翻譯標題**: OpenSeeker：透過完全開源訓練數據實現前沿搜尋代理民主化
- **作者**: Rui Ye 等（港大、阿里）
- **發布日期**: 2026-03-16
- **摘要 (English)**: Deep search capabilities have become an indispensable competency for frontier LLM agents, yet the development of high-performance search agents remains dominated by industrial giants due to a lack of transparent, high-quality training data. To bridge this gap, we introduce OpenSeeker, the first fully open-source search agent that achieves frontier-level performance through (1) Fact-grounded scalable controllable QA synthesis, and (2) Denoised trajectory synthesis. Trained on only 11.7k samples, OpenSeeker outperforms DeepDive (29.5% vs 15.3% on BrowseComp).
- **摘要 (中文)**: 深度搜尋能力已成為前沿 LLM 代理的必備技能，然而高性能搜尋代理的開發仍被產業巨頭壟斷，因為缺乏透明、高品質的訓練數據。為了解決這個差距，我們推出 OpenSeeker，首個完全開源的搜尋代理，透過 (1) 事實基礎的可擴展可控 QA 合成，以及 (2) 去噪軌跡合成實現前沿級性能。僅用 11.7k 樣本訓練，OpenSeeker 超越 DeepDive（BrowseComp 上 29.5% vs 15.3%）。
- **論文連結**: https://arxiv.org/abs/2603.15594

#### 📄 論文 5

- **標題**: OpenClaw-RL: Train Any Agent Simply by Talking
- **翻譯標題**: OpenClaw-RL：透過對話訓練任何代理
- **作者**: Ling Yang 等（Gen-Verse）
- **發布日期**: 2026-03-10
- **摘要 (English)**: Every agent interaction generates a next-state signal, namely the user reply, tool output, terminal or GUI state change that follows each action, yet no existing agentic RL system recovers it as a live, online learning source. We present OpenClaw-RL, a framework built on a simple observation: next-state signals are universal, and policy can learn from all of them simultaneously.
- **摘要 (中文)**: 每個代理互動都會產生下一狀態信號，即使用者回覆、工具輸出、終端或 GUI 狀態變更，但現有的代理強化學習系統都沒有将其作為即時、線上學習來源。我們提出 OpenClaw-RL，這個框架基於一個簡單的觀察：下一狀態信號是普遍存在的，原則可以同時從所有信號中學習。
- **論文連結**: https://arxiv.org/abs/2603.10165
- **代碼連結**: https://github.com/Gen-Verse/OpenClaw-RL
