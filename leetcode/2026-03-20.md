## 2️⃣ 核心知識（演算法解釋）

### 🔸 第一課：陣列（Array）— 最基本的資料結構

#### 💡 生活比喻

想象你有一排「置物櫃」📦
- 每個櫃子有一個編號（索引）
- 櫃子裡可以放東西（資料）
- 櫃子是連續排列的

```
┌─────┬─────┬─────┬─────┬─────┐
│  0  │  1  │  2  │  3  │  4  │
├─────┼─────┼─────┼─────┼─────┤
│ 🍎  │ 🍌  │ 🍇  │ 🍊  │ 🥝  │
└─────┴─────┴─────┴─────┴─────┘
```

這就是「陣列」！一排連續的格子，每格有編號。

#### 🔑 陣列的重點特性

| 特性 | 說明 |
|------|------|
| 📍 隨機存取 | 可以用編號直接找到任何位置（像用GPS定位） |
| ⏰ 時間複雜度 | 查詢快 O(1)，但插入/刪除慢 O(n) |
| 📦 固定大小 | 宣告時要決定多大（像買櫃子時決定要幾格） |

> **小提醒**：Python 的 list 就是動態陣列，可以自動變大！

## 3️⃣ 實際操作（經典題目示範）

### 📝 範例題目 1：反轉陣列

**題目**：把 `[1, 2, 3, 4, 5]` 反轉成 `[5, 4, 3, 2, 1]`

#### 👨‍🏫 老師帶著做

**第一步：理解問題**
> 就像把一疊書倒過來放，最上面變最下面

**第二步：思考解法**

解法一（直觀法）：
```
從前面取一個，放到新陣列的後面
```

解法二（聰明法）：
```
用兩個指標，從陣列兩端往中間靠，每次交換
```

**第三步：寫程式（Python）**

```python
# 方法一：使用內建功能（菜鳥會用）
def reverse_array(arr):
    return arr[::-1]  # 切片操作，從後往前

# 方法二：自己動手做（老手會用）
def reverse_array_manual(arr):
    left = 0
    right = len(arr) - 1
    
    while left < right:
        # 交換左右兩邊的元素
        arr[left], arr[right] = arr[right], arr[left]
        left += 1
        right -= 1
    
    return arr

# 測試
test = [1, 2, 3, 4, 5]
print(reverse_array_manual(test))
# 輸出：[5, 4, 3, 2, 1]
```

#### 📊 複雜度分析

| 方法 | 時間複雜度 | 空間複雜度 |
|------|-----------|-----------|
| 切片 [::-1] | O(n) | O(n) |
| 雙指標交換 | O(n) | O(1) ✅ |

> **雙指標法**比較厲害，因為不需要額外空間！

### 📝 範例題目 2：找最大值

**題目**：從陣列 `[3, 7, 2, 9, 1]` 中找到最大的數

```python
def find_max(arr):
    if not arr:
        return None
    
    max_val = arr[0]  # 假設第一個是最大的
    
    for num in arr:   # 一個一個檢查
        if num > max_val:
            max_val = num
    
    return max_val

# 測試
print(find_max([3, 7, 2, 9, 1]))  # 輸出：9
```

#### 💡 比喻
> 就像你要找出全班最高的人
> - 先假設第一個人最高
> - 然後一個個去比較
> - 遇到更高的就更新紀錄
