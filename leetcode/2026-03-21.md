### 演算法是什麼？

**比喻：** 演算法就像「食譜」🍳

假設你要做蛋炒飯：
```
食譜（演算法）：
1. 熱鍋
2. 倒油
3. 炒蛋
4. 加飯
5. 調味
```

## 2️⃣ 核心知識：陣列與字串

### 📦 什麼是陣列（Array）？

**比喻：** 陣列就像「停車場」🅿️

```
停車位  1    2    3    4    5
車輛   [A] [B] [C] [D] [E]
編號    0    1    2    3    4
```

- 每個位置有一個編號（索引）
- 從 0 開始數（不是 1！）
- 找車很快：直接看編號就好

**生活中的陣列：**
- 成績單上的分數列表
- 菜單上的菜名
- 籃球員的號碼

---

### 📝 什麼是字串（String）？

**比喻：** 字串就像「一串珠子」📿

```
珠子：C - O - D - E
字元： C   O   D   E
```

- 字串就是「字元的串」連在一起
- 每個字母、數字、符號都是一個字元

---

### 🔥 基礎操作示範

#### 1. 建立陣列（JavaScript）

```javascript
// 建立一個停車場，裡面有 5 個位置
let parking = ["A", "B", "C", "D", "E"];

// 或者從頭建立
let numbers = [1, 2, 3, 4, 5];
```

#### 2. 取值（像找停車位的車）

```javascript
let parking = ["A", "B", "C", "D", "E"];

// 編號 0 是第一個位置
console.log(parking[0]);  // 輸出: A
console.log(parking[2]);  // 輸出: C
```

#### 3. 改值（像把停車位的車換掉）

```javascript
let parking = ["A", "B", "C", "D", "E"];

// 把第 3 個位置（編號 2）的車換成 "X"
parking[2] = "X";
// 現在 parking = ["A", "B", "X", "D", "E"]
```

#### 4. 字串基本操作

```javascript
let message = "Hello";

// 取字元
console.log(message[0]);  // H

// 字串長度
console.log(message.length);  // 5

// 連接字串
let name = "小明";
console.log("你好，" + name);  // 你好，小明
```

## 3️⃣ 實際操作：經典題目示範

### 🅰️ 經典題目：反轉陣列

**題目：** 把 [1, 2, 3, 4, 5] 變成 [5, 4, 3, 2, 1]

**比喻：** 就像把排隊的人順序反過來站

```
原本：   1 → 2 → 3 → 4 → 5
反轉後： 5 → 4 → 3 → 2 → 1
```

**解題思路：**

```
想法：
1. 從兩端開始，慢慢往中間移動
2. 交換左右兩邊的位置
3. 直到相遇為止
```

**程式碼：**

```javascript
function reverseArray(arr) {
    let left = 0;           // 左邊指標
    let right = arr.length - 1;  // 右邊指標
    
    while (left < right) {
        // 交換左右兩邊的元素
        let temp = arr[left];
        arr[left] = arr[right];
        arr[right] = temp;
        
        // 往中間移動
        left++;
        right--;
    }
    
    return arr;
}

// 測試
let numbers = [1, 2, 3, 4, 5];
console.log(reverseArray(numbers));  // [5, 4, 3, 2, 1]
```

**圖解過程：**

```
Step 1: left=0, right=4
[1] [2] [3] [4] [5]
↑               ↑
left           right
→ 交換後 → [5] [2] [3] [4] [1]

Step 2: left=1, right=3
[5] [2] [3] [4] [1]
    ↑       ↑
   left    right
→ 交換後 → [5] [4] [3] [2] [1]

Step 3: left=2, right=2
相遇了！完成！
```

### 🅱️ 經典題目：計算字串中某字元出現幾次

**題目：** "apple" 中 'p' 出現幾次？

**解題思路：**

```
就像數鈔票：
一張一張檢查，看到 'p' 就計數 +1
```

**程式碼：**

```javascript
function countChar(str, target) {
    let count = 0;
    
    for (let i = 0; i < str.length; i++) {
        if (str[i] === target) {
            count = count + 1;
        }
    }
    
    return count;
}

// 測試
console.log(countChar("apple", "p"));  // 2
```

**圖解：**

```
字串: "apple"
索引:  0  1  2  3  4
字元:  a  p  p  l  e

檢查：
i=0: 'a' == 'p'? 否 → count=0
i=1: 'p' == 'p'? 是 → count=1
i=2: 'p' == 'p'? 是 → count=2
i=3: 'l' == 'p'? 否 → count=2
i=4: 'e' == 'p'? 否 → count=2

結果：2 次
```
