## 1️⃣ 基礎概念（用生活比喻）

### 🔗 什麼是連結串列？

**比喻：** 連結串列就像「火車」🚂

```
火車車廂： [車頭] ──→ [車廂1] ──→ [車廂2] ──→ [車廂3] ──→ null（末端）
         head
```

- 每節車廂只連接下一節
- 想知道第 5 節車廂？你必須從車頭開始，一節一節往下走
- 不能像停車場那樣直接跳到第 5 個位置

**連結串列 vs 停車場（陣列）**

| 特性 | 連結串列（火車） | 陣列（停車場） |
|------|-----------------|----------------|
| 記憶體 | 不連續，一節一節 | 連在一起 |
| 找東西 | 慢（要從頭走） | 快（直接跳到編號） |
| 加入/刪除 | 快（改變連結） | 慢（要搬動後面的） |
| 比喻 | 火車 | 停車場 |

---

### 📚 什麼是堆疊？

**比喻：** 堆疊就像「疊盤子」🥘

```
疊盤子：
   ┌─────┐
   │ 盤5 │ ← 最後放上去的，最先拿走（LIFO）
   ├─────┤
   │ 盤4 │
   ├─────┤
   │ 盤3 │
   ├─────┤
   │ 盤2 │
   ├─────┤
   │ 盤1 │ ← 最先放上去的，最後拿走
   └─────┘
```

**特點：後進先出（Last In First Out, LIFO）**

就像疊盤子、堆箱子一樣，最後放上去的東西最先被拿走。

**生活中的堆疊：**
- 瀏覽器的「上一頁」功能
- 函式呼叫（function call）
- 撤销操作（Ctrl+Z）

---

### 🚶 什麼是佇列？

**比喻：** 佇列就像「排隊」🈶

```
排隊買票：
[小明] → [小華] → [小美] → [小強] → (你)
 ↑                      ↑
先加入                  後加入
先離開                 後離開
```

**特點：先進先出（First In First Out, FIFO）**

就像排隊買票、捷運站排隊一樣，最先排隊的最先買到票。

**生活中的佇列：**
- 餐廳叫號
- 印表機列印排程
- 銀行櫃檯叫號

## 2️⃣ 核心知識：連結串列

### 📦 連結串列的組成

```
┌─────────────────────────────────────────┐
│           連結串列 Node 結構              │
├─────────────────────────────────────────┤
│  ┌───────┐      ┌───────┐               │
│  │  data │ ───→ │  next │               │
│  │  (資料)│      │ (指向│               │
│  └───────┘      │ 下一個)│               │
│                 └───────┘               │
│                                         │
│  data: 存放的值（可以是數字、字串等）       │
│  next: 指向下一個節點的指標（如果是最後    │
│        一個，則是 null）                  │
└─────────────────────────────────────────┘
```

### 🔥 連結串列基本操作

#### 1. 建立連結串列

```javascript
// 定義一個節點
class ListNode {
    constructor(value) {
        this.value = value;  // 存放的資料
        this.next = null;    // 指向下一個節點
    }
}

// 建立節點
let node1 = new ListNode(1);
let node2 = new ListNode(2);
let node3 = new ListNode(3);

// 把它們連在一起
node1.next = node2;
node2.next = node3;

// 現在是：1 → 2 → 3 → null
```

#### 2. 遍歷連結串列（從頭走到尾）

```javascript
function traverse(head) {
    let current = head;  // 從頭開始
    
    while (current !== null) {
        console.log(current.value);  // 印出目前節點的值
        current = current.next;       // 移動到下一個
    }
}

// 假設有：1 → 2 → 3 → null
// 輸出：1, 2, 3
```

**圖解遍歷過程：**

```
Step 1: current = node1 (值=1)
印出：1
移動：current = node1.next = node2

Step 2: current = node2 (值=2)
印出：2
移動：current = node2.next = node3

Step 3: current = node3 (值=3)
印出：3
移動：current = node3.next = null

Step 4: current = null
迴圈結束！
```

#### 3. 在串列末尾加入新節點

```javascript
function append(head, value) {
    let newNode = new ListNode(value);
    
    // 如果串列是空的
    if (head === null) {
        return newNode;
    }
    
    // 否則走到最後
    let current = head;
    while (current.next !== null) {
        current = current.next;
    }
    
    // 連上新節點
    current.next = newNode;
    return head;
}
```

#### 4. 刪除串列中的某個節點

```javascript
function deleteNode(head, value) {
    // 如果要刪的是第一個
    if (head.value === value) {
        return head.next;
    }
    
    let current = head;
    while (current.next !== null) {
        if (current.next.value === value) {
            current.next = current.next.next;  // 跳過要刪的節點
            return head;
        }
        current = current.next;
    }
    return head;
}
```

## 3️⃣ 實際操作：經典題目示範

### 🅰️ 經典題目：反轉連結串列

**題目：** 把 1 → 2 → 3 變成 3 → 2 → 1

**比喻：** 就像把火車的車頭和車尾對調！

**解題思路：**

```
三個指標來幫忙：
- prev：記住前一個節點
- current：現在處理的節點
- next：暫存下一個節點

一步一步把指向反過來：
1 → 2 → 3 → null
↑    ↑
next current

第一次反轉：
null ← 1   2 → 3 → null
          ↑
         current
```

**程式碼：**

```javascript
function reverseList(head) {
    let prev = null;
    let current = head;
    
    while (current !== null) {
        let next = current.next;  // 1. 記住下一個
        current.next = prev;       // 2. 反轉指向
        prev = current;            // 3. prev 往前一步
        current = next;            // 4. current 往前一步
    }
    
    return prev;  // 新的 head
}
```

**圖解過程：**

```
一開始：null   [1] → [2] → [3] → null
              prev  cur   next

Step 1: 反轉第一個
        null ← [1]   [2] → [3] → null
                  prev cur  next

Step 2: 反轉第二個
        null ← [1] ← [2]   [3] → null
                        prev cur

Step 3: 反轉第三個
        null ← [1] ← [2] ← [3]   null
                              prev cur

完成：prev 指向 3，就是新的 head！
結果：3 → 2 → 1 → null
```

### 🅱️ 經典題目：堆疊的應用 - 有效的括號

**題目：** 判斷 "({[]})" 是否為有效的括號組合

**思路：** 用堆疊來配對！

- 遇到開括號 `(` `[` `{` → 推進堆疊
- 遇到關括號 `)` `]` `}` → 檢查堆疊頂端是否匹配
- 最後堆疊應該是空的

**程式碼：**

```javascript
function isValid(s) {
    let stack = [];
    
    // 對應表
    let pairs = {
        ')': '(',
        ']': '[',
        '}': '{'
    };
    
    for (let char of s) {
        // 如果是關括號
        if (pairs[char]) {
            // 堆疊頂端必須匹配
            if (stack.length === 0 || stack.pop() !== pairs[char]) {
                return false;
            }
        } else {
            // 如果是開括號，推進堆疊
            stack.push(char);
        }
    }
    
    // 最後堆疊應該是空的
    return stack.length === 0;
}

// 測試
console.log(isValid("({[]})"));  // true
console.log(isValid("([)]"));    // false
```

**圖解 "({[]})" 的處理過程：**

```
字串： ( { [ ] } )
索引： 0  1  2  3  4

Step 1: char = '(' (開括號)
堆疊： ['(']

Step 2: char = '{' (開括號)
堆疊： ['(', '{']

Step 3: char = '[' (開括號)
堆疊： ['(', '{', '[']

Step 4: char = ']' (關括號)
→ 堆疊頂端是 '['，匹配！pop 掉
堆疊： ['(', '{']

Step 5: char = '}' (關括號)
→ 堆疊頂端是 '{'，匹配！pop 掉
堆疊： ['(']

Step 6: char = ')' (關括號)
→ 堆疊頂端是 '('，匹配！pop 掉
堆疊： []

最終堆疊是空的 → 回傳 true ✓
```

### 🅾️ 經典題目：佇列的應用 - binary tree 层序遍历

**題目：** 用佇列實現二元樹的層序遍歷（Level Order Traversal）

**比喻：** 就像餐廳叫號，一組一組處理

**程式碼：**

```javascript
function levelOrder(root) {
    if (!root) return [];
    
    let queue = [root];  // 佇列
    let result = [];
    
    while (queue.length > 0) {
        let levelSize = queue.length;
        let currentLevel = [];
        
        for (let i = 0; i < levelSize; i++) {
            let node = queue.shift();  // 取出最前面的
            currentLevel.push(node.val);
            
            if (node.left) queue.push(node.left);
            if (node.right) queue.push(node.right);
        }
        
        result.push(currentLevel);
    }
    
    return result;
}
```

function reverseString(str) {
    let stack = [];
    
    // 把字元推進堆疊
    for (let char of str) {
        stack.push(char);
    }
    
    // 彈出來就是反轉
    let result = "";
    while (stack.length > 0) {
        result += stack.pop();
    }
    return result;
}
