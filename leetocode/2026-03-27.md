# 📚 【Day 4: 滑動視窗（Sliding Window）- 深入講解】

## 1️⃣ 基礎概念（用生活比喻）

### 🎯 什麼是滑動視窗？

**比喻**：想象你在看一捲連續的幻燈片膠卷：
- 膠卷上有 100 張照片
- 你手裡拿著一個「放大鏡」，一次只能看 5 張
- 每看完 5 張，你把放大鏡往右移一格
- 每次移動，你都在檢視新的 5 張照片

這個「放大鏡」就是**滑動視窗**！它會在陣列上**滑動**，每次看一組連續的元素。

### 🔧 滑動視窗的運作原理

```
原始陣列：[1, 2, 3, 4, 5, 6, 7]
視窗大小：3

視窗移動過程：
[1, 2, 3] → 4, 5, 6, 7  → 移動 1 格
 1 [2, 3, 4] → 5, 6, 7  → 移動 1 格
 1, 2 [3, 4, 5] → 6, 7  → 移動 1 格
 1, 2, 3 [4, 5, 6] → 7  → 移動 1 格
 1, 2, 3, 4 [5, 6, 7]   → 視窗到終點
```

### 📊 滑動視窗的兩大類型

| 類型 | 比喻 | 何時使用 |
|------|------|----------|
| **固定大小視窗** | 放大鏡大小不變 | 求子陣列最大和、子陣列平均值 |
| **可變大小視窗** | 橡皮筋可伸可縮 | 最長子字串、滿足條件的最短子陣列 |

## 2️⃣ 核心知識（時間/空間複雜度分析）

### ⏰ 滑動視窗複雜度分析

| 情境 | 時間複雜度 | 空間複雜度 | 說明 |
|------|------------|------------|------|
| **暴力法** | O(n×k) | O(1) | 每個位置都重新計算視窗 |
| **滑動視窗** | O(n) | O(1) | 只移動邊界，增量更新 |

**關鍵優化**：不要每次重新計算整個視窗，而是**利用上次結果，只做增量更新**！

### 🎯 滑動視窗的核心思維

1. **維護窗口**：用兩個指標（left, right）維護當前窗口範圍
2. **增量更新**：每次移動視窗時，移除左邊元素，加入右邊元素
3. **最優化問題**：持續更新最大/最小值

### 🔄 滑動視窗模板（固定大小）

```python
def fixed_sliding_window(nums, k):
    """
    固定大小滑動視窗模板
    
    步驟：
    1. 初始化窗口（前 k 個元素）
    2. 遍歷剩餘元素，每次窗口向右滑動一格
    3. 移除左邊元素，加入右邊元素，更新答案
    """
    # 邊界處理
    if not nums or k <= 0 or k > len(nums):
        return 0
    
    # 第一步：計算第一個窗口的和（初始化）
    window_sum = sum(nums[0:k])
    max_sum = window_sum  # 假設第一個是最大值
    
    # 第二步：滑動窗口
    # 從第 k 個位置開始，到陣列末端
    for i in range(k, len(nums)):
        # 滑動：減去左邊的，加上右邊的
        # 這就是「增量更新」的核心！
        window_sum = window_sum - nums[i - k] + nums[i]
        
        # 更新最大/最小值
        max_sum = max(max_sum, window_sum)
    
    return max_sum
```

### 📌 LeetCode 643 - 子陣列最大平均數

**題目**：給定一個陣列和一個整數 k，返回最大平均數。

**範例輸入**：`nums = [1,12,-5,-6,50,3], k = 4`
**範例輸出**：`12.75`（12.75 是因為 (12-5-6+50)/4 = 51/4 = 12.75）

### 💻 Python 程式碼（詳細註解）

def findMaxAverage(nums, k):
    """
    子陣列最大平均數 - 固定大小滑動視窗
    
    比喻：就像一個「移動的計算機」
    每次向右移動一位，重新計算平均值，找最大的
    
    參數:
        nums: List[float] - 浮點數陣列
        k: int - 子陣列的長度
    返回:
        float - 最大平均數
    
    時間複雜度: O(n) - 只遍歷一次
    空間複雜度: O(1) - 只用幾個變數
    """
    
    # 邊界情況處理
    if not nums or k <= 0:
        return 0.0
    
    # ==========================================
    # 第一步：計算第一個窗口的和（初始化）
    # ==========================================
    # 取前 k 個元素的和
    # 例如：nums = [1,12,-5,-6,50,3], k = 4
    # 第一個窗口是 [1,12,-5,-6]，和 = 1+12+(-5)+(-6) = 2
    window_sum = sum(nums[0:k])
    
    # 假設第一個窗口的和是最大值
    max_sum = window_sum
    
    # ==========================================
    # 第二步：滑動窗口遍歷
    # ==========================================
    # 從索引 k 開始，到陣列最後一個元素
    # 每次向右滑動一格
    for i in range(k, len(nums)):
        # 核心滑動公式：減去左邊，加入右邊
        # i = 4 時：window_sum = 2 - nums[0] + nums[4] = 2 - 1 + 50 = 51
        # 新的窗口是 [12,-5,-6,50]，和 = 12 + (-5) + (-6) + 50 = 51
        window_sum = window_sum - nums[i - k] + nums[i]
        
        # 更新最大值（保留最大的窗口和）
        max_sum = max(max_sum, window_sum)
    
    # 計算平均數並返回
    return max_sum / k


# 🧪 測試
print(findMaxAverage([1,12,-5,-6,50,3], 4))  # 輸出: 12.75
print(findMaxAverage([5], 1))                # 輸出: 5.0
print(findMaxAverage([1,-1], 1))             # 輸出: 1.0


### 📌 LeetCode 3 - 無重複字符的最長子字串

**題目**：找出不包含重複字符的最長子字串長度。

**範例輸入**：`s = "abcabcbb"`
**範例輸出**：`3`（"abc" 是最長的無重複子字串）

### 💻 Python 程式碼（詳細註解）

def lengthOfLongestSubstring(s):
    """
    無重複字符的最長子字串 - 可變大小滑動視窗
    
    比喻：就像一個「跑步訓練」
    - right 像教練喊「前進」，一直向右跑
    - left 像助教喊「太近了，退後」，收縮隊伍
    - 用 set 檢查有沒有重複的「隊員」
    
    參數:
        s: str - 輸入字串
    返回:
        int - 最長無重複子字串的長度
    
    時間複雜度: O(n) - 每個字符最多被訪問兩次
    空間複雜度: O(min(m, n)) - m 是字符集大小
    """
    
    # 字串為空直接返回 0
    if not s:
        return 0
    
    # ==========================================
    # 初始化
    # ==========================================
    # char_set：用來存儲當前窗口中的字符（像「隊員名單」）
    # left：窗口左邊界
    # max_length：記錄最長長度
    char_set = set()        # 用 set 快速檢查字符是否存在
    left = 0                # 左指標從 0 開始
    max_length = 0         # 初始化最大長度為 0
    
    # ==========================================
    # 滑動窗口遍歷
    # ==========================================
    # right 指標遍歷整個字串
    for right in range(len(s)):
        # 獲取當前字符
        current_char = s[right]
        
        # ======================================
        # 收縮窗口：當遇到重複字符時
        # ======================================
        # 當前字符已經在 set 中，說明有重複
        # 需要移動 left，收縮窗口直到重複字符消失
        while current_char in char_set:
            # 移除左邊的字符（像「退後」）
            # 例如：窗口是 "abc"，遇到新的 'a'
            # 移除 'a'，窗口變成 "bc"
            char_set.remove(s[left])
            left += 1  # 左指標向右移動
        
        # ======================================
        # 擴展窗口：加入新字符
        # ======================================
        # 把當前字符加入 set（像「加入新隊員」）
        char_set.add(current_char)
        
        # ======================================
        # 更新答案
        # ======================================
        # 計算當前窗口長度 = right - left + 1
        # 更新最大長度
        current_length = right - left + 1
        max_length = max(max_length, current_length)
    
    return max_length


# 🧪 測試
print(lengthOfLongestSubstring("abcabcbb"))  # 輸出: 3 ("abc")
print(lengthOfLongestSubstring("bbbbb"))    # 輸出: 1 ("b")
print(lengthOfLongestSubstring("pwwkew"))   # 輸出: 3 ("wke")
print(lengthOfLongestSubstring(""))          # 輸出: 0


### 📌 LeetCode 209 - 長度最小的子陣列

**題目**：找出滿足 sum >= target 的最短子陣列長度。

**範例輸入**：`nums = [2,3,1,2,4,3], target = 7`
**範例輸出**：`2`（[4,3] 的和是 7，長度為 2）

### 💻 Python 程式碼（詳細註解）

def minSubArrayLen(target, nums):
    """
    長度最小的子陣列 - 可變大小滑動視窗（最短窗口）
    
    比喻：就像找「最短的隊伍」
    - right 不斷向右擴展，找滿足條件的隊伍
    - left 嘗試收縮，找更短的隊伍
    - 每次找到滿足條件的，就更新最短長度
    
    參數:
        target: int - 目標和
        nums: List[int] - 正整數陣列
    返回:
        int - 滿足條件的最短子陣列長度
    
    時間複雜度: O(n) - 每個元素最多被訪問兩次
    空間複雜度: O(1) - 只用幾個變數
    """
    
    # 邊界處理
    if not nums:
        return 0
    
    # ==========================================
    # 初始化
    # ==========================================
    n = len(nums)           # 陣列長度
    left = 0                # 窗口左邊界
    current_sum = 0        # 當前窗口的和
    min_length = float('inf')  # 最短長度，初始化為無限大
    
    # ==========================================
    # 滑動窗口遍歷
    # ==========================================
    # right 指標遍歷整個陣列
    for right in range(n):
        # 擴展窗口：把新元素加入
        current_sum += nums[right]
        
        # ======================================
        # 收縮窗口：嘗試縮短
        # ======================================
        # 當前 sum >= target 時，嘗試收縮
        # 收縮到 sum < target 為止（或 left 超過 right）
        while current_sum >= target:
            # 計算當前窗口長度
            current_length = right - left + 1
            
            # 更新最短長度
            min_length = min(min_length, current_length)
            
            # 收縮：從左邊移除元素
            current_sum -= nums[left]
            left += 1
    
    # 如果沒有找到，返回 0
    # 為什麼不是返回 min_length？因為可能是 inf（沒找到）
    return 0 if min_length == float('inf') else min_length


# 🧪 測試
print(minSubArrayLen(7, [2,3,1,2,4,3]))  # 輸出: 2
print(minSubArrayLen(4, [1,4,4]))         # 輸出: 1
print(minSubArrayLen(11, [1,1,1,1,1,1]))  # 輸出: 0


### 📌 LeetCode 567 - 字串的排列

**題目**：判斷 s2 是否包含 s1 的某種排列作為子串。

**範例輸入**：`s1 = "ab", s2 = "eidbaooo"`
**範例輸出**：`True`（s2 包含 "ab" 的排列："ba"）

### 💻 Python 程式碼（詳細註解）

def checkInclusion(s1, s2):
    """
    字串的排列 - 固定大小滑動視窗 + 字符計數
    
    比喻：就像玩「拼圖遊戲」
    - s1 是「原始拼圖」（需要重新排列）
    - s2 是「連續的幾塊區域」
    - 檢查哪個區域能拼出 s1 的排列
    
    核心技巧：使用兩個數組（計數器）比較
    - 一個是 s1 的字符計數
    - 一個是當前窗口的字符計數
    - 如果兩者相等，說明找到了！
    
    參數:
        s1: str - 短字串（要檢查的排列）
        s2: str - 長字串（要在裡面找）
    返回:
        bool - 是否包含
    
    時間複雜度: O(n) - n 是 s2 的長度
    空間複雜度: O(1) - 數組大小固定（26 個字母）
    """
    
    # 邊界處理
    if len(s1) > len(s2):
        return False
    
    # ==========================================
    # 初始化字符計數
    # ==========================================
    # s1_len 是窗口大小
    s1_len = len(s1)
    
    # 初始化兩個計數數組（26 個字母）
    # count1 存放 s1 的字符計數
    # count2 存放當前窗口的字符計數
    count1 = [0] * 26
    count2 = [0] * 26
    
    # ==========================================
    # 第一步：初始化第一個窗口
    # ==========================================
    # 統計 s1 的字符頻率
    for i in range(s1_len):
        # ord(c) - ord('a') 把 'a'-'z' 轉換為 0-25
        count1[ord(s1[i]) - ord('a')] += 1
        count2[ord(s2[i]) - ord('a')] += 1
    
    # ==========================================
    # 第二步：比較第一個窗口
    # ==========================================
    # 如果相等，說明找到了！
    if count1 == count2:
        return True
    
    # ==========================================
    # 第三步：滑動窗口遍歷
    # ==========================================
    # 從 s1_len 位置開始，到 s2 末端
    for i in range(s1_len, len(s2)):
        # 加入右邊的新字符
        # 例如：窗口向右移一格
        # new char 加入，右邊加入
        count2[ord(s2[i]) - ord('a')] += 1
        
        # 移除左邊的舊字符
        # window 向右移，左邊要移除
        count2[ord(s2[i - s1_len]) - ord('a')] -= 1
        
        # 比較兩個計數數組
        if count1 == count2:
            return True
    
    # 沒找到，返回 False
    return False


# 🧪 測試
print(checkInclusion("ab", "eidbaooo"))  # 輸出: True ("ba")
print(checkInclusion("ab", "eidboaoo"))  # 輸出: False
print(checkInclusion("abc", "bbbca"))    # 輸出: True ("bca")
