### 1️⃣ 基礎概念（用生活比喻，越詳細越好）

#### 🎯 什麼是陣列？

想象你家裡的**信箱**。每一個信箱格有一個固定的**編號**（索引），你可以：
- 直接跳到第 5 個信箱查看信件 → O(1) 時間
- 知道第 3 個信箱編號，就能直接打開它 → O(1) 時間

這就是陣列（Array）的核心概念：**連續的記憶體空間**，每個元素透過索引快速存取。

#### 📦 陣列的比喻

| 比喻 | 說明 |
|------|------|
| 停車場車位 | 每個車位有編號，車子（資料）固定停在指定位置 |
| 書架書格 | 每格只能放一本書，知道編號就能直接拿取 |
| 火車車廂 | 車廂編號連續，找第 10 節車廂不需要經過 1-9 節 |

#### 🔑 陣列的關鍵特性

1. **連續記憶體**：元素一個接一個排列在記憶體中
2. **固定大小**：宣告時決定大小（Python 列表會自動擴展）
3. **索引存取**：支援 O(1) 隨機存取

### 2️⃣ 核心知識（演算法解釋、時間/空間複雜度分析）

#### 🔍 陣列常見操作與複雜度

```
操作                    時間複雜度    空間複雜度    說明
─────────────────────────────────────────────────────────
存取元素 arr[i]          O(1)          O(1)        直接計算記憶體位址
搜尋元素                 O(n)          O(1)        需遍歷所有元素
插入元素                 O(n)          O(1)        可能需要搬移後續元素
刪除元素                 O(n)          O(1)        可能需要搬移後續元素
取得長度 len(arr)        O(1)          O(1)        Python 直接回傳
切片操作 arr[i:j]        O(k)          O(k)        k 為切片長度
```

#### 📊 為什麼存取是 O(1)？

```
記憶體位址公式：address = base + (index × element_size)

Example:
- 基礎位址：1000
- 每個元素大小：4 bytes
- 存取 index = 3 的元素
- 計算：1000 + (3 × 4) = 1012 → 直接到達！
```

#### ⚠️ 插入和刪除為何是 O(n)？

想象一個已經坐滿人的電影院，中間有人要坐下或離開：

```
初始：[A, B, C, D, E, F, G]
           ↑
        要在 C 和 D 之間插入 X

步驟 1: D, E, F, G 全部往後挪動一位
步驟 2: 放入 X

結果：[A, B, C, X, D, E, F, G]
        ↑                    ↑
     搬移起點           需搬移 4 個元素
```

**最壞情況**：在開頭插入 → 挪動 n 個元素 → O(n)

#### 📌 LeetCode 26. 移除重複元素（Remove Duplicates from Sorted Array）

**題目要求**：給定一個已排序陣列，原地移除重複元素，回傳新長度

```python
def removeDuplicates(nums):
    """
    移除已排序陣列中的重複元素
    
    解題思路：
    使用快慢指標技巧（雙指標）
    - 快指標：用來遍歷所有元素
    - 慢指標：用來追蹤不重複元素應該存放的位置
    
    為什麼這樣有效？
    因為陣列已排序，重複元素一定相鄰
    """
    
    # 邊界情況處理：空陣列或只有一個元素
    if not nums:
        return 0
    
    # 慢指標從索引 1 開始（因為第一個元素一定不重複）
    # 這個指標指向下一個不重複元素應該放的位置
    slow = 1
    
    # 快指標遍歷整個陣列（從索引 1 開始）
    for fast in range(1, len(nums)):
        # 當前元素 != 前一個元素 → 發現新值
        if nums[fast] != nums[fast - 1]:
            # 將新值放到慢指標位置
            nums[slow] = nums[fast]
            # 慢指標前進，準備下一個位置
            slow += 1
    
    # 回傳不重複元素的數量
    return slow


# ============== 測試範例 ==============
# Example 1:
nums1 = [1, 1, 2, 2, 3, 4, 4, 5]
result1 = removeDuplicates(nums1)
print(f"新長度: {result1}")           # 輸出: 5
print(f"陣列內容: {nums1[:result1]}")  # 輸出: [1, 2, 3, 4, 5]

# Example 2:
nums2 = [0, 0, 1, 1, 1, 2, 2, 3, 3, 4]
result2 = removeDuplicates(nums2)
print(f"新長度: {result2}")           # 輸出: 5
print(f"陣列內容: {nums2[:result2]}")  # 輸出: [0, 1, 2, 3, 4]
```

**時間複雜度分析**：
- 快指標走完陣列一次：O(n)
- 慢指標最多移動 n 次：O(n)
- **總時間：O(n)**

**空間複雜度分析**：
- 只使用兩個指標變數（slow, fast）
- 不需要額外資料結構
- **總空間：O(1)**

#### 📌 LeetCode 189. 旋轉陣列（Rotate Array）

**題目要求**：將陣列向右旋轉 k 個位置

```python
def rotate(nums, k):
    """
    原地旋轉陣列
    
    方法：三次反轉法
    原理：右旋 = 先反轉全部 → 再反轉前 k 個 → 再反轉後 n-k 個
    
    Example: [1,2,3,4,5,6,7], k=3
    步驟 1: 反轉全部    → [7,6,5,4,3,2,1]
    步驟 2: 反轉前 k 個  → [5,6,7,4,3,2,1]
    步驟 3: 反轉後 n-k 個 → [5,6,7,1,2,3,4]  ✓
    """
    
    n = len(nums)                     # 取得陣列長度
    k = k % n                          # 處理 k 大於 n 的情況
    
    # 定義反轉函數（指定範圍內的元素順序反轉）
    def reverse(start, end):
        # 雙指標從兩端向中間移動交換
        while start < end:
            # 交換 start 和 end 位置的元素
            nums[start], nums[end] = nums[end], nums[start]
            start += 1                # 左指標向右移動
            end -= 1                  # 右指標向左移動
    
    # 步驟 1: 反轉整個陣列
    reverse(0, n - 1)
    
    # 步驟 2: 反轉前 k 個元素
    reverse(0, k - 1)
    
    # 步驟 3: 反轉剩餘的 n-k 個元素
    reverse(k, n - 1)


# ============== 測試範例 ==============
nums = [1, 2, 3, 4, 5, 6, 7]
rotate(nums, 3)
print(nums)  # 輸出: [5, 6, 7, 1, 2, 3, 4]

nums2 = [-1, -100, 3, 99]
rotate(nums2, 2)
print(nums2)  # 輸出: [3, 99, -1, -100]
```

**時間複雜度分析**：
- 三次反轉，每次最多遍歷 n/2 個元素
- **總時間：O(n)**

**空間複雜度分析**：
- 只使用常數個指標變數
- **總空間：O(1)**
