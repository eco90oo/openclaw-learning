# 📚 Day 1：Python 基礎語法 — 變數、資料型態、邏輯控制

## 1️⃣ 基礎概念（用生活比喻）

### 🎯 什麼是程式？

想像一下，程式就像是「食譜」。廚師（電腦）按照食譜上的步驟一步一步做，最後做出美味的菜（結果）。

- **食譜** = 你的 Python 程式碼
- **廚師** = 電腦（Python 直譯器）
- **菜** = 程式執行的結果

### 🎯 變數是什麼？

**比喻**：變數就像是「貼紙」或「標籤」。

想像你有很多盒子，每個盒子裡放不同的東西：
- 盒子 A 放蘋果 → 貼上標籤「水果」
- 盒子 B 放襪子 → 貼上標籤「襪子」

在 Python 裡：
- 「盒子」= 記憶體空間
- 「標籤」= 變數名稱
- 「盒子裡的東西」= 變數的值

```python
水果 = "蘋果"  # 把 "蘋果" 放進盒子，貼上 "水果" 的標籤
襪子 = 3      # 把數字 3 放進盒子，貼上 "襪子" 的標籤
```

### 🎯 資料型態

**比喻**：就像不同的容器装不同的東西。

| 容器類型 | 可以装什麼 | Python 類型 |
|---------|-----------|------------|
| 籃子 | 文字、訊息 | `str`（字串） |
| 桶子 | 數字（整數） | `int`（整數） |
| 量杯 | 小數（精確測量） | `float`（浮點數） |
| 開關 | 是/否、對/錯 | `bool`（布林值） |

## 2️⃣ 語法解說（Python 程式碼）

### 📌 變數命名規則

```python
# ✅ 合法的命名
name = "小明"
age = 25
price = 99.9
is_student = True

# ❌ 不合法的命名（會出錯！）
# 2name = "小明"     # 不能用數字開頭
# my-name = "小明"   # 不能用連字符號
# if = 10            # 不能用關鍵字
```

### 📌 基本資料型態

```python
# 字串（文字）
name = "小明"
message = '你好，世界！'

# 整數（沒有小數點的數字）
age = 25
count = 100
year = 2026

# 浮點數（有小數點的數字）
price = 99.9
pi = 3.14159
height = 175.5

# 布林值（True/False）
is_active = True
is_empty = False
```

### 📌 print() — 輸出東西

```python
# 輸出文字
print("你好！")

# 輸出變數
name = "小明"
print(name)  # 會顯示：小明

# 混合輸出
age = 25
print("我叫" + name + "，今年" + str(age) + "歲")
# 會顯示：我叫小明，今年25歲

# 用 f-string（更方便！）
print(f"我叫{name}，今年{age}歲")
# 會顯示：我叫小明，今年25歲
```

### 📌 if/else — 條件判斷

```python
# 基本語法
age = 18

if age >= 18:
    print("成年人")
else:
    print("未成年人")

# 多重條件
score = 85

if score >= 90:
    print("A 等級")
elif score >= 80:
    print("B 等級")
elif score >= 70:
    print("C 等級")
else:
    print("需要加油！")
```

### 📌 for 迴圈 — 重複執行

```python
# 基本的 for 迴圈
for i in range(5):
    print(i)
# 輸出：0, 1, 2, 3, 4

# 遍歷列表
fruits = ["蘋果", "香蕉", "橘子"]
for fruit in fruits:
    print(fruit)
# 輸出：蘋果、香蕉、橘子

# 帶索引的遍歷
for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")
# 輸出：0: 蘋果、1: 香蕉、2: 橘子
```

### 📌 while 迴圈 — 條件為真時重複

```python
# 當條件成立時，不斷執行
count = 0
while count < 5:
    print(count)
    count = count + 1  # 記得增加計數器，否則會无限循環！
# 輸出：0, 1, 2, 3, 4
```

### 📌 input() — 讓使用者輸入

```python
# 讓使用者輸入文字
name = input("請輸入你的名字：")
print(f"你好，{name}！")

# 讓使用者輸入數字（需要轉換）
age = int(input("請輸入你的年齡："))
if age >= 18:
    print("你可以投票！")
else:
    print("你還不能投票！")
```

    result = num1 - num2
    print(f"結果：{num1} - {num2} = {result}")

    print(f"結果：{num1} * {num2} = {result}")

        result = num1 / num2
        print(f"結果：{num1} / {num2} = {result}")
    else:
        print("錯誤：不能除以 0！")
else:
    print("無效的選項")

## 4️⃣ 常見錯誤提醒

### ⚠️ 錯誤 1：變數未定義
```python
# 錯！
print(age)  # NameError: name 'age' is not defined

# 對！
age = 25
print(age)  # 輸出：25
```

### ⚠️ 錯誤 2：資料型態搞混
```python
# 錯！
name = "小明"
age = 18
print(name + age)  # TypeError: can only concatenate str (not "int") to str

# 對！
print(name + str(age))  # 輸出：小明18
# 或者
print(f"{name} {age}歲")  # 輸出：小明 18歲
```

### ⚠️ 錯誤 3：縮排錯誤
```python
# 錯！縮排不一致
if age >= 18:
print("成年人")  # IndentationError

# 對！要有縮排
if age >= 18:
    print("成年人")  # 正確的縮排
```

### ⚠️ 錯誤 4：無限迴圈
```python
# 錯！沒有停止條件
i = 0
while True:
    print(i)  # 永遠不會停止！

# 對！記得加計數器
i = 0
while i < 5:
    print(i)
    i = i + 1  # 記得增加計數器
```

### ⚠️ 錯誤 5：除以零
```python
# 錯！
result = 10 / 0  # ZeroDivisionError: division by zero

# 對！先檢查
if num2 != 0:
    result = num1 / num2
else:
    print("不能除以 0")
```
