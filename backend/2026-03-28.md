## 1️⃣ 基礎概念（用生活比喻，越詳細越好）

### 🧠 變數是什麼？
想像你在搬家，你需要一堆紙箱來放東西。每個紙箱外面貼上一張標籤，裡面放不同的東西。這些**紙箱**就像是電腦記憶體中的「空間」，而**標籤**就是「變數名稱」。

舉例來說：
- 有一個紙箱叫「體重」，裡面放数字 70
- 有一個紙箱叫「名字」，裡面放文字 "小明"
- 有一個紙箱叫「是否結婚」，裡面放「是」或「否」

在 Python 中，我們不需要先宣布紙箱的大小或類型，Python 會自動根據放的東西，幫你判斷這是什麼類型。這就是 Python 稱為「動態型別」語言的原因！

### 📦 資料型態三大天王
電腦世界裡，資料型態就像不同的容器：

| 型態 | 比喻 | Python 範例 | 用途 |
|------|------|-------------|------|
| **整數 (int)** | 完整一顆苹果，不能切 | `10`, `-5`, `100` | 計數、數學運算 |
| **浮點數 (float)** | 切成好幾份的苹果 | `3.14`, `-0.5` | 精確小數、科學計算 |
| **字串 (str)** | 一串文字組成的句子 | `"Hello"`, `'Python'` | 文字處理、訊息 |

### 🚦 if/else：如果...否則...
這就像生活中的選擇題：
- **如果**今天下雨 → 帶傘
- **否則**（不下雨）→ 不帶傘

或是停紅綠燈：
- **如果**是紅燈 → 停車
- **如果**是黃燈 → 減速
- **否則**（綠燈）→ 前行

### 🔄 for/while 迴圈：重複做一樣的事

**for 迴圈**就像是你去超商買東西，看著購物清單：
1. 第一項：牛奶 → 放到購物車
2. 第二項：麵包 → 放到購物車
3. 第三項：雞蛋 → 放到購物車
...清單上的項目都買完為止

**while 迴圈**就像是你在等人：
- **當**（while）對方還沒來，你就繼續等
- 什麼時候結束？等到對方來為止

## 2️⃣ 語法解說（Python 程式碼，每行都要有中文註解）

### 2.1 變數與資料型態

```python
# ====== 變數與基本資料型態 ======

# 建立一個叫 "name" 的紙箱，放進字串 "小明"
# 這行表示：讓 name 這個變數，存放 "小明" 這個字串
name = "小明"

# 建立一個叫 "age" 的紙箱，放進整數 25
# age 是 "年齡" 的意思，25 是阿拉伯數字（整數）
age = 25

# 建立一個叫 "height" 的紙箱，放進浮點數 170.5
# 人類身高常有小數點，所以用 float（浮點數）
height = 170.5

# 建立一個叫 "is_student" 的紙箱，放進布林值 True
# 布林值只有兩種：True（真/是）或 False（假/否）
is_student = True

# 用 print() 函式把這些變數印出來給你看
# end="\n" 表示每個 print 結果後換行（預設就是換行）
print(name)          # 輸出：小明
print(age)           # 輸出：25
print(height)        # 輸出：170.5
print(is_student)    # 輸出：True
```

### 2.2 type() 查看資料型態

```python
# ====== 查看變數的資料型態 ======

# type() 是 Python 內建的函式，可以告诉我们這個變數是什麼類型
print(type(name))        # 輸出：<class 'str'>，str 就是字串 string
print(type(age))         # 輸出：<class 'int'>，int 就是整數 integer
print(type(height))      # 輸出：<class 'float'>，float 就是浮點數
print(type(is_student))  # 輸出：<class 'bool'>，bool 就是布林值 boolean
```

### 2.3 if/else 條件判斷

```python
# ====== if/else 條件判斷 ======

# 情境：判斷年齡是否已經成年（18 歲以上）
age = 20  # 設定年齡為 20 歲

# 如果 age（年齡）大於或等於 18
if age >= 18:
    # 就執行這個區塊：印出 "已成年"
    print("已成年，可以投票")
# 否則（年齡小於 18）
else:
    # 就執行這個區塊：印出 "未成年"
    print("未成年，不能投票")
```

```python
# ====== if/elif/else 多重判斷 ======

# 情境：判斷考試成績的等級
score = 85  # 設定分數為 85 分

# 如果分數 >= 90
if score >= 90:
    print("成績等級：A")
# 否則如果分數 >= 80
elif score >= 80:
    print("成績等級：B")
# 否則如果分數 >= 70
elif score >= 70:
    print("成績等級：C")
# 否則如果分數 >= 60
elif score >= 60:
    print("成績等級：D")
# 以上的條件都不成立（分數小於 60）
else:
    print("成績等級：F，需要補考")
```

### 2.4 for 迴圈

```python
# ====== for 迴圈基礎 ======

# 建立一個清單（list），裡面有 5 個數字
# list（列表）就像是一排有編號的紙箱
numbers = [1, 2, 3, 4, 5]

# for 是「對每一個」的意思
# for n in numbers 的意思是：對 numbers 裡的每一個元素
# 依序取出來，存到變數 n 裡面，然後執行下面的程式碼
for n in numbers:
    # 把 n 印出來
    print(n)
```

```python
# ====== for 迴圈 + range() ======

# range(5) 會產生 0, 1, 2, 3, 4 這五個數字
# range(1, 6) 會產生 1, 2, 3, 4, 5 這五個數字
# range(起始值, 結束值, 間隔) - 注意：結束值不算喔！
for i in range(1, 6):
    print(f"第 {i} 次迴圈")  # f-string 是格式化字串
```

```python
# ====== for 迴圈遍歷字串 ======

# 字串其實也可以一個字一個字取出來
message = "Hello"

# 對 message 裡的每個字元（character）進行迴圈
for char in message:
    print(char)
# 輸出：H, e, l, l, o（每個字一行）
```

### 2.5 while 迴圈

```python
# ====== while 迴圈基礎 ======

# 情境：計算從 1 加到 5 的總和
# 一開始 sum 是 0
sum = 0
# 一開始 i 是 1
i = 1

# 當 i <= 5 的時候，就繼續執行下面的程式碼
while i <= 5:
    # 把 i 加到 sum 裡面
    sum = sum + i
    # 把 i 增加 1（否則會變成無限迴圈！）
    i = i + 1

# 迴圈結束後，印出總和
print(f"1+2+3+4+5 = {sum}")  # 輸出：15
```

```python
# ====== while 迴圈 + break（中斷） ======

# 情境：一直問使用者要不要繼續，直到他說不為止
# 注意：這是示範用，實際上通常用更優雅的方式
count = 0

# 當 True（永遠為真）的時候，持續執行
while True:
    count = count + 1
    print(f"這是第 {count} 次迴圈")
    
    # 如果 count 等於 3，就跳出迴圈
    if count == 3:
        print("達到 3 次，跳出迴圈！")
        break  # break 的意思是「停止這個迴圈」
```

### 2.6 邏輯運算子

```python
# ====== 邏輯運算子 ======

# and（且）：兩個條件都要滿足
age = 25
has_license = True

# 如果 age >= 18 且 has_license 為 True
if age >= 18 and has_license:
    print("可以開車")

# or（或）：至少一個條件滿足
is_weekend = False
is_holiday = True

# 如果 is_weekend 或 is_holiday 為 True
if is_weekend or is_holiday:
    print("今天不用上班")

# not（不是）：反轉布林值
is_sleeping = False

# 如果 is_sleeping 不是 True（即正在清醒）
if not is_sleeping:
    print("起床了！")
```

## 4️⃣ 常見錯誤提醒

### ❌ 錯誤 1：變數名稱大小寫搞混
Python 是**大小寫敏感**的語言，`Age` 和 `age` 是不同的變數！

```python
# 錯誤範例
age = 20
print(Age)  # 💥 會报错！NameError: name 'Age' is not defined
```

### ❌ 錯誤 2：忘記加冒號
if、for、while 後面記得加冒號 `:`，這是語法規定！

```python
# 錯誤範例
if age > 18    # 💥 少了冒號！
    print("成年")
```

### ❌ 錯誤 3：迴圈沒設結束條件（無限迴圈）
while 迴圈一定要記得遞增/遞減變數，否則會永遠執行下去！

```python
# 💥 危險！這會造成無限迴圈，電腦可能會當掉！
# i = 1
# while i > 0:
#     print(i)  # 永遠不會停止！
```

### ❌ 錯誤 4：縮排錯誤
Python 用縮排（indentation）來判斷程式碼屬於哪個區塊，千萬不要用 Tab 和空格混用！

```python
# 錯誤範例
if age > 18:
print("成年")   # 💥 縮排錯誤！這行應該要往右縮

# 正確寫法
if age > 18:
    print("成年")  # ✅ 正確：縮排 4 個空格
```

### ❌ 錯誤 5：字串引號用錯
Python 的字串可以用單引號 `''` 或雙引號 `""`，但不能混用！

```python
# 錯誤範例
message = 'Hello"   # 💥 開始用單引號，結束用雙引號，錯誤！

# 正確寫法
message = 'Hello'   # ✅ 單引號
message = "Hello"   # ✅ 雙引號
message = 'Hello'   # ✅ 配對正確
```

請複製以下程式碼並修改，練習變數、if/else 和 for 迴圈：

### 任務需求：
1. 建立一個購物清單（list），至少 5 樣商品
2. 用 for 迴圈逐一列出每樣商品，並編號
3. 設定一個預算（budget），用 if/else 判斷總金額是否超過預算
4. 如果超過，印出「💸 預算不足，需要省錢！」；如果沒超過，印出「✅ 買得起，开心！」

### 提示：
```python
# 購物清單範例
shopping_list = ["蘋果", "香蕉", "牛奶", "麵包", "雞蛋"]
budget = 100  # 預算 100 元

# 計算總金額（先假設每樣都是 20 元）
total = len(shopping_list) * 20  # len() 可以計算 list 的長度

# 用 for 迴圈列出商品
print("===== 購物清單 =====")
for i, item in enumerate(shopping_list):
    print(f"{i+1}. {item}")

print(f"\n總金額：{total} 元")
print(f"預算：{budget} 元")

# 判斷是否超過預算
if total > budget:
    print("💸 預算不足，需要省錢！")
else:
    print("✅ 買得起，开心！")
```
