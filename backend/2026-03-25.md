## 1️⃣ 基礎概念（用生活比喻）

### 🎯 什麼是 Pydantic？

想象一下：

- **一般變數** = 把東西隨便丟進盒子裡，不知道裡面是什麼
- **Pydantic** = 一個**有規定的盒子**，只能放入符合規定的東西

就像機場的行李安檢：
- 行李箱（相當於你的資料）必須符合規定
- 不能帶危險物品（類型錯誤）
- 液體不能超過 100ml（數值範圍限制）
- 護照必須有效（格式驗證）

Pydantic 就是程式的「安檢系統」！

### 🔍 什麼是 BaseModel？

**BaseModel** 是 Pydantic 的核心類別，相當於：

```
想像你在填寫入學申請表：
┌─────────────────────────────┐
│ 🎓 入學申請表（BaseModel）  │
├─────────────────────────────┤
│ 姓名：________（必填）       │
│ 年齡：________（數字）       │
│ Email：________（格式驗證）  │
└─────────────────────────────┘
```

這張表格規定了：
1. 需要哪些欄位
2. 欄位是什麼類型
3. 哪些是必填、哪些可選
4. 欄位有什麼限制

### 📏 什麼是 Field？

**Field** 用來定義欄位的詳細規則，就像表格旁邊的備註：

```
姓名：________ （必填，2-50個字）
年齡：________ （可選，0-150歲）
Email：________ （必須包含 @）
```

### 🔎 什麼是 Validator？

**Validator** 是自訂驗證規則，就像：

```
┌─────────────────────────────┐
│ 🔍 自訂驗證規則              │
├─────────────────────────────┤
│ • 密碼必須包含大小寫+數字    │
│ • 確認密碼必須跟密碼相同     │
│ • 年齡不能是負數             │
│ • 使用者名稱不能是保留字     │
└────────────────────────────────┘
```

```python
from pydantic import BaseModel, Field
from typing import Optional

class User(BaseModel):
    """用戶資料模型 - 完整版"""
    
    # 👉 必填欄位（使用 ... 表示）
    username: str = Field(
        ...,                      # ... 表示必填（required）
        min_length=3,             # 最小長度 3
        max_length=50,           # 最大長度 50
        description="用戶名稱"    # 描述（用于 API 文件）
    )
    
    # 👉 可選欄位（使用 None 或 default）
    age: Optional[int] = Field(
        None,                    # 可選，預設 None
        ge=0,                    # greater than or equal >= 0
        le=150,                  # less than or equal <= 150
        description="年齡（0-150）"
    )
    
    # 👉 有預設值的欄位
    is_active: bool = Field(
        True,                    # 預設值為 True
        description="是否啟用"
    )
    
    # 👉 電子郵件（自動驗證格式）
    email: str = Field(
        ...,
        description="電子郵件"
    )
```

### 🔎 自訂 Validator

```python
from pydantic import BaseModel, Field, validator
from typing import List

class UserRegister(BaseModel):
    """用戶註冊模型 - 包含自訂驗證"""
    
    username: str = Field(..., min_length=3, max_length=20)
    password: str = Field(..., min_length=8)
    confirm_password: str
    age: Optional[int] = Field(None, ge=0, le=150)
    emails: List[str]
    
    # 👉 驗證方法 1：檢查密碼確認
    @validator('confirm_password')
    def passwords_match(cls, v, values):
        """驗證兩次輸入的密碼是否相同"""
        if 'password' in values and v != values['password']:
            raise ValueError('密碼與確認密碼不一致')
        return v
    
    # 👉 驗證方法 2：檢查年齡是否合理
    @validator('age')
    def age_must_be_reasonable(cls, v):
        """驗證年齡是否合理"""
        if v is not None and v < 0:
            raise ValueError('年齡不能是負數')
        return v
    
    # 👉 驗證方法 3：檢查列表中的 Email 格式
    @validator('emails', each_item=True)
    def emails_must_be_valid(cls, v):
        """驗證每個 Email 格式"""
        if '@' not in v:
            raise ValueError(f'{v} 不是有效的 Email 格式')
        return v.lower()  # 統一轉為小寫
    
    # 👉 驗證方法 4：檢查使用者名稱（類別方法）
    @validator('username')
    def username_must_be_alphanumeric(cls, v):
        """驗證使用者名稱只能包含英文和數字"""
        if not v.isalnum():
            raise ValueError('使用者名稱只能包含英文和數字')
        return v

# 測試驗證
try:
    user = UserRegister(
        username="test123",
        password="password123",
        confirm_password="password123",  # 密碼不同會失敗
        age=25,
        emails=["test@example.com", "admin@example.com"]
    )
    print("驗證成功！")
    print(user.dict())
except Exception as e:
    print(f"驗證失敗：{e}")
```

### 🔧 進階：Root Validator

```python
from pydantic import BaseModel, Field, root_validator
from typing import Optional

class Order(BaseModel):
    """訂單模型 - 使用 Root Validator"""
    
    items: List[str]
    total_price: float
    shipping_fee: float = Field(0, description="運費")
    discount: float = Field(0, description="折扣")
    
    @root_validator
    def calculate_final_price(cls, values):
        """計算最終價格"""
        total = values.get('total_price', 0)
        shipping = values.get('shipping_fee', 0)
        discount = values.get('discount', 0)
        
        final_price = total + shipping - discount
        
        # 最終價格不能為負數
        if final_price < 0:
            raise ValueError('最終價格不能為負數')
        
        values['final_price'] = final_price
        return values

# 測試
order = Order(
    items=["商品A", "商品B"],
    total_price=1000,
    shipping_fee=100,
    discount=50
)
print(order.dict())
# 輸出：{'items': [...], 'total_price': 1000, 'shipping_fee': 100, 
#        'discount': 50, 'final_price': 1050}
```

### 🎯 Field 的各種限制

```python
from pydantic import BaseModel, Field, constr, conint, conlist
from typing import List

# 各種限制類型
class Product(BaseModel):
    """產品模型 - 各種欄位限制"""
    
    # 字串限制（使用 constr）
    name: constr(min_length=3, max_length=100)  # 字串長度限制
    
    # 數字限制（使用 conint）
    quantity: conint(ge=0, le=1000)  # 數量 0-1000
    price: float = Field(..., gt=0)  # 價格 > 0
    
    # 列表限制（使用 conlist）
    tags: conlist(str, min_items=1, max_items=5)  # 標籤 1-5 個
    
    # 百分比（0-100）
    discount_rate: float = Field(0, ge=0, le=100)

# 枚舉類型
from enum import Enum
from pydantic import BaseModel

class UserRole(str, Enum):
    """用戶角色枚舉"""
    ADMIN = "admin"
    USER = "user"
    GUEST = "guest"

class UserWithRole(BaseModel):
    """有角色的用戶"""
    name: str
    role: UserRole

# 使用
user = UserWithRole(name="小明", role="admin")
print(user.role)           # 輸出：admin
print(user.role.value)     # 輸出：admin
print(user.role.name)      # 輸出：USER_ROLE
```

### 📋 練習 2：整合 FastAPI

```python
# 檔案：api_with_pydantic.py
# FastAPI + Pydantic 完整範例

from fastapi import FastAPI, HTTPException, status
from pydantic import BaseModel, Field, EmailStr, validator
from typing import Optional, List
from datetime import datetime

app = FastAPI(title="會員管理系統 API")

# ========================================
# 資料模型定義
# ========================================

class MemberCreate(BaseModel):
    """會員註冊模型"""
    username: str = Field(..., min_length=3, max_length=20)
    email: EmailStr = Field(...)
    password: str = Field(..., min_length=8)
    confirm_password: str = Field(...)
    age: Optional[int] = Field(None, ge=13, le=150)
    
    @validator('confirm_password')
    def passwords_match(cls, v, values):
        if 'password' in values and v != values['password']:
            raise ValueError('密碼不一致')
        return v
    
    @validator('username')
    def username_alphanumeric(cls, v):
        if not v.isalnum():
            raise ValueError('使用者名稱只能包含英文和數字')
        return v.lower()

class MemberResponse(BaseModel):
    """會員回傳模型"""
    id: int
    username: str
    email: str
    age: Optional[int]
    created_at: datetime

class MemberUpdate(BaseModel):
    """會員更新模型"""
    email: Optional[EmailStr] = None
    age: Optional[int] = Field(None, ge=13, le=150)

# ========================================
# 模擬資料庫
# ========================================

members_db = []
next_id = 1

# ========================================
# API 端點
# ========================================

@app.get("/")
def root():
    """首頁"""
    return {"message": "歡迎來到會員管理系統", "docs": "訪問 /docs 查看 API 文件"}

@app.post("/members", response_model=MemberResponse, status_code=status.HTTP_201_CREATED)
def create_member(member: MemberCreate):
    """創建新會員"""
    global next_id
    
    # 檢查 Email 是否已存在
    for m in members_db:
        if m["email"] == member.email:
            raise HTTPException(status_code=400, detail="Email 已被使用")
    
    # 檢查使用者名稱是否已存在
    for m in members_db:
        if m["username"] == member.username:
            raise HTTPException(status_code=400, detail="使用者名稱已被使用")
    
    # 建立會員
    new_member = {
        "id": next_id,
        "username": member.username,
        "email": member.email,
        "password": member.password,  # 實際開發要雜湊！
        "age": member.age,
        "created_at": datetime.now()
    }
    
    members_db.append(new_member)
    next_id += 1
    
    # 回傳（不含密碼）
    return new_member

@app.get("/members", response_model=List[MemberResponse])
def get_members(
    min_age: Optional[int] = Field(None, ge=0, le=150, description="最小年齡"),
    max_age: Optional[int] = Field(None, ge=0, le=150, description="最大年齡")
):
    """取得所有會員（可篩選）"""
    results = members_db
    
    if min_age is not None:
        results = [m for m in results if m.get("age", 0) >= min_age]
    if max_age is not None:
        results = [m for m in results if m.get("age", 0) <= max_age]
    
    return results

@app.get("/members/{member_id}", response_model=MemberResponse)
def get_member(member_id: int):
    """取得特定會員"""
    for member in members_db:
        if member["id"] == member_id:
            return member
    raise HTTPException(status_code=404, detail="找不到該會員")

@app.patch("/members/{member_id}", response_model=MemberResponse)
def update_member(member_id: int, update_data: MemberUpdate):
    """更新會員資料"""
    for member in members_db:
        if member["id"] == member_id:
            # 只更新有提供的欄位
            if update_data.email is not None:
                member["email"] = update_data.email
            if update_data.age is not None:
                member["age"] = update_data.age
            return member
    raise HTTPException(status_code=404, detail="找不到該會員")

@app.delete("/members/{member_id}")
def delete_member(member_id: int):
    """刪除會員"""
    for i, member in enumerate(members_db):
        if member["id"] == member_id:
            members_db.pop(i)
            return {"message": "刪除成功", "id": member_id}
    raise HTTPException(status_code=404, detail="找不到該會員")

# ========================================
# 啟動伺服器
# ========================================

if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="0.0.0.0", port=8000)
```

```python
    def age_must_be_positive(cls, v):

            raise ValueError('年齡必須是正數')
        return v
```

```python
# product.py
from fastapi import FastAPI, HTTPException
from pydantic import BaseModel, Field
from typing import Optional, List

app = FastAPI()

# 產品模型
class Product(BaseModel):
    """產品模型"""
    name: str = Field(..., min_length=3, max_length=100, description="產品名稱")
    price: float = Field(..., gt=0, description="價格")
    description: Optional[str] = Field(None, description="產品描述")
    quantity: int = Field(0, ge=0, description="數量")

# 模擬資料庫
products_db = []
next_id = 1

@app.post("/products", status_code=201)
def create_product(product: Product):
    """新增產品"""
    global next_id
    
    new_product = {
        "id": next_id,
        **product.dict()
    }
    products_db.append(new_product)
    next_id += 1
    
    return new_product

@app.get("/products", response_model=List[Product])
def get_products():
    """取得所有產品"""
    return products_db

@app.get("/products/{product_id}")
def get_product(product_id: int):
    """取得單一產品"""
    for product in products_db:
        if product["id"] == product_id:
            return product
    raise HTTPException(status_code=404, detail="找不到產品")

if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="0.0.0.0", port=8000)
```
