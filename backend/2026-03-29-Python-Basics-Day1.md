## 1️⃣ 基礎概念（用生活比喻）

### 🎯 什麼是程式語言？

想象一下：

- **電腦 = 一個超級聽話的機器人員工**
- **程式碼 = 給這個員工的指令清單**
- **Python = 一種人類和機器人都能理解的「翻譯語言」**

就像我們寫下「請幫我買一杯咖啡」給員工，Python 就是讓電腦「聽懂」你要它做什麼的橋樑。

### 🎯 變數（Variable）- 像貼標籤的盒子

**生活比喻**：想象你有幾個收納盒，你在每個盒子上貼標籤（標籤 = 變數名），然後把東西放進去（東西 = 變數值）。

```
📦 盒子（名字叫「年齡」）→ 裡面放 25
📦 盒子（名字叫「名字」）→ 裡面放 "小明"
📦 盒子（名字叫「是學生」）→ 裡面放 True
```

### 🎯 資料型態（Data Types）- 盒子的種類

就像收納盒有不同的尺寸和用途：

| 資料型態 | 比喻 | 範例 |
|---------|------|------|
| `int` (整數) | 裝整數物品的盒子 | 25, 100, -5 |
| `float` (浮點數) | 裝有小數的盒子 | 3.14, 99.9 |
| `str` (字串) | 裝文字訊息的盒子 | "你好", 'hello' |
| `bool` (布林值) | 裝是非對錯的盒子 | True, False |

### 🎯 條件判斷（if/else）- 紅綠燈邏輯

**生活比喻**：過馬路時

```
如果 紅燈亮 → 停下來等待
否則（綠燈）→ 走過去
```

這就是 `if` 和 `else` 的邏輯！

### 🎯 迴圈（Loop）- 重複做同一件事

**生活比喻**：洗一疊碗

```
第一個碗 → 洗乾淨
第二個碗 → 洗乾淨
第三個碗 → 洗乾淨
...（直到洗完所有碗）
```

這就是 `for` 迴圈在做的事！

### 📝 2.1 變數建立

```python
# ===== 變數建立 =====
# Python 的變數不需要先宣告類型，直接給值就好！

height = 165.5  # float（浮點數）

is_student = True  # bool（布林值），True=是，False=否

# 建立一個叫 "成績" 的盒子，放入 None（代表「空」或「還沒決定」）
score = None  # None 代表「尚未設定」或「無值」
```

### 📝 2.4 條件判斷 if/elif/else

```python
# ===== 條件判斷 =====
# 如果...否則...的邏輯

年齡 = 20  # 可以改成不同的數字測試

# 基本語法：if 條件:（記得加冒號！）
if 年齡 >= 18:
    # 如果條件成立（True），執行這裡的程式碼
    # 注意：這裡要縮排（按 Tab 或 4 個空格）
    print("成年人，可以投票！")

# else：如果上面的條件不成立
else:
    print("未成年，還不能投票！")

# ===== 更複雜的條件：elif =====
分數 = 85  # 可以改成不同分數測試

if 分數 >= 90:
    print("成績：A（優秀）")
elif 分數 >= 80:
    print("成績：B（良好）")
elif 分數 >= 70:
    print("成績：C（中等）")
elif 分數 >= 60:
    print("成績：D（及格邊緣）")
else:
    print("成績：F（需要補考）")

# ===== 邏輯運算子：and / or / not =====
年齡 = 25
有錢 = True

# and：兩個條件都要成立
if 年齡 >= 18 and 有錢 == True:
    print("成年又有錢，兩個條件都滿足！")

# or：至少一個條件成立
是學生 = False
是老師 = True
if 是學生 or 是老師:
    print("是學生或老師，至少有一個身份！")

# not：反轉真假
不是學生 = not 是學生  # not True 變成 False
print("不是學生？", 不是學生)  # 輸出: False
```

### 📝 2.5 for 迴圈 - 固定次數重複

```python
# ===== for 迴圈 =====
# 用於「我知道要跑多少次」的情況

# 經典範例：從 1 數到 5
# "range(5)" 會產生 0, 1, 2, 3, 4（不包括 5）
for i in range(5):
    print("現在數到：", i)

# 輸出：
# 現在數到： 0
# 現在數到： 1
# 現在數到： 2
# 現在數到： 3
# 現在數到： 4

print("-" * 20)  # 分隔線

# 從 1 數到 5（用 range(1, 6)）
for i in range(1, 6):
    print("現在數到：", i)

# 輸出：
# 現在數到： 1
# 現在數到： 2
# 現在數到： 3
# 現在數到： 4
# 現在數到： 5

print("-" * 20)

# 跳著數（range(起點, 終點, 間隔)）
for i in range(0, 10, 2):
    print("偶數：", i)

# 輸出：
# 偶數： 0
# 偶數： 2
# 偶數： 4
# 偶數： 6
# 偶數： 8
```

### 📝 2.6 while 迴圈 - 條件成立時重複

```python
# ===== while 迴圈 =====
# 用於「我不知道要跑多少次，只要條件對就一直跑」的情況

# 範例：倒數計時
倒數 = 5

while 倒數 > 0:
    # 當 倒數 > 0 時，持續執行
    print("倒數：", 倒數)
    倒數 = 倒數 - 1  # 每次減 1

print("時間到！")

# ===== 讓使用者決定的範例 =====
# 這是一個簡單的密碼驗證程式
密碼 = ""
正確密碼 = "1234"

# 只要密碼錯誤，就一直問
while 密碼 != 正確密碼:
    密碼 = input("請輸入密碼：")  # input() 讓使用者輸入
    
    if 密碼 != 正確密碼:
        print("密碼錯誤，請重新輸入！")
    else:
        print("登入成功！")

# ===== 巢狀迴圈（迴圈裡還有迴圈）=====
# 範例：打印九九乘法表
for i in range(1, 10):     # 外層迴圈：1 到 9
    for j in range(1, 10): # 內層迴圈：1 到 9
        結果 = i * j
        # end=" " 代表不要換行，用空格分開
        print(f"{i} x {j} = {結果}", end=" ")
    print()  # 內層跑完後換行
```

### 📝 2.7 字串操作

```python
# ===== 字串常用操作 =====
名字 = "小明"

# 大小寫轉換
print(名字.upper())      # "小明" 沒有英文，但示範：全部變大寫
print(名字.lower())      # 全部變小寫

# 字串替換
句子 = "我喜歡貓"
新句子 = 句子.replace("貓", "狗")  # 把 "貓" 換成 "狗"
print(新句子)  # 輸出: "我喜歡狗"

# 切割字串
日期 = "2026-03-29"
切割後 = 日期.split("-")  # 用 "-" 切割
print(切割後)  # 輸出: ['2026', '03', '29']

# 字串切片（取一部分）
花草 = "玫瑰花百合花荷花"
取前三個字 = 花草[:3]   # 從開頭取 3 個字
取中間 = 花草[3:6]      # 從第 3 個取到第 6 個（不含第 6 個）
取最後三個字 = 花草[-3:] # 取最後 3 個字
print("前三個字:", 取前三個字)
print("中間:", 取中間)
print("最後三個字:", 取最後三個字)

# f-string（格式化字串）- 超級好用！
年齡 = 25
姓名 = "小明"
print(f"我叫{name}，今年{年齡}歲")  # 輸出: 我叫小明，今年25歲
```

### 🛠️ 練習 3：九九乘法表（用 for 迴圈）

```python
# ===== 打印九九乘法表 =====

# 使用雙層迴圈
for i in range(1, 10):      # i = 1 到 9
    for j in range(1, 10):  # j = 1 到 9
        result = i * j
        # f-string 格式化：固定寬度讓表格整齊
        print(f"{i}*{j}={result:2d}", end=" ")  # :2d 代表固定 2 位數
    print()  # 內圈跑完後換行，準備印下一列
```

### 🛠️ 練習 4：猜數字遊戲（結合 if/while/for）

```python
# ===== 猜數字遊戲 =====

import random  # 引入隨機模組

# 電腦隨機選一個數字（1-100）
答案是 random.randint(1, 100)

print("=== 猜數字遊戲 ===")
print("電腦已經選了一個 1-100 的數字，請猜猜看！")

猜測次數 = 0  # 記錄猜了幾次

while True:  # 一直猜，直到猜對
    # 讓使用者輸入數字
    猜測 = input("請輸入你的猜測：")
    
    # 轉換成整數（因為 input 回傳的是字串）
    猜測 = int(猜測)
    
    猜測次數 = 猜測次數 + 1  # 計次 +1
    
    if 猜測 == 答案:
        print(f"🎉 答對了！你花了 {猜測次數} 次猜對")
        break  # 跳出迴圈，遊戲結束
    elif 猜測 < 答案:
        print("太小了！再試試看～")
    else:
        print("太大了！再試試看～")

print("遊戲結束！")
```

## 4️⃣ 常見錯誤提醒

### ⚠️ 錯誤 1：縮排錯誤（IndentError）

```python
# ❌ 錯誤示範
if True:
print("這會出錯！")  # 沒有縮排

# ✅ 正確寫法
if True:
    print("這是對的！")  # 记得缩排
```

**⚡ 小提醒**：Python 用縮排（ indentation）來判斷程式碼的「歸屬」，千萬不要用 Tab 和空格混用！

### ⚠️ 錯誤 2：資料型態搞混

```python
# ❌ 錯誤示範
數字 = "10"  # 這是字串，不是數字！
結果 = 數字 + 5  # 會出錯！字串 + 數字 不行

# ✅ 正確寫法
數字 = 10     # 這是整數
結果 = 數字 + 5  # 可以正常運算

# 或者轉換類型
字串數字 = "10"
結果 = int(字串數字) + 5  # 把字串轉成整數
```

### ⚠️ 錯誤 3：迴圈沒設結束條件（無限迴圈）

```python
# ❌ 錯誤示範（小心！這會讓程式卡住）
count = 0
while count < 10:
    print(count)
    # 忘記讓 count 增加，永遠停在 0

# ✅ 正確寫法
count = 0
while count < 10:
    print(count)
    count = count + 1  # 每次增加，確保會離開迴圈
```

### ⚠️ 錯誤 4：變數名稱大小寫不同

```python
# Python 是「大小寫敏感」的！
年齡 = 25
年齡 = 30  # 這是同一個變數，重新賦值

age = 20  # 這是另一個不同的變數！

print(年齡)  # 輸出 30
print(age)   # 輸出 20
```

### ⚠️ 錯誤 5：忘記加冒號

```python
# ❌ 錯誤示範
if True     # 忘記加冒號
    print("錯誤")

# ✅ 正確寫法
if True:    # 記得加冒號
    print("正確")
```

#### 任務 5：BMI 計算機（進階）

```python
# BMI = 體重(公斤) / 身高(公尺)的平方
體重 = 70    # 公斤
身高 = 175   # 公分

# 轉換單位：公分轉公尺
身高公尺 = 身高 / 100

# 計算 BMI
bmi = 體重 / (身高公尺 ** 2)

# 判斷體重狀態
if bmi < 18.5:
    狀態 = "體重過輕"
elif bmi < 24:
    狀態 = "正常範圍"
elif bmi < 27:
    狀態 = "稍微過重"
else:
    狀態 = "肥胖"

print(f"身高 {身高} 公分，體重 {體重} 公斤")
print(f"BMI 是 {bmi:.2f}")  # :.2f 代表顯示到小數點後 2 位
print(f"體重狀態：{狀態}")
```
