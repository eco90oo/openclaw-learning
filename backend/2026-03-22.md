## 1️⃣ 基礎概念（用生活比喻）

### 什麼是 FastAPI？

想像一下：

- **傳統 Python 網頁伺服器（如 Flask）** 像是一家老餐廳，廚師什麼都要自己來，安排座位、點餐、端菜、上菜...效率較低
- **FastAPI** 像是一家現代化的**得來速餐廳**，有明確的窗口（接口）、標準化的流程、快速的服務

FastAPI 的特點：
- **快**：效能接近 Node.js 和 Go
- **易學**：語法簡單直觀
- **自動生成文件**：像餐廳有菜單讓客人知道可以點什麼
- **類型提示**：像餐廳的食材標籤，清楚標示種類

### 什麼是 API？

API = **Application Programming Interface**（應用程式介面）

生活比喻：
- 餐廳的**菜單**就是 API —— 告訴客人有哪些菜可以點
- **點餐**就是發送請求（Request）
- **廚房做好端出來**就是回應（Response）

### 什麼是 Route（路由）？

Route 就是 API 的**網址路徑**，像是：
- `/menu` → 菜單（GET）
- `/order` → 點餐（POST）
- `/order/{id}` → 查看某個訂單（GET）

## 2️⃣ 語法解說（Python 程式碼）

### 第一步：安裝 FastAPI

```python
# 在終端機執行
pip install fastapi uvicorn
```

解說：
- `fastapi` 是主框架
- `uvicorn` 是執行伺服器的引擎（像是餐廳的服務生）

### 第二步：第一個 FastAPI 程式

```python
# main.py
from fastapi import FastAPI

# 建立 FastAPI 實例（像開一家餐廳）
app = FastAPI()

# 建立 GET 路由（像在菜單上增加一道菜）
@app.get("/")
def read_root():
    return {"message": "歡迎光臨！這是我的第一個 API"}

# 建立帶参数的路由
@app.get("/hello/{name}")
def hello(name: str):
    return {"message": f"你好，{name}！"}

if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="0.0.0.0", port=8000)
```

### 第三步：Query Parameters（查詢參數）

Query Parameters 是 URL 中 `?` 後面的參數，像是：
```
/search?name=apple&price=100
```

```python
from fastapi import FastAPI, Query

app = FastAPI()

# Query Parameters 示例
@app.get("/search")
def search_products(
    name: str = Query(None, description="產品名稱"),
    min_price: float = Query(0, description="最小價格"),
    max_price: float = Query(9999, description="最大價格")
):
    # 模擬搜尋結果
    products = [
        {"name": "蘋果", "price": 50},
        {"name": "香蕉", "price": 30},
        {"name": "橘子", "price": 40},
    ]
    
    # 過濾結果
    results = [
        p for p in products 
        if p["price"] >= min_price and p["price"] <= max_price
    ]
    
    if name:
        results = [p for p in results if name in p["name"]]
    
    return {"results": results}
```

### 第四步：Path Parameters（路徑參數）

Path Parameters 是 URL 路徑中的變數，像是：
```
/user/123  →  id=123
/order/456 →  order_id=456
```

```python
from fastapi import FastAPI

app = FastAPI()

# 模擬資料庫
users = {
    "1": {"name": "小明", "age": 25},
    "2": {"name": "小華", "age": 30},
    "3": {"name": "小美", "age": 28},
}

@app.get("/user/{user_id}")
def get_user(user_id: str):
    """取得特定用戶"""
    if user_id not in users:
        return {"error": "用戶不存在"}
    return {"user": users[user_id]}

@app.get("/user/{user_id}/order/{order_id}")
def get_user_order(user_id: str, order_id: str):
    """取得用戶的特定訂單"""
    return {
        "user_id": user_id,
        "order_id": order_id,
        "order": {"item": "炸雞", "price": 150}
    }
```

### 步驟 2：建立 main.py

```python
# main.py
from fastapi import FastAPI, Query

app = FastAPI()

# 模擬產品資料
products = [
    {"id": 1, "name": "筆電", "price": 25000},
    {"id": 2, "name": "手機", "price": 15000},
    {"id": 3, "name": "平板", "price": 12000},
    {"id": 4, "name": "耳機", "price": 3000},
    {"id": 5, "name": "鍵盤", "price": 1500},
]

@app.get("/")
def read_root():
    """首頁"""
    return {
        "message": "歡迎來到產品 API",
        "docs": "請訪問 /docs 查看 API 文件"
    }

@app.get("/products")
def get_products(
    min_price: float = Query(0, description="最小價格"),
    max_price: float = Query(100000, description="最大價格"),
    name: str = Query(None, description="產品名稱關鍵字")
):
    """取得產品列表（支持價格篩選）"""
    results = [
        p for p in products 
        if min_price <= p["price"] <= max_price
    ]
    
    if name:
        results = [p for p in results if name in p["name"]]
    
    return {"total": len(results), "products": results}

@app.get("/products/{product_id}")
def get_product(product_id: int):
    """取得單一產品"""
    for p in products:
        if p["id"] == product_id:
            return {"product": p}
    return {"error": "產品不存在"}

# 啟動伺服器
if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="0.0.0.0", port=8000)
```

## 4️⃣ 常見錯誤提醒

### ❌ 忘記加 `@app.get()` 裝飾器

```python
# 錯誤 ❌
def read_root():
    return {"message": "hello"}

# 正確 ✅
@app.get("/")
def read_root():
    return {"message": "hello"}
```

### ❌ 路徑參數類型錯誤

```python
# 錯誤 ❌ - user_id 應該是 str，但傳入數字會造成問題
@app.get("/user/{user_id}")
def get_user(user_id: int):  # 類型要對應
    ...

# 正確 ✅
@app.get("/user/{user_id}")
def get_user(user_id: str):
    ...
```

### ❌ Query Parameters 寫錯位置

```python
# 錯誤 ❌ - Query Parameters 放在路徑中
@app.get("/products?min_price=100")
def get_products():
    ...

# 正確 ✅ - Query Parameters 在 ? 後面
@app.get("/products")
def get_products(min_price: int):
    ...
```

### ❌ 忘記 return

```python
# 錯誤 ❌
@app.get("/test")
def test():
    print("Hello")  # 這只是print，不會回傳

# 正確 ✅
@app.get("/test")
def test():
    return {"message": "Hello"}
```

### 📝 練習解答

```python
# basic_task.py
from fastapi import FastAPI, Query

app = FastAPI()

@app.get("/greeting")
def greeting(name: str = Query("朋友"), hour: int = Query(12)):
    """問候語 API"""
    if 6 <= hour < 12:
        greeting_msg = "早安"
    elif 12 <= hour < 18:
        greeting_msg = "午安"
    else:
        greeting_msg = "晚安"
    
    return {"message": f"{name}，{greeting_msg}！"}

@app.get("/calculator")
def calculator(
    a: float = Query(..., description="第一個數字"),
    b: float = Query(..., description="第二個數字"),
    operation: str = Query(..., description="運算符號：add, subtract, multiply, divide")
):
    """計算機 API"""
    operations = {
        "add": a + b,
        "subtract": a - b,
        "multiply": a * b,
        "divide": a / b if b != 0 else "錯誤：不能除以零"
    }
    
    result = operations.get(operation, "不支援的運算")
    return {"a": a, "b": b, "operation": operation, "result": result}

if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="0.0.0.0", port=8000)
```

執行後測試：
- `http://localhost:8000/greeting?name=小明&hour=9`
- `http://localhost:8000/calculator?a=10&b=5&operation=multiply`
