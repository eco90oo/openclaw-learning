## 1️⃣ 基礎概念（用生活比喻，越詳細越好）

### 🎯 什麼是函式？
想象你在餐廳點餐：
- 你跟服務生說「我要一份蛋炒飯」（就像呼叫一個函式）
- 廚房收到指令後，開始備料、炒飯、裝盤（函式內部執行的程式碼）
- 最後端出一盤香噴噴的蛋炒飯給你（函式回傳的結果）

**比喻**：函式就像「 recipe（食譜）」：
- 輸入：食材（參數）
- 處理：烹飪步驟（函式內的程式碼）
- 輸出：美味料理（回傳值）

> **為什麼要用函式？**
> - **重複使用**：不用每次都重新寫一樣的程式碼
> - **易於維護**：改一處就能更新所有用到的地方
> - **分工合作**：每個人負責寫不同的函式，最後組合起來

---

### 📦 什麼是模組？
就像「工具箱」：
- 廚房裡有各種工具：刀、砧板、鍋鏟、碗盤...
- 這些工具不是平時隨便亂放，而是分門別類放在不同的盒子裡
- 需要用刀的時候，去「刀具盒」找；需要碗盤的時候，去「餐具盒」找

**比喻**：模組就像「模組化傢俱」：
- 每個抽屜（模組）有不同的功能
- 需要什麼功能，就打開對應的抽屜（import 對應的模組）
- 這樣桌面（程式碼）不會亂成一團

> **Python 的模組例子**：
> - `math` 數學工具箱（sqrt, sin, cos...）
> - `random` 隨機工具箱（擲骰子、亂數...)
> - `datetime` 時間工具箱（日期、時間...）

---

### 🔄 參數傳遞是什麼？
就像「傳話遊戲」：
- 你跟 A 說：「幫我買一杯珍珠奶茶」
- A 把話傳給 B：「幫我買一杯珍珠奶茶」
- B 去買珍珠奶茶給你

**比喻**：參數就像「訂單」：
- 你給餐廳一張訂單（傳入參數）
- 餐廳按照訂單內容製作（處理）
- 端出你點的料理（回傳結果）

## 2️⃣ 語法解說（Python 程式碼，每行都要有中文註解）

### 🚀 建立第一個函式

```python
# 定義一個叫做 greet 的函式
# def 是「define」的縮寫，意思是「定義」
# greet 是函式的名字（你可以自己取）
# () 裡面放參數，這裡先放空的
def greet():
    # 函式裡面的程式碼必須縮排（按 Tab 或 4 個空格）
    # 這行是要回傳的內容
    return "你好！歡迎來到 Python 世界！"

# 呼叫函式：叫出這個函式來執行
# 就像點餐時喊「我要一份蛋炒飯」
result = greet()

# 印出結果
print(result)  # 輸出：你好！歡迎來到 Python 世界！
```

### 📥 帶參數的函式

```python
# 定義一個函式，接受一個參數叫做 name
# name 就像是一个「容器」，用來接收外面傳進來的值
def say_hello(name):
    # f-string：Python 3.6+ 的格式化字串方法
    # 把 {name} 的位置替換成變數的值
    return f"你好，{name}！歡迎光臨！"

# 呼叫時傳入 "小明" 作為參數
message = say_hello("小明")
print(message)  # 輸出：你好，小明！歡迎光臨！

# 再傳入不同的值測試
print(say_hello("大明"))  # 輸出：你好，大明！歡迎光臨！
```

### 🔢 多個參數

# 定義一個計算總價的函式
# 接受兩個參數：price（單價）和 quantity（數量）
def calculate_total(price, quantity):

# 定義一個函式，參數有預設值

### 🔄 參數傳遞的方式：位置 vs 關鍵字

```python
# 定義一個函式，計算會員折扣
def calculate_discount(price, discount_rate, member_level):
    # 計算折數後的價格
    final_price = price * (1 - discount_rate)
    return f"會員等級：{member_level}，折後價格：{final_price} 元"

# 方法1：按位置順序傳遞（位置引數）
result1 = calculate_discount(1000, 0.2, "金牌會員")
print(result1)

# 方法2：用關鍵字傳遞（關鍵字引數）
# 這樣不用記順序，更清楚
result2 = calculate_discount(
    price=1000,
    discount_rate=0.2,
    member_level="金牌會員"
)
print(result2)

# 方法3：混用（但位置參數要在關鍵字參數前面）
result3 = calculate_discount(1000, member_level="銀牌會員", discount_rate=0.1)
print(result3)
```

### 📤 回傳多個值（回傳 tuple）

```python
# 定義一個函式，回傳多個值
# Python 會自動把它們包成一個 tuple（元組）
def get_user_info():
    name = "小明"
    age = 25
    city = "台北"
    # 回傳三個值
    return name, age, city

# 接收回傳的 tuple
user_data = get_user_info()
print(user_data)  # 輸出：('小明', 25, '台北')

# 可以用解包（unpacking）的方式分別接收
name, age, city = get_user_data()
print(f"姓名：{name}，年齡：{age}，城市：{city}")
```

### 📦 import 模組

```python
# 方法1：import 整個模組
import math  # 匯入數學工具箱

# 使用時需要加上模組名稱
result = math.sqrt(16)  # 計算平方根，結果是 4.0
print(result)

# 方法2：只 import 需要的東西
from math import pi, sin  # 只匯入 pi 和 sin

# 使用時可以直接用，不用加模組名稱
print(pi)  # 3.141592653589793
print(sin(0))  # 0.0

# 方法3：幫模組取個別名（簡寫）
import datetime as dt  # 把 datetime 簡寫成 dt

now = dt.datetime.now()  # 取得現在時間
print(now)
```

### 🛠️ 建立自己的模組

假設我們建立一個檔案叫 `my_utils.py`：

```python
# my_utils.py（這是一個自訂模組）

# 定義一個計算BMI的函式
def calculate_bmi(height, weight):
    """
    計算身體質量指數（BMI）
    height: 身高（公尺）
    weight: 體重（公斤）
    """
    bmi = weight / (height ** 2)
    return round(bmi, 2)

# 定義一個判斷BMI等級的函式
def get_bmi_level(bmi):
    if bmi < 18.5:
        return "體重過輕"
    elif bmi < 24:
        return "正常範圍"
    elif bmi < 27:
        return "輕度肥胖"
    else:
        return "中度或重度肥胖"
```

然後在另一個檔案使用：

```python
# main.py

# 匯入自己的模組
import my_utils

# 使用模組裡的函式
height = 1.7  # 170公分 = 1.7公尺
weight = 65   # 65公斤

bmi = my_utils.calculate_bmi(height, weight)
level = my_utils.get_bmi_level(bmi)

print(f"BMI：{bmi}")  # 輸出：BMI：22.49
print(f"等級：{level}")  # 輸出：等級：正常範圍
```

### 🔁 *args 和 **kwargs（彈性參數）

```python
# *args：接收不確定的多個位置參數
# args 會變成一個 tuple
def print_all(*args):
    print("收到的參數：")
    for item in args:
        print(f"  - {item}")

print_all("蘋果", "香蕉", "橘子")
# 輸出：
# 收到的參數：
#   - 蘋果
#   - 香蕉
#   - 橘子

# **kwargs：接收不確定的多個關鍵字參數
# kwargs 會變成一個 dict（字典）
def print_info(**kwargs):
    print("收到的資訊：")
    for key, value in kwargs.items():
        print(f"  {key}: {value}")

print_info(name="小明", age=25, city="台北")
# 輸出：
# 收到的資訊：
#   name: 小明
#   age: 25
#   city: 台北
```

```python
# step3.py - 第三步：變成一個小工具（整合版）

def calculator(num1, num2, operation):
    """
    簡易計算機
    num1: 第一個數字
    num2: 第二個數字
    operation: 運算符號（+、-、*、/）
    """
    if operation == "+":
        return num1 + num2
    elif operation == "-":
        return num1 - num2
    elif operation == "*":
        return num1 * num2
    elif operation == "/":
        if num2 == 0:
            return "錯誤：除數不能為0"
        return num1 / num2
    else:
        return "錯誤：不支援的運算符號"

# 使用計算機
print(calculator(100, 20, "+"))  # 120
print(calculator(100, 20, "-"))  # 80
print(calculator(100, 20, "*"))  # 2000
print(calculator(100, 20, "/"))  # 5.0
```

## 4️⃣ 常見錯誤提醒

### ❌ 錯誤1：忘記缩排

```python
# 錯誤示範
def greet():
return "你好"  # 沒有縮排，會報錯！

# 正確寫法
def greet():
    return "你好"  # 必須缩排
```

### ❌ 錯誤2：參數數量不對

```python
def add(a, b):
    return a + b

# 錯誤：只傳一個參數
# add(10)  # 會報錯：缺少1個必要參數

# 正確
add(10, 5)  # 15
```

### ❌ 錯誤3：搞混位置參數和關鍵字參數

```python
def greet(name, greeting="你好"):
    return f"{greeting}，{name}！"

# 錯誤：關鍵字參數放在位置參數前面
# greet(name="小明", "早安")  # 語法錯誤！

# 正確
greet("小明", greeting="早安")  # 早安，小明！
```

### ❌ 錯誤4：模組路徑錯誤

```python
# 假設 my_module.py 在資料夾 utils/ 裡面
# 錯誤：直接 import
# import my_module  # 找不到！

# 正確：需要加上資料夾路徑
from utils import my_module

result = my_module.some_function()
```
