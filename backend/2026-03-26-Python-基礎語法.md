## 1️⃣ 基礎概念（用生活比喻，越詳細越好）

### 🎯 變數是什麼？
想像你有個「標籤貼紙」。當你說：
```python
name = "小明"
```
意思是：拿一張貼紙，寫上 `name`，然後把「小明」這個名字貼上去。

這張貼紙（變數）可以隨時撕下來，貼上新的東西：
```python
name = "大明"  # 撕下舊的，貼上新的
```

**比喻**：變數就像便利貼，你可以隨時改寫上面的內容，但標籤名稱（name）保持不變。

---

### 📦 資料型態
Python 裡的資料就像不同種類的盒子：

| 型態 | 盒子類型 | 範例 | 用途 |
|------|---------|------|------|
| `str` | 文字盒 | `"Hello"` | 存放文字 |
| `int` | 整數盒 | `18`、`100` | 存放數字（無小數點） |
| `float` | 小數盒 | `3.14`、`19.9` | 存放有小數點的數字 |
| `bool` | 開關盒 | `True`、`False` | 存放真假值（對/錯） |

**比喻**：
- `str` = 紙盒（只能放文字）
- `int` = 透明盒（放整數）
- `float` = 有刻度的量杯（放小數）
- `bool` = 電燈開關（只有開/關兩種狀態）

---

### 🔀 if/else（如果...否則...）
這就是「決策時刻」！就像：
- **如果**今天下雨 → 帶傘
- **否則**（不下雨）→ 不帶傘

```python
if 條件:
    # 條件成立時執行的程式碼
else:
    # 條件不成立時執行的程式碼
```

**比喻**：就像紅綠燈，根據不同情況走不同道路。

---

### 🔄 for/while 迴圈
**for 迴圈**：像是「按照清單順序處理每一項」
- 「對清單上的每個人，都說聲嗨」

**while 迴圈**：像是「直到...為止」
- 「一直吃到飽為止」（直到你說飽了）

## 2️⃣ 語法解說（Python 程式碼，每行都要有中文註解）

### 🟢 變數與基本資料型態

```python
# ===== 變數定義 =====
# 定義一個叫 name 的變數，裡面存放文字 "小明"
# str = string = 字串 = 一串文字
name = "小明"

# 定義一個叫 age 的變數，裡面存放整數 18
# int = integer = 整數（沒有小數點）
age = 18

# 定義一個叫 height 的變數，裡面存放小數 165.5
# float = floating point = 浮點數（小數）
height = 165.5

# 定義一個叫 is_student 的變數，存放 True（真）
# bool = boolean = 布林值（True/False）
is_student = True

# ===== 打印輸出 =====
# print() 是 Python 用來把內容顯示在螢幕上的函式
# f-string 是「格式化字串」，用 f"" 包住，在裡面用 {變數} 插入變數
print(f"姓名：{name}")      # 輸出：姓名：小明
print(f"年齡：{age}")      # 輸出：年齡：18
print(f"身高：{height}")   # 輸出：身高：165.5
print(f"是學生嗎？{is_student}")  # 輸出：是學生嗎？True
```

### 🟢 if/else 條件判斷

```python
# ===== 基本 if/else =====
# 假設年齡是 18
age = 18

# 如果 age >= 18（年齡大於或等於 18）
if age >= 18:
    # 這行會執行（因為 18 >= 18 為 True）
    print("你是成年人！")
else:
    # 如果上面不成立，就執行這裡
    print("你是未成年人！")

# ===== if/elif/else（多條件） =====
score = 85  # 分數

# 如果分數 >= 90
if score >= 90:
    print("成績：A")
# 否則如果分數 >= 80
elif score >= 80:
    print("成績：B")
# 否則如果分數 >= 70
elif score >= 70:
    print("成績：C")
# 上述都不成立
else:
    print("成績：D")

# ===== 邏輯運算子 =====
# and（且）：兩個條件都要成立
# or（或）：至少一個條件成立
# not（不是）：反轉真假值

age = 25
has_license = True

# AND：年齡 >= 18 且 有駕照
if age >= 18 and has_license:
    print("可以開車！")

# OR：年齡 < 18 或 沒駕照
if age < 18 or not has_license:
    print("不能開車！")
```

### 🟢 for 迴圈

```python
# ===== 基本 for 迴圈 =====
# 有一个列表（list），像是「水果籃」
fruits = ["蘋果", "香蕉", "橘子"]

# for fruit in fruits 的意思是：
# 「從水果籃裡，一次拿一個水果出來」
for fruit in fruits:
    print(f"我喜歡吃：{fruit}")

# 輸出：
# 我喜歡吃：蘋果
# 我喜歡吃：香蕉
# 我喜歡吃：橘子

# ===== range() 產生數字序列 =====
# range(5) 會產生 0, 1, 2, 3, 4（不包含 5）
for i in range(5):
    print(f"第 {i} 次")

# 輸出：
# 第 0 次
# 第 1 次
# 第 2 次
# 第 3 次
# 第 4 次

# range(1, 6) 產生 1, 2, 3, 4, 5（從1開始，到6之前）
for i in range(1, 6):
    print(f"數字：{i}")

# ===== enumerate() 同時取得索引和值 =====
colors = ["紅", "綠", "藍"]
# enumerate 會回傳 (索引, 值) 的 tuple
for index, color in enumerate(colors):
    print(f"第 {index} 個顏色是：{color}")
```

### 🟢 while 迴圈

```python
# ===== 基本 while 迴圈 =====
# while 的意思是「當...為真時，就一直做」
count = 0

# 當 count < 3 時，就一直執行迴圈裡的程式碼
while count < 3:
    print(f"計數：{count}")
    count = count + 1  # 每次加 1

# 輸出：
# 計數：0
# 計數：1
# 計數：2
# 當 count 變成 3 時，3 < 3 為 False，迴圈結束

# ===== break（打斷迴圈） =====
# break 會立即結束整個迴圈
i = 0
while True:  # 這是「無窮迴圈」，會一直跑
    print(f"i = {i}")
    if i == 3:  # 如果 i 等於 3
        print("找到目標，打破迴圈！")
        break  # 強制結束迴圈
    i = i + 1

# ===== continue（跳過這次） =====
# continue 會跳過這一次迴圈，直接進入下一次
for num in range(5):
    if num == 2:  # 如果是 2
        continue  # 跳過這次，不執行下面的 print
    print(f"數字：{num}")
# 輸出：0, 1, 3, 4（跳過了 2）
```

### 🟢 資料型態轉換

```python
# ===== 型態轉換（Type Conversion）=====
# str → int：文字轉數字
num_str = "18"
num_int = int(num_str)  # 轉成整數 18

# int → str：數字轉文字
age = 18
age_str = str(age)  # 轉成 "18"

# str → float：文字轉小數
price_str = "99.9"
price_float = float(price_str)  # 轉成 99.9

# float → int：小數轉整數（會去掉小數部分）
pi = 3.14
pi_int = int(pi)  # 轉成 3（不是四捨五入！）

# ===== type() 檢查型態 =====
x = 10
print(type(x))  # 輸出：<class 'int'>

y = "Hello"
print(type(y))  # 輸出：<class 'str'>

z = True
print(type(z))  # 輸出：<class 'bool'>
```

for i in range(1, 10):

### 📝 練習四：while 迴圈實作

```python
# 模擬「猜數字」遊戲（電腦選一個數，你來猜）

import random  # 引入隨機模組

target = random.randint(1, 10)  # 電腦選 1-10 之間的數
guess = 0  # 初始化猜測值
attempts = 0  # 嘗試次數

# 只要猜錯，就繼續猜
while guess != target:
    # 讓使用者輸入數字
    guess = int(input("請猜一個 1-10 的數字："))
    attempts = attempts + 1
    
    if guess < target:
        print("太小了！再大一點！")
    elif guess > target:
        print("太大了！再小一點！")
    else:
        print(f"🎉 猜對了！答案是 {target}")
        print(f"你總共猜了 {attempts} 次")
```

## 4️⃣ 常見錯誤提醒

### ❌ 錯誤 1：變數名稱錯誤（大小寫）
```python
# 錯誤示範
Name = "小明"
print(name)  # 錯誤！name 不等於 Name
```
**提醒**：Python 大小寫敏感，`Name` 和 `name` 是不同的變數！

---

### ❌ 錯誤 2：縮排錯誤（IndentationError）
```python
# 錯誤示範
if age >= 18:
print("成年人")  # 沒有縮排，會報錯！
```
**提醒**：if/else/for/while 裡的程式碼**必須**縮排（通常空 4 格或 1 個 Tab）

---

### ❌ 錯誤 3：型態不匹配
```python
# 錯誤示範
age = "18"  # age 是字串
print(age + 1)  # 錯誤！不能把字串和數字相加
```
**修正**：
```python
age = int("18")  # 先轉成數字
print(age + 1)   # 輸出：19
```

---

### ❌ 錯誤 4：while 迴圈沒設結束條件（無窮迴圈）
```python
# 錯誤示範
while True:
    print("永遠不會停！")
```
**提醒**：while 迴圈一定要有讓它停止的條件，否則程式會卡住！

---

### ❌ 錯誤 5：使用未定義的變數
```python
# 錯誤示範
print(my_name)  # my_name 從未被定義，會報 NameError
```
**提醒**：使用變數前一定要先賦值（給它一個值）
