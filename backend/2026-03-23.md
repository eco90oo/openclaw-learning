昨天我們學會了用 GET 方式讀取資料（像是去餐廳看菜單），今天要學會**接收客人提交的資料**——這就是 **POST 請求**！我們還會學會用 **Pydantic Model** 來驗證資料，就像餐廳要檢查客人點的餐是否有效。

## 1️⃣ 基礎概念（用生活比喻）

### 什麼是 POST 請求？

| GET vs POST | 比喻 |
|-------------|------|
| **GET** | 像是**看菜單**——只讀取資訊，不會改變什麼 |
| **POST** | 像是**下訂單**——把資料送給廚房，廚房會做事（創建新訂單）|

POST 的特性：
- 可以攜帶**大量資料**（不像 URL 有長度限制）
- 不會在網址列顯示（更安全）
- 通常用於**創建**或**提交**資料

生活例子：
- 填寫註冊表單 → POST
- 發布文章 → POST  
- 購物結帳 → POST

### 什麼是 Request Body（請求主體）？

當我們用 POST 送資料時，資料放在哪裡？

```
HTTP 請求就像一封信：
┌─────────────────────────┐
│ 信封（Header）           │  → 誰寄的？什麼格式？
├─────────────────────────┤
│ 信紙內容（Body）         │  → 實際要處理的資料
└─────────────────────────┘
```

FastAPI 中，Body 通常是 **JSON** 格式。

### 什麼是 Pydantic Model？

**Pydantic** 是一個 Python 庫，用於**定義資料模型**和**驗證資料**。

比喻：
- 想像你在填寫**入學申請表**
- 表格上清楚寫著：姓名（必填）、年齡（數字）、Email（必須有 @）
- Pydantic 就像這張表格，幫你檢查資料是否正確

### 第二步：定義 Pydantic Model

```python
from pydantic import BaseModel, Field, EmailStr
from typing import Optional

class User(BaseModel):
    """用戶資料模型"""
    name: str = Field(..., description="用戶名稱，必填")
    age: int = Field(None, description="年齡，可選")
    email: EmailStr = Field(..., description="電子郵件")
    is_active: bool = Field(True, description="是否啟用")
```

解說：
- `str`、`int`、`bool`、`float` 是**基本類型**
- `Field(...)` 表示這個欄位**必填**
- `Field(None)` 表示可選
- `EmailStr` 是特殊類型，會自動驗證 Email 格式
- `Field(default=...)` 設定預設值

### 第三步：建立 POST API

```python
from fastapi import FastAPI
from pydantic import BaseModel, Field

app = FastAPI()

# 定義資料模型
class Product(BaseModel):
    name: str = Field(..., description="產品名稱")
    price: float = Field(..., description="產品價格")
    description: Optional[str] = Field(None, description="產品描述")

# 模擬資料庫
products_db = []
next_id = 1

# POST - 創建產品
@app.post("/products")
def create_product(product: Product):
    """創建新產品"""
    global next_id
    
    new_product = {
        "id": next_id,
        "name": product.name,
        "price": product.price,
        "description": product.description
    }
    
    products_db.append(new_product)
    next_id += 1
    
    return {
        "message": "產品創建成功",
        "product": new_product
    }

# GET - 取得所有產品
@app.get("/products")
def get_products():
    return {"products": products_db}
```

### 第四步：更複雜的 Model（巢狀結構）

```python
from typing import List
from pydantic import BaseModel, Field

class Item(BaseModel):
    """訂單中的單一商品"""
    name: str
    quantity: int = Field(..., gt=0, description="數量必須大於 0")
    price: float

class Order(BaseModel):
    """完整訂單"""
    customer_name: str
    items: List[Item]
    total_price: float
    
    # 自訂驗證方法
    def calculate_total(self) -> float:
        return sum(item.quantity * item.price for item in self.items)

# API 端點
orders_db = []

@app.post("/orders")
def create_order(order: Order):
    """創建訂單"""
    # 驗證總金額是否正確
    calculated = order.calculate_total()
    
    if abs(calculated - order.total_price) > 0.01:
        return {"error": "總金額不正確"}
    
    orders_db.append(order.dict())
    return {"message": "訂單已建立", "order": order.dict()}
```

### 第五步：Response Model（回傳格式控制）

```python
from typing import Optional, List
from pydantic import BaseModel

class ProductResponse(BaseModel):
    """產品回傳模型（只顯示部分欄位）"""
    id: int
    name: str
    price: float

class ProductListResponse(BaseModel):
    """產品列表回傳模型"""
    total: int
    products: List[ProductResponse]

@app.get("/products", response_model=ProductListResponse)
def get_products():
    """取得產品列表"""
    # 假設這是從資料庫取得的資料
    db_products = [
        {"id": 1, "name": "筆電", "price": 25000, "description": "高效能"},
        {"id": 2, "name": "手機", "price": 15000, "description": "最新款"},
    ]
    
    # response_model 會自動過濾，只回傳指定欄位
    return {
        "total": len(db_products),
        "products": db_products
    }
```

```python
# main.py
from fastapi import FastAPI, HTTPException, status
from pydantic import BaseModel, Field, EmailStr
from typing import Optional, List
from datetime import datetime

app = FastAPI()

# ========== 第一個 Model：用戶 ==========
class UserCreate(BaseModel):
    """創建用戶的資料模型"""
    username: str = Field(..., min_length=3, max_length=50, description="用戶名，3-50字")
    email: EmailStr = Field(..., description="有效的電子郵件")
    password: str = Field(..., min_length=6, description="密碼，至少6個字")
    age: Optional[int] = Field(None, ge=0, le=150, description="年齡，可選")

class UserResponse(BaseModel):
    """回傳用戶時使用的模型（不回傳密碼）"""
    id: int
    username: str
    email: str
    age: Optional[int]
    created_at: datetime

# ========== 第二個 Model：文章 ==========
class ArticleCreate(BaseModel):
    """創建文章的資料模型"""
    title: str = Field(..., min_length=1, max_length=200, description="標題")
    content: str = Field(..., min_length=10, description="內容，至少10個字")
    tags: List[str] = Field(default=[], description="標籤列表")
    published: bool = Field(False, description="是否發布")

class ArticleResponse(BaseModel):
    """文章回傳模型"""
    id: int
    title: str
    content: str
    tags: List[str]
    published: bool
    created_at: datetime

# ========== 模擬資料庫 ==========
users_db = []
articles_db = []
next_user_id = 1
next_article_id = 1

# ========== API 端點 ==========

# 首頁
@app.get("/")
def read_root():
    return {
        "message": "歡迎來到 POST API 測試",
        "docs": "請訪問 /docs 查看互動式 API 文件"
    }

# ---------- 用戶相關 ----------

@app.post("/users", response_model=UserResponse, status_code=status.HTTP_201_CREATED)
def create_user(user: UserCreate):
    """創建新用戶"""
    global next_user_id
    
    # 檢查 email 是否已存在
    for u in users_db:
        if u["email"] == user.email:
            raise HTTPException(
                status_code=400,
                detail="此 email 已被使用"
            )
    
    # 建立用戶資料
    new_user = {
        "id": next_user_id,
        "username": user.username,
        "email": user.email,
        "password": user.password,  # 實際開發中不應該這樣存！
        "age": user.age,
        "created_at": datetime.now()
    }
    
    users_db.append(new_user)
    next_user_id += 1
    
    # 回傳時不包含密碼（因為 response_model 會過濾）
    return new_user

@app.get("/users", response_model=List[UserResponse])
def get_users():
    """取得所有用戶"""
    return users_db

@app.get("/users/{user_id}", response_model=UserResponse)
def get_user(user_id: int):
    """取得特定用戶"""
    for user in users_db:
        if user["id"] == user_id:
            return user
    raise HTTPException(status_code=404, detail="用戶不存在")

# ---------- 文章相關 ----------

@app.post("/articles", response_model=ArticleResponse, status_code=status.HTTP_201_CREATED)
def create_article(article: ArticleCreate):
    """創建新文章"""
    global next_article_id
    
    new_article = {
        "id": next_article_id,
        "title": article.title,
        "content": article.content,
        "tags": article.tags,
        "published": article.published,
        "created_at": datetime.now()
    }
    
    articles_db.append(new_article)
    next_article_id += 1
    
    return new_article

@app.get("/articles", response_model=List[ArticleResponse])
def get_articles(published_only: bool = False):
    """取得文章列表"""
    if published_only:
        return [a for a in articles_db if a["published"]]
    return articles_db

@app.get("/articles/{article_id}", response_model=ArticleResponse)
def get_article(article_id: int):
    """取得特定文章"""
    for article in articles_db:
        if article["id"] == article_id:
            return article
    raise HTTPException(status_code=404, detail="文章不存在")

# ========== 啟動伺服器 ==========
if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="0.0.0.0", port=8000)
```

## 4️⃣ 常見錯誤提醒

### ❌ 忘記設定欄位為必填

```python
# 錯誤 ❌
class User(BaseModel):
    name: str  # 沒有 Field，可能造成混淆

# 正確 ✅
class User(BaseModel):
    name: str = Field(...)  # ... 表示必填
```

### ❌ 類型驗證失敗

```python
# 錯誤 ❌ - age 應該是數字，但傳了字串
{"name": "小明", "age": "二十歲"}

# 錯誤 ❌ - email 格式不正確
{"name": "小明", "email": "not-email"}

# FastAPI 會自動回傳錯誤訊息
{"detail": [{"loc": ["body", "email"], "msg": "value is not a valid email address", "type": "value_error"}]}
```

### ❌ 密碼長度不足

```python
# 錯誤 ❌ - 密碼少於 6 個字
{"username": "test", "email": "test@test.com", "password": "123"}

# 正確 ✅
{"username": "test", "email": "test@test.com", "password": "123456"}
```

### ❌ 使用 POST 但沒傳 Body

```python
# 錯誤 ❌
curl -X POST "http://localhost:8000/users"

# 正確 ✅ - 一定要加 -d 或 --data
curl -X POST "http://localhost:8000/users" -d '{"username":"test","email":"test@test.com","password":"123456"}'
```

```python
# basic_task.py
from fastapi import FastAPI, HTTPException
from pydantic import BaseModel, Field
from typing import List, Optional

app = FastAPI()

# 書籍模型
class Book(BaseModel):
    title: str = Field(..., description="書名")
    author: str = Field(..., description="作者")
    price: float = Field(..., gt=0, description="價格，正數")
    published_year: Optional[int] = Field(None, ge=1000, le=2100, description="出版年份")

# 模擬資料庫
books_db = []
next_id = 1

@app.post("/book", status_code=201)
def create_book(book: Book):
    """創建書籍"""
    global next_id
    
    new_book = {
        "id": next_id,
        **book.dict()
    }
    books_db.append(new_book)
    next_id += 1
    
    return {"message": "書籍創建成功", "book": new_book}

# 測試用：取得所有書籍
@app.get("/books")
def get_books():
    return books_db

if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="0.0.0.0", port=8000)
```
