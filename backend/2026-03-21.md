# 📚 Day 8：FastAPI 基礎 — Route、Query Parameters

## 1️⃣ 基礎概念（用生活比喻）

### 🎯 什麼是 FastAPI？

**比喻**：FastAPI 就像是一家「餐廳」的「服務生」。

想像你去一家餐廳吃飯：
- **顧客** = 使用者（Client）
- **服務生** = FastAPI伺服器
- **菜單** = API 的端點（routes）
- **廚房** = 你的 Python 程式碼

當顧客（使用者）點餐時：
1. 服務生收到訂單（HTTP 請求）
2. 服務生把訂單送到廚房（執行 Python 函式）
3. 廚師做好菜（程式邏輯處理）
4. 服務生把菜端給顧客（回傳 HTTP 回應）

### 🎯 什麼是 API？

**比喻**：API 就像是「翻譯員」或「橋樑」。

- 你不懂英文，對方不懂中文
- 翻譯員幫你們溝通
- API 就是電腦和電腦之間的「翻譯員」

### 🎯 什麼是 Route（路由）？

**比喻**：Route 就像是餐廳的「桌號」。

- 桌號 1 號：給「套餐 A」的客人
- 桌號 2 號：給「單點」的客人
- 桌號 3 號：給「甜點」的客人

在 FastAPI 中：
- `/users` = 用戶相關的請求
- `/products` = 產品相關的請求
- `/orders` = 訂單相關的請求

### 🎯 什麼是 HTTP 方法？

| 方法 | 比喻 | 用途 |
|------|------|------|
| GET | 閱讀菜單 | 獲取資料 |
| POST | 點餐 | 新增資料 |
| PUT | 更換餐點 | 更新資料 |
| DELETE | 退餐 | 刪除資料 |

### 🎯 什麼是 Query Parameters（查詢參數）？

**比喻**：Query Parameters 就像是「餐廳的篩選條件」。

例如你跟服務生說：
- 「**給我**（GET）**所有**（無參數）」→ 全部菜單
- 「**給我**（GET）**葷的**（?type=meat）」→ 只有葷食
- 「**給我**（GET）**200元以下**（?max_price=200）」→ 200元以下的菜
- 「**給我**（GET）**葷的且200元以下**（?type=meat&max_price=200）」→ 兩個條件

URL 範例：
```
https://restaurant.com/menu?type=meat&max_price=200
                ↑          ↑              ↑
            路由（桌號）   參數1           參數2
```

### 📌 Query Parameters 用法

```python
from fastapi import FastAPI

app = FastAPI()

# 沒有參數的 GET 路由
@app.get("/")
def read_root():
    return {"message": "歡迎來到 API！"}

# 有 Query Parameters 的 GET 路由
@app.get("/items")
def read_items(skip: int = 0, limit: int = 10):
    """
    skip: 跳過前幾筆資料（預設 0）
    limit: 最多回傳幾筆（預設 10）
    """
    items = ["蘋果", "香蕉", "橘子", "葡萄", "西瓜",
             "草莓", "芒果", "鳳梨", "水蜜桃", "奇異果",
             "梨子", "柚子", "石榴", "藍莓", "荔枝"]
    
    # 切片：從 skip 開始，取 limit 個
    return {"items": items[skip : skip + limit]}

# 另一個範例：搜尋功能
@app.get("/search")
def search_products(q: str = "", min_price: float = 0, max_price: float = 9999):
    """
    q: 搜尋關鍵字
    min_price: 最低價格
    max_price: 最高價格
    """
    products = [
        {"name": "iPhone", "price": 30000},
        {"name": "Android手機", "price": 15000},
        {"name": "筆記本電腦", "price": 25000},
        {"name": "耳機", "price": 1500},
        {"name": "鍵盤", "price": 800},
    ]
    
    # 過濾產品
    results = []
    for p in products:
        # 檢查關鍵字（不區分大小寫）
        if q.lower() in p["name"].lower():
            # 檢查價格範圍
            if min_price <= p["price"] <= max_price:
                results.append(p)
    
    return {"results": results, "count": len(results)}
```

### 📌 路徑參數（Path Parameters）

```python
from fastapi import FastAPI

app = FastAPI()

# 產品列表（模擬資料庫）
products = {
    1: {"name": "iPhone", "price": 30000},
    2: {"name": "Android手機", "price": 15000},
    3: {"name": "筆記本電腦", "price": 25000},
}

# 路徑參數：用 {變數} 表示
@app.get("/products/{product_id}")
def get_product(product_id: int):
    """根據 ID 獲取產品"""
    if product_id in products:
        return products[product_id]
    return {"error": "產品不存在"}

# 多個路徑參數
@app.get("/users/{user_id}/orders/{order_id}")
def get_user_order(user_id: int, order_id: int):
    """獲取某用戶的某筆訂單"""
    return {
        "user_id": user_id,
        "order_id": order_id,
        "status": "已出貨",
        "total": 1500
    }
```

### 📌 組合：路徑參數 + Query Parameters

```python
from fastapi import FastAPI

app = FastAPI()

# 模擬資料
all_orders = [
    {"id": 1, "user_id": 1, "product": "iPhone", "price": 30000, "status": "已完成"},
    {"id": 2, "user_id": 1, "product": "耳機", "price": 1500, "status": "配送中"},
    {"id": 3, "user_id": 2, "product": "鍵盤", "price": 800, "status": "已完成"},
    {"id": 4, "user_id": 2, "product": "滑鼠", "price": 500, "status": "待付款"},
    {"id": 5, "user_id": 3, "product": "筆電", "price": 25000, "status": "已完成"},
]

@app.get("/users/{user_id}/orders")
def get_user_orders(
    user_id: int,           # 路徑參數
    status: str = None,     # Query參數（可選）
    min_price: float = 0   # Query參數（可選）
):
    """獲取某用戶的訂單，可依狀態和價格篩選"""
    # 先找出該用戶的所有訂單
    user_orders = [o for o in all_orders if o["user_id"] == user_id]
    
    # 如果有指定狀態，再過濾
    if status:
        user_orders = [o for o in user_orders if o["status"] == status]
    
    # 如果有指定最低價格，再過濾
    if min_price > 0:
        user_orders = [o for o in user_orders if o["price"] >= min_price]
    
    return {
        "user_id": user_id,
        "orders": user_orders,
        "total_orders": len(user_orders)
    }

# 測試網址：
# http://127.0.0.1:8000/users/1/orders
# http://127.0.0.1:8000/users/1/orders?status=已完成
# http://127.0.0.1:8000/users/1/orders?min_price=1000
```

```python
# day8_fastapi/query_demo.py
from fastapi import FastAPI
from typing import Optional

app = FastAPI()

# 模擬書籍資料
books = [
    {"id": 1, "title": "Python 入門", "author": "小明", "price": 350},
    {"id": 2, "title": "FastAPI 實戰", "author": "大華", "price": 500},
    {"id": 3, "title": "JavaScript 精通", "author": "小美", "price": 450},
    {"id": 4, "title": "資料庫設計", "author": "阿強", "price": 400},
    {"id": 5, "title": "Docker 部署", "author": "小明", "price": 380},
    {"id": 6, "title": "Git 版本控制", "author": "小美", "price": 300},
    {"id": 7, "title": "演算法之道", "author": "大華", "price": 600},
]

# 練習 1：基本的 Query Parameters
@app.get("/books")
def get_books():
    """獲取所有書籍"""
    return {"books": books, "total": len(books)}

# 練習 2：可選的 Query Parameters
@app.get("/books/search")
def search_books(
    author: Optional[str] = None,
    min_price: Optional[float] = None,
    max_price: Optional[float] = None
):
    """搜尋書籍，可依作者和價格範圍篩選"""
    results = books
    
    # 依作者過濾
    if author:
        results = [b for b in results if author in b["author"]]
    
    # 依最低價格過濾
    if min_price is not None:
        results = [b for b in results if b["price"] >= min_price]
    
    # 依最高價格過濾
    if max_price is not None:
        results = [b for b in results if b["price"] <= max_price]
    
    return {"results": results, "count": len(results)}

# 練習 3：分頁功能
@app.get("/books/page")
def get_books_page(page: int = 1, page_size: int = 3):
    """分頁獲取書籍"""
    start = (page - 1) * page_size
    end = start + page_size
    page_books = books[start:end]
    
    total_pages = (len(books) + page_size - 1) // page_size
    
    return {
        "page": page,
        "page_size": page_size,
        "total_pages": total_pages,
        "total_books": len(books),
        "books": page_books
    }
```

### 🛠️ 練習 3：實作 Path Parameters

**步驟 1**：建立 `path_demo.py`
```python
# day8_fastapi/path_demo.py
from fastapi import FastAPI, HTTPException

app = FastAPI()

# 模擬學生資料
students = {
    1: {"id": 1, "name": "小明", "age": 20, "grade": "A"},
    2: {"id": 2, "name": "小美", "age": 19, "grade": "B"},
    3: {"id": 3, "name": "大華", "age": 21, "grade": "A"},
    4: {"id": 4, "name": "阿強", "age": 20, "grade": "C"},
}

# 練習 1：根據 ID 獲取單一學生
@app.get("/students/{student_id}")
def get_student(student_id: int):
    """根據 ID 獲取學生資料"""
    if student_id not in students:
        raise HTTPException(status_code=404, detail="學生不存在")
    return students[student_id]

# 練習 2：計算 BMI
@app.get("/bmi/{name}/{height}/{weight}")
def calculate_bmi(name: str, height: float, weight: float):
    """計算 BMI"""
    height_m = height / 100  # 公分轉公尺
    bmi = weight / (height_m ** 2)
    
    # 判斷體重狀態
    if bmi < 18.5:
        status = "體重過輕"
    elif bmi < 24:
        status = "體重正常"
    elif bmi < 27:
        status = "體重過重"
    else:
        status = "肥胖"
    
    return {
        "name": name,
        "height": height,
        "weight": weight,
        "bmi": round(bmi, 2),
        "status": status
    }

# 練習 3：貨幣轉換
@app.get("/exchange/{from_currency}/{to_currency}/{amount}")
def exchange_currency(from_currency: str, to_currency: str, amount: float):
    """簡單的貨幣轉換（模擬）"""
    rates = {
        "USD": 1.0,
        "TWD": 31.0,
        "JPY": 150.0,
        "EUR": 0.92,
    }
    
    # 轉換成 USD 再轉換成目標貨幣
    usd_amount = amount / rates.get(from_currency.upper(), 1.0)
    result = usd_amount * rates.get(to_currency.upper(), 1.0)
    
    return {
        "from": from_currency.upper(),
        "to": to_currency.upper(),
        "amount": amount,
        "result": round(result, 2)
    }
```

- http://127.0.0.1:8000/bmi/小明/170/70 → 計算小明的 BMI
- http://127.0.0.1:8000/exchange/USD/TWD/100 → 100 美金換台幣

## 4️⃣ 常見錯誤提醒

### ⚠️ 錯誤 1：忘記 import FastAPI

```python
# 錯！
app = FastAPI()  # NameError: name 'FastAPI' is not defined

# 對！
from fastapi import FastAPI
app = FastAPI()
```

### ⚠️ 錯誤 2：參數型態錯誤

```python
# 錯！product_id 應該是 int，但傳了字串
# URL: /products/abc
@app.get("/products/{product_id}")
def get_product(product_id: int):
    return {"id": product_id}
# 會回傳 422 Validation Error

# 對！傳入正確的數字
# URL: /products/1
@app.get("/products/{product_id}")
def get_product(product_id: int):
    return {"id": product_id}
```

### ⚠️ 錯誤 3：Query Parameters 沒設預設值又沒給可選

```python
# 錯！author 沒有預設值，會變成必要參數
@app.get("/books")
def get_books(author: str):
    return {"author": author}

# 對！設為可選（用 Optional 或給預設值 None）
from typing import Optional
@app.get("/books")
def get_books(author: Optional[str] = None):
    return {"author": author}

# 或者給預設值
@app.get("/books")
def get_books(author: str = ""):
    return {"author": author}
```

### ⚠️ 錯誤 4：路徑參數和 Query Parameters 名稱衝突

```python
# 錯！會衝突
@app.get("/users/{user_id}")
def get_user(user_id: int, user_id: str = "default"):
    # SyntaxError: duplicate argument
    pass

# 對！用不同名稱
@app.get("/users/{user_id}")
def get_user(user_id: int, query: str = "default"):
    return {"user_id": user_id, "query": query}
```

### ⚠️ 錯誤 5：忘記 return 東西

```python
# 錯！沒有回傳值
@app.get("/hello")
def say_hello():
    print("Hello")  # 這個只會印在伺服器主控台

# 對！要回傳
@app.get("/hello")
def say_hello():
    return {"message": "Hello"}
```

## 📚 今日重點回顧

| 概念 | 說明 |
|------|------|
| FastAPI | Python 的現代化 Web 框架 |
| Route（路由） | API 的端點，例如 `/users`、`/products` |
| HTTP 方法 | GET（獲取）、POST（新增）、PUT（更新）、DELETE（刪除） |
| Path Parameters | URL 中的參數，例如 `/users/{user_id}` |
| Query Parameters | URL 中 `?` 後面的參數，例如 `?name=小明&age=20` |
| uvicorn | 執行 FastAPI 的伺服器 |
| /docs | FastAPI 自動產生的 API 文件 |
