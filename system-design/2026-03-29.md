# 📚 【今日學習：URL 短網址服務（TinyURL）】

> 這是一道經典的系統設計面試題，也是各大公司（Google、Facebook、Twitter）喜歡考的題目。它涵蓋了許多核心系統設計概念：資料庫設計、快取、唯一 ID 生成、RESTful API。

## 1️⃣ 題目說明

### 面試官會怎麼問？

「假設你要設計一個 TinyURL 這樣的 URL 短網址服務，請問你會如何設計？例如：如何將 `https://example.com/very/long/path/to/some/resource?id=12345` 轉換成 `https://tinyurl.com/abc123`？」

### 需要澄清什麼問題？

在開始設計之前，你應該先問清楚以下問題：

1. **流量规模**：每天有多少人造訪？100 萬？1 億？
2. **功能需求**：只需要基本的短網址生成？還是要支援自訂網址？分析流量？
3. **可用性要求**：可以接受多少停機時間？（99.9% vs 99.99%）
4. **儲存需求**：要儲存多久？永遠？一年？
5. **是否需要認證**：誰可以創建短網址？公開還是會員制？

**關鍵設計決策：為什麼用數字 ID 而不是隨機字串？**

> **比喻**：想像你在圖書館管理書籍
> - **隨機字串** = 隨機把書放在書架上，找的時候要整個書架搜尋
> - **數字 ID** = 按編號排列，找的時候直接走過來

**選擇：auto-increment ID + base62 編碼**

```python
# 如何將數字 ID 轉換成短網址？
import string

BASE62 = string.ascii_letters + string.digits  # a-z, A-Z, 0-9 (62個字元)

def encode_id(num):
    """將數字 ID 轉換成短字串"""
    if num == 0:
        return BASE62[0]
    
    arr = []
    base = len(BASE62)
    
    while num > 0:
        num, rem = divmod(num, base)
        arr.append(BASE62[rem])
    
    return ''.join(reversed(arr))

def decode_id(short_code):
    """將短字串轉換回數字 ID"""
    base = len(BASE62)
    num = 0
    
    for char in short_code:
        num = num * base + BASE62.index(char)
    
    return num

# 測試
print(encode_id(12345))   # 輸出: "3DX" (62 進制)
print(decode_id("3DX"))  # 輸出: 12345
```

### 核心演算法：如何生成唯一短網址？

```
┌─────────────────────────────────────────────────────────────┐
│              三種生成短網址的方法                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  方法 1: MD5/SHA 雜湊                                        │
│  ────────────────────────────────────                      │
│  優點：簡單                                                 │
│  缺點：可能有碰撞（兩個長網址產生相同短網址）               │
│         需要處理衝突，增加複雜度                               │
│                                                             │
│  方法 2: 資料庫 auto-increment                              │
│  ────────────────────────────────────                      │
│  優點：不會碰撞、簡單                                       │
│  缺點：如果有多台 DB，需要 ID 生成服務                      │
│                                                             │
│  方法 3: 雪花演算法（Snowflake ID）                         │
│  ────────────────────────────────────                      │
│  優點：分散式產生唯一 ID、高效能                             │
│  缺點：需要額外的服務                                       │
│                                                             │
│  推薦：方法 2（單機）或方法 3（多機）                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 完整程式碼範例（Python + Flask）

```python
from flask import Flask, request, redirect, jsonify
import string
import sqlite3
from functools import wraps

app = Flask(__name__)

# ===== 資料庫初始化 =====
def init_db():
    conn = sqlite3.connect('tinyurl.db')
    cursor = conn.cursor()
    cursor.execute('''
        CREATE TABLE IF NOT EXISTS urls (
            id INTEGER PRIMARY KEY AUTOINCREMENT,
            short_code TEXT UNIQUE NOT NULL,
            original_url TEXT NOT NULL,
            created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
            click_count INTEGER DEFAULT 0
        )
    ''')
    conn.commit()
    conn.close()

# ===== Base62 編碼/解碼 =====
BASE62 = string.ascii_letters + string.digits  # a-zA-Z0-9

def encode_id(num):
    if num == 0:
        return BASE62[0]
    arr = []
    base = len(BASE62)
    while num > 0:
        num, rem = divmod(num, base)
        arr.append(BASE62[rem])
    return ''.join(reversed(arr))

def decode_id(short_code):
    base = len(BASE62)
    num = 0
    for char in short_code:
        num = num * base + BASE62.index(char)
    return num

# ===== API 路由 =====
@app.route('/api/shorten', methods=['POST'])
def shorten():
    data = request.get_json()
    original_url = data.get('url')
    
    if not original_url:
        return jsonify({'error': 'url is required'}), 400
    
    # 存入資料庫
    conn = sqlite3.connect('tinyurl.db')
    cursor = conn.cursor()
    cursor.execute('INSERT INTO urls (original_url) VALUES (?)', (original_url,))
    url_id = cursor.lastrowid
    conn.commit()
    conn.close()
    
    # 轉換成短網址
    short_code = encode_id(url_id)
    
    return jsonify({
        'short_url': f'https://tinyurl.com/{short_code}',
        'short_code': short_code
    })

@app.route('/<short_code>')
def redirect_to_url(short_code):
    conn = sqlite3.connect('tinyurl.db')
    conn.row_factory = sqlite3.Row
    cursor = conn.cursor()
    
    url_id = decode_id(short_code)
    cursor.execute('SELECT original_url FROM urls WHERE id = ?', (url_id,))
    row = cursor.fetchone()
    
    if row:
        # 更新點擊次數
        cursor.execute('UPDATE urls SET click_count = click_count + 1 WHERE id = ?', (url_id,))
        conn.commit()
        original_url = row['original_url']
        conn.close()
        return redirect(original_url, code=302)
    
    conn.close()
    return jsonify({'error': 'URL not found'}), 404

# ===== 啟動 =====
if __name__ == '__main__':
    init_db()
    app.run(debug=True, port=5000)
```
